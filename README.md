## 三大模块说明

### 1. `备赛系统/`：长期沉淀
平时训练用。按 **方向 → 真题 → 通用能力 → 物资 → 赛场** 五段式组织，是仓库最核心的部分。

### 2. `24/`：实战示例
2024 年实际做过的 C 题工程，包含完整的算法、驱动、文档，可作为新项目的代码骨架参考。

### 3. `赛题/`：原始题库
官方真题原始 docx，供检索和审题使用。

## 真题目录约定（强制）

`备赛系统/B_历年真题实战/<年份>/<题号_题名>/` 下统一遵守以下结构：

```
00_做题指南.md            # 审题、方案论证、分工
00_深度审题与方案论证.md  # 可选，重点题目使用
01_代码/                  # 工程代码
02_硬件/                  # 原理图、PCB、BOM
03_报告/                  # 最终设计报告（可选）
04_调试记录/              # 检查清单、踩坑总结
```

新增题目时务必保持这个结构，方便横向对比与复用。

## 使用方式

**带队老师**：先读 `备赛系统/总纲.md`，再按方向选 1~2 个主攻路线带训练。

**参赛学生**：
1. `C_通用能力/` 补齐基础
2. `A_方向训练/` 深入方向
3. `B_历年真题实战/` 按 4 天 3 夜节奏模拟
4. 赛前过一遍 `E_赛场实战/`
