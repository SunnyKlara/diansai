# Requirements — 2021 A 信号失真度测量装置

## 项目目标

把仓库已有 ~80% 完成度的 2021A 工程升级到 95%+，达成"第一梯队 5 道全部完成"里程碑。

## 来源约束（已审）

### 强制硬约束
1. **必须用 TI MCU + 片内 ADC**（题目第 4 条）
2. 输入峰峰值 300~600 mV（基本）/ 30~600 mV（发挥）
3. 基频 1 kHz（基本）/ 1~100 kHz（发挥）
4. THD 范围 5%~50%
5. 测量误差 Δ = THDx - THDo ≤ 5%（基本）/ ≤ 3%（发挥）— **绝对误差**
6. 谐波到 5 次
7. 一键启动后 ≤ 10 秒
8. 期间不得有人工操作

### 评分指标

| 项 | 分值 |
|---|---|
| 基本(1~3) 输入指标 | 10 |
| 基本(4) 误差 ≤ 5% | 20 |
| 基本(5) 显示 THDx | 15 |
| 基本(6) 时间 ≤ 10s | 5 |
| 发挥(1~5) 30mV~600mV / 1~100kHz / 误差 ≤ 3% / 波形 / 归一化 | 35 |
| 发挥(6) 手机显示 | 10 |
| 发挥(7) 其他 | 5 |
| 报告 | 20 |
| **合计** | **120** |
