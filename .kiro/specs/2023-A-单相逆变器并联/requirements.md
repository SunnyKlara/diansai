# Requirements — 2023 A 单相逆变器并联运行系统

## 项目目标

把仓库 7/10 完成度的 2023 A 工程升级到标杆 95%+，作为 2025 A 电源标杆题的姊妹作（共享 SVPWM/SPWM + 电压闭环骨架）。

## 来源约束（已审）

### 强制硬约束
1. 单相全桥逆变，输出 24V/50Hz/2A
2. THD ≤ 2%
3. 效率 ≥ 88%
4. 负载调整率 ≤ 0.2%
5. 发挥：双机并联（4A）、并网电流可设定、并网电流比可调

## 验收标准

### 必须修复
1. 写《00_深度审题与方案论证.md》五层结构
2. 重组代码为 Core / Drivers / Algorithm 分层（与 5 道标杆对齐）
3. 新建 `config.h` 集中所有可调参数
4. 算法层（spwm + control）去除 HAL 依赖
5. 新建 `Drivers/pwm_full_bridge.{c,h}` 封装 PWM 启停
6. 重命名 `报告正文.md` → `设计报告.md`（命名一致）
7. 升级 README

### 不在范围
