# Requirements — 2024 B 单相功率分析仪

## 项目目标

完成第一梯队最后一道题的最高标准刷题，达成"控制 / 仪表DSP / 电源 / THD / 功率分析"五方向标杆全覆盖。

## 来源约束（已审）

### 强制硬约束
1. **必须用 TI MCU**
2. 涉及 220V 市电 → 必须电气隔离（电流互感器 + 电压互感器）
3. 系统功耗 **≤ 50 mW**（极难，制约 MCU、显示、放大器选型）
4. 谐波分析至少到 **10 次**
5. 电压 / 电流 / 功率 / PF 误差 ≤ 1%
6. THD / 谐波误差 ≤ 2%

### 评分指标

| 项 | 分值 |
|---|---|
| (1) 电流 / 电压测量误差 ≤ 1% | 20 |
| (2) 功率 / 功率因数误差 ≤ 1% | 25 |
| (3) THD 和谐波（10 次）误差 ≤ 2% | 40 |
| (4) 系统功耗 ≤ 50mW | 10 |
| (5) 其他 | 5 |
| 报告 | 20 |
| **合计** | **120** |
