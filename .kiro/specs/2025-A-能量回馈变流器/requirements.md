# Requirements — 2025 A 能量回馈的变流器负载试验装置

## 项目目标

把仓库现有 ~70% 完成度的 2025A 工程升级到 95%+ 国一标准，覆盖审题、硬件、软件、调试、报告全链路。

## 来源约束（已审）

### 强制硬约束
1. 变流器 1 输出三相对称交流，线电压 **32V ± 0.25V** 有效值
2. 输出额定电流 **2A** 有效值/相
3. 频率 **20~100Hz 步进 1Hz** 可调
4. THD **≤ 2%**（线电压）
5. 负载调整率 **≤ 0.3%**（≈ 96mV）
6. 发挥(1)：组成回馈回路，直流电源 I1 ≥ 1A
7. 发挥(2)：整体 Pd 越小越好（30 分核心）

### 评分指标

| 项 | 分值 |
|---|---|
| 基本(1) 三相 32V/50Hz/2A | 14 |
| 基本(2) THD ≤ 2% | 13 |
| 基本(3) 负载调整率 ≤ 0.3% | 13 |
| 基本(4) 频率 20~100Hz 可调 | 10 |
| 发挥(1) I1 ≥ 1A | 15 |
| 发挥(2) Pd 越小越好 | **30** |
| 报告 | 20 |
| **合计** | **115（其他 5 分）** |
