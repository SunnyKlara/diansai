# Design — 2025 A 能量回馈的变流器负载试验装置

> 详细技术决策见 `备赛系统/B_历年真题实战/2025/A_能量回馈变流器/00_深度审题与方案论证.md`。

## 总体方案

**变流器 1**：STM32G474 + TIM1 三相互补 PWM + SVPWM 七段式调制 + LC 滤波 + 电压外环 PI 闭环。  
**变流器 2**：基本要求阶段用三相不控整流；冲刺 Pd 时升级为**三相同步整流**（用 MOSFET 替代二极管，省 8W 损耗）。

## 关键技术决策

| 决策项 | 选择 | 理由 |
|---|---|---|
| MCU | STM32G474RET6 | HRTIM + CORDIC + FMAC，是 G4 的电源/数字电源专用旗舰 |
| 调制 | SVPWM 七段式 | 比 SPWM 直流电压利用率高 15.5%，谐波更低 |
| 主开关 | IRF3205（基本）→ IRFB7434（冲 Pd）| Rds_on 8mΩ → 1.4mΩ |
| 整流 | 二极管（拿 15 分）→ 同步整流（冲 Pd）| 同步整流是国一的关键技术 |
| 电感 | T130-26 铁粉芯 1.5mH | 饱和电流 5A > 3A 峰值，三相一致性可控 |
| 控制环路 | 电压外环 10ms + SVPWM 内环 50μs | 解耦控制，外环慢内环快 |
| 状态机 | 7 状态（IDLE → PRECHARGE → RAMPUP → RUN_INV/FB → FAULT → STOP）| 含预充电和软启动，避免冲击 |

## 软件分层

```
01_代码/
├── config.h                    # 所有可调参数集中
├── Core/
│   └── main.c                  # 主状态机 + 时序调度
├── Drivers/                    # 硬件抽象（HAL）
│   ├── adc_3phase.c/.h         # 三相 ADC 同步采样 + DMA + 回调
│   ├── pwm_3phase.c/.h         # TIM1 启停 + 安全关断
│   ├── oled.c/.h
│   ├── button.c/.h
│   └── led_buzzer.c/.h
└── Algorithm/                  # 与硬件解耦
    ├── svpwm_3phase.c/.h       # 已有，保留
    ├── voltage_loop.c/.h       # 三相电压闭环 PI（线电压 RMS 反馈）
    ├── rms_meter.c/.h          # 滑动窗口 RMS（每基波周期更新）
    └── feedback_control.c/.h   # 能量回馈控制（同步整流时序，30 分核心）
```

## 状态机

```
ST_IDLE
  │ 按 START
  ▼
ST_PRECHARGE       母线通过 100Ω 电阻预充电 ~ 200ms
  │ Vbus > 0.9 × Vdc_target
  ▼
ST_RAMPUP          调制比从 0.1 软启动到稳态
  │ Vrms > 0.9 × Vref
  ▼
┌─────────────────────────────┐
│  ST_RUN_INVERTER（基本要求）│ ← 默认
│  ST_RUN_FEEDBACK（发挥）    │ ← 按 TASK 切换
└─────────────────────────────┘
  │ 故障 / 按 STOP
  ▼
ST_FAULT / ST_STOP   关 PWM + 蜂鸣 + 红灯
```

## 时序

| 周期 | 任务 |
|---|---|
| 50μs (TIM1 中断) | SVPWM 角度更新 + PWM 写入 |
| 50μs (ADC DMA 满) | 三相 ADC 采样 + RMS 累加 |
| 1ms (SysTick) | 系统计时 |
| 10ms | 电压外环 PI + 设定值更新 |
| 50ms | 故障检测（过压/过流/温度）|
| 200ms | OLED 刷新 + 按键扫描 |

## 误差预算

详见深度审题第二层。  
THD < 1.5%（裕 0.5%），负载调整率 < 50mV（裕 46mV），Pd 目标 < 5%（同步整流）。

## 兜底方案

| 风险点 | 主方案 | 兜底 |
|---|---|---|
| 同步整流时序错→反向电流 | HRTIM + 比较器精确触发 | 退回二极管整流（仍拿 15 分）|
| 三相电感不一致 | LCR 筛选 | 多绕几个备选 |
| HRTIM 配置卡死 | HRTIM | 用 TIM1（已有代码）|
| 母线预充电继电器烧 | 备用继电器 | 直接接（限流电阻保护）|
