# 飞行器与无人机方向 —— 总览与训练计划

## 为什么选/不选这个方向
- **优势**：每年必出1题，选的人少，做好了非常出彩
- **劣势**：成本高(¥2000+)、风险高(炸机)、调试周期长
- **适合**：有飞控经验、有充足准备时间(≥3个月)的队伍
- **绝对不适合**：零基础临时上手

## 近五年真题

## 硬件平台选择

### 方案A：Pixhawk飞控（推荐）
- 成熟稳定，社区资源丰富
- 支持ArduPilot/PX4固件
- 可通过MAVLink协议二次开发
- 价格：飞控板¥200~500

### 方案B：STM32自研飞控
- 完全自主可控
- 但开发周期极长（至少3个月）
- 不推荐电赛使用

### 飞行器平台
- 推荐：F450/F330机架 + 2212电机 + 30A电调
- 轴距≤45cm（题目要求）
- 必须加防护圈
- 电池：3S/4S 2200mAh锂电池

## 训练计划（20周，比其他方向多4周）

### 第1~4周：飞行基础
- [ ] 组装飞行器
- [ ] Pixhawk固件刷写和基本配置
- [ ] 手动遥控飞行练习（至少飞20次）
- [ ] 定高悬停调试

### 第5~8周：自主飞行
- [ ] 定点悬停（光流/UWB）
- [ ] 航点飞行（预设路径）
- [ ] 自主起飞和降落
- [ ] MAVLink通信协议

### 第9~12周：视觉识别
- [ ] OpenMV色块识别
- [ ] 二维码/条形码识别
- [ ] 目标检测（YOLO，用Jetson Nano）
- [ ] 视觉与飞控的数据融合

### 第13~16周：任务集成
- [ ] 全覆盖路径规划
- [ ] 目标发现→接近→识别→动作 完整流程
- [ ] 地面站开发（LCD显示+无线通信）

### 第17~20周：真题实战
- [ ] 做2025H模拟赛
- [ ] 做2024D模拟赛
- [ ] 应急预案（炸机后快速修复）

## 安全须知（极其重要）
1. **必须佩戴防护眼镜和手套**
2. 调试时在安全网内飞行
3. 电池充电时不要离开
4. 桨叶旋转时远离
5. 准备急停开关
6. 备用桨叶至少4套
