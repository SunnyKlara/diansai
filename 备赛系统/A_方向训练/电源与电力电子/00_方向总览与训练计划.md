# 电源与电力电子方向 —— 总览与训练计划

## 为什么选这个方向
- 每年A题必出电源，最稳定的方向
- 选的人相对少（门槛高），竞争压力小
- 分值结构清晰，基本要求容易拿满
- 适合硬件功底扎实的队伍

## 训练计划（16周）

### 第1~4周：基础拓扑
- [ ] Buck电路：搭建、调试、闭环稳压
- [ ] Boost电路：搭建、调试、闭环稳压
- [ ] 全桥逆变电路：SPWM调制、LC滤波
- [ ] 每个拓扑都要实际焊板子跑起来

### 第5~8周：进阶技术
- [ ] 双闭环控制（电压环+电流环）
- [ ] SVPWM算法实现（三相逆变用）
- [ ] 效率优化：同步整流、死区优化
- [ ] 功率因数校正（PFC）基础

### 第9~12周：真题实战
- [ ] 选一道往年真题，4天模拟赛
- [ ] 重点练习：从零搭建到基本要求完成
- [ ] 练习报告撰写

### 第13~16周：冲刺
- [ ] 再做一道真题模拟赛
- [ ] 练习发挥部分（并联均流、并网等）
- [ ] 物资检查、工具箱整理
- [ ] 报告模板准备

## 队伍配置建议
- 硬件主力1人：负责主电路设计、焊接、功率级调试
- 硬件辅助/软件1人：负责控制算法、MCU编程、采样电路
- 全能/报告1人：负责辅助电路、测试、报告撰写
