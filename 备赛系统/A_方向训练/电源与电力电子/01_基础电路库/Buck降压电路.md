# Buck降压电路 —— 从原理到实物

## 一、电路原理

### 拓扑结构
```
Vin ──┬── [MOSFET Q1] ──┬── [电感 L] ──┬── Vout
      │                  │              │
      │            [二极管 D1]      [电容 C]
      │                  │              │
      GND ───────────────┴──────────────┘
```

### 工作原理
- Q1导通：Vin通过L给C充电，电感电流上升
- Q1关断：电感通过D1续流，电感电流下降
- 输出电压：Vout = D × Vin（D为占空比）

### 关键公式
- 电感值：L = Vout × (1-D) / (ΔIL × fsw)
- 电容值：C = ΔIL / (8 × fsw × ΔVout)
- 其中 fsw 为开关频率，ΔIL 为电感纹波电流，ΔVout 为输出纹波电压

## 二、设计实例：12V→5V/2A

### 参数选择
| 参数 | 值 | 说明 |
|------|-----|------|
| Vin | 12V | 输入电压 |
| Vout | 5V | 输出电压 |
| Iout | 2A | 输出电流 |
| fsw | 100kHz | 开关频率 |
| ΔIL | 0.6A (30%Iout) | 电感纹波 |
| ΔVout | 50mV (1%Vout) | 输出纹波 |

### 计算结果
- D = 5/12 = 0.417
- L = 5 × (1-0.417) / (0.6 × 100k) = 48.6μH → 选 **47μH**
- C = 0.6 / (8 × 100k × 0.05) = 15μF → 选 **22μF 陶瓷电容**

## 三、控制电路

### 方案A：硬件PWM控制（简单快速）
- 用TL494或SG3525产生PWM
- 电压反馈通过分压电阻+误差放大器
- 适合快速搭建验证

### 方案B：MCU数字控制（推荐，比赛用）
```c
// STM32 Buck闭环控制伪代码
// 定时器中断中执行（频率=开关频率）

void TIM1_UP_IRQHandler(void) {
    // 1. ADC采样输出电压
    float vout = ADC_Read(CH_VOUT) * 3.3 / 4096 * VOUT_RATIO;
    
    // 2. 计算误差
    float error = VOUT_REF - vout;
    
    // 3. PI控制器
    integral += error;
    if (integral > INTEGRAL_MAX) integral = INTEGRAL_MAX;
    if (integral < INTEGRAL_MIN) integral = INTEGRAL_MIN;
    
    float duty = Kp * error + Ki * integral;
    
    // 4. 限幅
    if (duty > 0.9) duty = 0.9;  // 最大占空比限制
    if (duty < 0.05) duty = 0.05;
    
    // 5. 更新PWM占空比
    TIM1->CCR1 = (uint16_t)(duty * TIM1->ARR);
}
```

### PI参数整定
- Kp起始值：0.1，逐步增大到输出开始振荡，然后减半
- Ki起始值：Kp/100，逐步增大到稳态误差消除
- 实际调参需要在实物上反复测试

## 四、PCB设计要点

### 布局原则
1. 功率回路面积最小化（Vin→Q1→L→Cout→GND）
2. 驱动信号走线尽量短
3. 采样电阻靠近输出端
4. 输入输出电容紧贴功率管

### 常见错误
- ❌ 功率回路面积太大→EMI严重
- ❌ 采样点在电感前面→测量不准
- ❌ 驱动走线太长→开关损耗大
- ❌ 没有自举电容→高侧MOSFET无法驱动

## 五、调试步骤

1. **先不上电**：检查焊接，万用表测短路
2. **空载上电**：只接输入电源，不接负载，看PWM波形
3. **开环测试**：手动设置占空比，测输出电压是否=D×Vin
4. **闭环测试**：使能PI控制，看输出是否稳定在设定值
5. **加载测试**：逐步增加负载电流，观察输出电压变化
6. **效率测试**：测量输入功率和输出功率，计算效率
7. **纹波测试**：示波器AC耦合看输出纹波

## 六、常见问题排查

| 现象 | 可能原因 | 解决方法 |
|------|----------|----------|
| 输出电压不对 | 占空比计算错误 | 检查PWM频率和占空比 |
| 输出振荡 | PI参数不当 | 减小Kp |
| 效率低 | 开关损耗大 | 检查驱动波形上升沿 |
| MOSFET发热 | 导通电阻大或开关损耗 | 换更好的管子或加散热 |
| 输出纹波大 | 电容ESR高 | 换陶瓷电容或并联 |
