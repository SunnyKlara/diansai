# 全桥逆变电路 —— 从原理到实物

## 一、电路原理

### 拓扑结构
```
        Vdc+
         │
    ┌────┴────┐
    │         │
  [Q1]      [Q3]
    │         │
    ├── a     ├── b    a-b之间接LC滤波器→负载
    │         │
  [Q2]      [Q4]
    │         │
    └────┬────┘
         │
        Vdc-
```

### SPWM调制原理
- 正弦参考波 vs 三角载波比较
- 参考波>载波：Q1/Q4导通，Vab=+Vdc
- 参考波<载波：Q2/Q3导通，Vab=-Vdc
- 经LC滤波后得到正弦输出

## 二、设计实例：36Vdc→24Vac/50Hz/2A

### 这是2023A基本要求的核心电路

### 参数选择
| 参数 | 值 |
|------|-----|
| 直流母线 | 36V |
| 输出电压 | 24Vrms (33.9V峰值) |
| 输出频率 | 50Hz |
| 输出电流 | 2A |
| 开关频率 | 20kHz |
| 调制比 | 33.9/36 = 0.94 |

### LC滤波器设计
- 截止频率：选在开关频率的1/10 = 2kHz
- L = 1/(4π²×fc²×C)
- 选C=4.7μF → L = 1/(4π²×2000²×4.7e-6) = 1.35mH → 选 **1.5mH**
- 实际需要根据THD要求微调

## 三、SPWM代码实现

```c
// STM32G4 SPWM生成（利用HRTIM或高级定时器）
// 核心思路：在定时器更新中断中计算正弦表查表值

#define SINE_TABLE_SIZE 200   // 一个正弦周期200个点
#define PWM_PERIOD 4000       // 20kHz开关频率对应的ARR值

// 正弦表（0~1范围，乘以调制比后使用）
const float sine_table[SINE_TABLE_SIZE]; // 预计算sin(2π*n/200), n=0~199

volatile uint16_t sine_index = 0;
volatile float mod_index = 0.94;  // 调制比

// 定时器中断：每个开关周期更新一次占空比
void TIM1_UP_IRQHandler(void) {
    // 查正弦表
    float sine_val = sine_table[sine_index];
    
    // 计算占空比（单极性SPWM）
    float duty = 0.5 + 0.5 * mod_index * sine_val;
    
    // 更新PWM（互补输出，Q1/Q2和Q3/Q4互补）
    uint16_t ccr = (uint16_t)(duty * PWM_PERIOD);
    TIM1->CCR1 = ccr;       // Q1/Q2桥臂
    TIM1->CCR2 = PWM_PERIOD - ccr;  // Q3/Q4桥臂（反相）
    
    // 正弦表索引递增
    sine_index++;
    if (sine_index >= SINE_TABLE_SIZE) sine_index = 0;
    
    // 清除中断标志
    TIM1->SR &= ~TIM_SR_UIF;
}

// 输出频率 = 开关频率 / 正弦表大小 = 20000/200 = 100Hz
// 要得到50Hz：正弦表大小改为400，或开关频率改为10kHz
// 推荐：保持20kHz开关频率，正弦表400点
```

## 四、闭环稳压

### 电压环
```c
// 输出电压有效值闭环
// 每个正弦周期计算一次RMS值

float vout_rms_measure(void) {
    float sum_sq = 0;
    for (int i = 0; i < SAMPLES_PER_CYCLE; i++) {
        float v = adc_buffer[i] * SCALE_FACTOR;
        sum_sq += v * v;
    }
    return sqrt(sum_sq / SAMPLES_PER_CYCLE);
}

// PI控制器调节调制比
void voltage_loop(void) {
    float vout_rms = vout_rms_measure();
    float error = VOUT_REF_RMS - vout_rms;  // 24V
    
    static float integral = 0;
    integral += error * Ki_v;
    integral = CLAMP(integral, 0.5, 1.0);
    
    mod_index = Kp_v * error + integral;
    mod_index = CLAMP(mod_index, 0.1, 0.95);
}
```

## 五、THD优化技巧

要达到THD≤2%的要求：
1. **提高开关频率**：20kHz以上，LC滤波效果更好
2. **死区补偿**：死区会引入低次谐波，需要软件补偿
3. **正弦表精度**：至少200点/周期，用float计算
4. **LC参数匹配**：滤波器截止频率要合理
5. **负载前馈**：根据负载电流调整输出

## 六、调试步骤

1. 先用示波器看PWM波形，确认SPWM正确
2. 不接负载，看LC滤波后的输出波形
3. 接电阻负载，测输出电压有效值
4. 使能闭环，测负载调整率
5. 用功率分析仪测THD
6. 测效率（输入功率vs输出功率）
