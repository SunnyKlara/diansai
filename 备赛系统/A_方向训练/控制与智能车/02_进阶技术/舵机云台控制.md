# 舵机云台控制 —— 2023E/2025E必备技术

## 一、二轴云台结构

```
        [激光笔]
           │
      ┌────┴────┐
      │ 俯仰舵机 │ ← 控制上下角度
      └────┬────┘
           │
      ┌────┴────┐
      │ 偏航舵机 │ ← 控制左右角度
      └────┬────┘
           │
        [底座]
```

## 二、舵机PWM控制

### 标准舵机信号
- 频率：50Hz（周期20ms）
- 脉宽：500μs~2500μs
- 500μs → 0°
- 1500μs → 90°（中位）
- 2500μs → 180°

### STM32定时器配置
```
PSC = 71 (72MHz/72 = 1MHz)
ARR = 19999 (1MHz/20000 = 50Hz)

脉宽(μs) = CCR值
```

### 代码
```c
// 设置舵机角度 (0~180度)
void servo_set_angle(TIM_HandleTypeDef* htim, uint32_t channel, float angle)
{
    // 角度到脉宽的映射
    float pulse_us = 500.0f + angle / 180.0f * 2000.0f;
    
    // 限幅
    if (pulse_us < 500) pulse_us = 500;
    if (pulse_us > 2500) pulse_us = 2500;
    
    __HAL_TIM_SET_COMPARE(htim, channel, (uint16_t)pulse_us);
}

// 使用示例
servo_set_angle(&htim2, TIM_CHANNEL_1, 90);   // 偏航90度(中位)
servo_set_angle(&htim2, TIM_CHANNEL_2, 45);   // 俯仰45度
```

## 三、视觉引导PID瞄准

### 闭环控制流程
```
摄像头检测目标位置(像素) → 计算偏差 → PID → 调整舵机角度 → 循环
```

### 完整代码
```c
// 云台PID瞄准控制
typedef struct {
    float kp, ki, kd;
    float integral;
    float prev_error;
    float angle;        // 当前角度
} GimbalAxis_t;

GimbalAxis_t yaw  = {0.05f, 0.001f, 0.02f, 0, 0, 90.0f};
GimbalAxis_t pitch = {0.05f, 0.001f, 0.02f, 0, 0, 90.0f};

void gimbal_track(int16_t target_cx, int16_t target_cy)
{
    // 偏航控制 (X方向)
    float err_x = (float)target_cx;  // 像素偏差，中心=0
    yaw.integral += err_x;
    yaw.integral = CLAMP(yaw.integral, -500, 500);
    float d_x = err_x - yaw.prev_error;
    yaw.prev_error = err_x;
    
    float adj_x = yaw.kp * err_x + yaw.ki * yaw.integral + yaw.kd * d_x;
    yaw.angle += adj_x;
    yaw.angle = CLAMP(yaw.angle, 10, 170);
    
    // 俯仰控制 (Y方向)
    float err_y = (float)target_cy;
    pitch.integral += err_y;
    pitch.integral = CLAMP(pitch.integral, -500, 500);
    float d_y = err_y - pitch.prev_error;
    pitch.prev_error = err_y;
    
    float adj_y = pitch.kp * err_y + pitch.ki * pitch.integral + pitch.kd * d_y;
    pitch.angle -= adj_y;  // 注意方向可能需要取反
    pitch.angle = CLAMP(pitch.angle, 10, 170);
    
    // 输出到舵机
    servo_set_angle(&htim2, TIM_CHANNEL_1, yaw.angle);
    servo_set_angle(&htim2, TIM_CHANNEL_2, pitch.angle);
}
```

## 四、调参技巧

### Kp调参
- 从0.01开始，逐步增大
- 太小：响应慢，追不上目标
- 太大：振荡，激光笔抖动
- 合适值通常在0.03~0.1之间

### Kd调参
- 从Kp的一半开始
- 抑制振荡，让运动更平滑
- 太大会导致响应变慢

### Ki调参
- 通常很小或不加
- 用于消除静态偏差
- 太大会导致积分饱和

### 实际调参流程
1. 先调偏航轴（水平），固定俯仰
2. 偏航调好后再调俯仰轴
3. 两轴参数可能不同（因为负载不同）
4. 在实际距离(1m)和实际靶标上调参

## 五、常见问题

| 问题 | 原因 | 解决 |
|------|------|------|
| 舵机抖动 | Kp太大 | 减小Kp |
| 追踪滞后 | 控制频率太低 | 提高到50Hz+ |
| 静态偏差 | 没有积分项 | 加小Ki |
| 运动中偏移 | 没有运动补偿 | 加入位置前馈 |
| 舵机堵转发热 | 角度超出范围 | 加限幅保护 |
