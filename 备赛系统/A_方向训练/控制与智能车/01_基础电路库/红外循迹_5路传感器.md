# 红外循迹 —— 5路传感器 + PID

## 一、传感器原理

TCRT5000红外对管：
- 发射管发出红外光
- 接收管检测反射光强度
- 黑色表面吸收红外光→反射弱→输出高电平(1)
- 白色表面反射红外光→反射强→输出低电平(0)

## 二、5路传感器排列

```
行驶方向 →

    [L2] [L1] [C] [R1] [R2]
     ●    ●    ●    ●    ●
  ─────────────────────────── 黑线(1.8cm)
  ═══════════════════════════ 白色地面
```

传感器间距约8~10mm，总宽度约40mm，覆盖1.8cm黑线两侧。

## 三、偏差计算

```c
// 传感器状态: 在黑线上=1, 白色=0
// 加权平均法计算偏差

float calc_error(uint8_t L2, uint8_t L1, uint8_t C, uint8_t R1, uint8_t R2)
{
    float weighted = -2.0f*L2 + -1.0f*L1 + 0.0f*C + 1.0f*R1 + 2.0f*R2;
    float sum = L2 + L1 + C + R1 + R2;
    
    if (sum == 0) {
        // 全白：丢线了，返回上次偏差方向的最大值
        return last_error > 0 ? 2.5f : -2.5f;
    }
    
    last_error = weighted / sum;
    return last_error;
}
```

### 典型传感器状态与偏差值

| L2 | L1 | C | R1 | R2 | 偏差 | 含义 |
|----|----|----|----|----|------|------|
| 0 | 0 | 1 | 0 | 0 | 0.0 | 正中 |
| 0 | 1 | 1 | 0 | 0 | -0.5 | 略偏左 |
| 0 | 0 | 1 | 1 | 0 | +0.5 | 略偏右 |
| 1 | 1 | 0 | 0 | 0 | -1.5 | 严重偏左 |
| 0 | 0 | 0 | 1 | 1 | +1.5 | 严重偏右 |
| 1 | 1 | 1 | 1 | 1 | 0.0 | 十字路口/T字路口 |
| 0 | 0 | 0 | 0 | 0 | ±2.5 | 丢线 |

## 四、PID循迹完整代码

```c
// PID参数（需要实际调参）
#define KP  25.0f
#define KI  0.1f
#define KD  15.0f
#define BASE_SPEED 60  // 基础速度(0~100)

static float integral = 0;
static float prev_error = 0;
static float last_error = 0;  // 丢线时用

void track_control(void)
{
    // 1. 读传感器
    uint8_t L2 = read_sensor(0);
    uint8_t L1 = read_sensor(1);
    uint8_t C  = read_sensor(2);
    uint8_t R1 = read_sensor(3);
    uint8_t R2 = read_sensor(4);
    
    // 2. 计算偏差
    float error = calc_error(L2, L1, C, R1, R2);
    
    // 3. PID计算
    integral += error;
    // 积分限幅
    if (integral > 200) integral = 200;
    if (integral < -200) integral = -200;
    
    float derivative = error - prev_error;
    prev_error = error;
    
    float correction = KP * error + KI * integral + KD * derivative;
    
    // 4. 差速控制
    int left_speed  = BASE_SPEED - (int)correction;
    int right_speed = BASE_SPEED + (int)correction;
    
    // 限幅
    if (left_speed > 100) left_speed = 100;
    if (left_speed < -50) left_speed = -50;
    if (right_speed > 100) right_speed = 100;
    if (right_speed < -50) right_speed = -50;
    
    // 5. 输出到电机
    Motor_Set(left_speed, right_speed);
}

// 在定时器中断中每10ms调用一次
void TIM3_IRQHandler(void)
{
    if (__HAL_TIM_GET_FLAG(&htim3, TIM_FLAG_UPDATE)) {
        __HAL_TIM_CLEAR_FLAG(&htim3, TIM_FLAG_UPDATE);
        track_control();
    }
}
```

## 五、PID调参步骤

1. **先只用P**：KI=0, KD=0, KP从10开始
   - KP太小：小车反应慢，弯道冲出去
   - KP太大：小车左右摆动（振荡）
   - 找到刚好开始振荡的KP值，记为Ku

2. **加D**：KD从KP的一半开始
   - D项抑制振荡，让小车更平稳
   - KD太大：小车反应变慢

3. **加I**（通常很小或不加）：
   - I项消除稳态偏差
   - 循迹场景下通常不需要太大的I

4. **典型参数范围**：
   - KP: 15~40
   - KI: 0~0.5
   - KD: 5~25
   - 具体值取决于小车速度、传感器间距、黑线宽度

## 六、弯道处理技巧

- 弯道时减速：检测到偏差大时降低BASE_SPEED
- 急弯时单轮反转：correction超过BASE_SPEED时一侧反转
- 十字路口：全黑时直行（或根据任务需求转弯）
