# TB6612电机驱动 —— 从接线到代码

## 一、TB6612模块引脚

```
TB6612模块:
  VM    ← 电机电源 (5~12V)
  VCC   ← 逻辑电源 (3.3V/5V)
  GND   ← 地
  STBY  ← 待机控制 (高=工作, 低=待机)
  
  AIN1  ← 电机A方向1 (GPIO)
  AIN2  ← 电机A方向2 (GPIO)
  PWMA  ← 电机A速度 (PWM)
  AO1   → 电机A输出+
  AO2   → 电机A输出-
  
  BIN1  ← 电机B方向1 (GPIO)
  BIN2  ← 电机B方向2 (GPIO)
  PWMB  ← 电机B速度 (PWM)
  BO1   → 电机B输出+
  BO2   → 电机B输出-
```

## 二、控制逻辑

| AIN1 | AIN2 | PWMA | 电机状态 |
|------|------|------|----------|
| 1 | 0 | PWM | 正转，速度=PWM占空比 |
| 0 | 1 | PWM | 反转，速度=PWM占空比 |
| 1 | 1 | X | 刹车 |
| 0 | 0 | X | 滑行 |

## 三、STM32接线示例

| TB6612 | STM32 | 说明 |
|--------|-------|------|
| VM | 7.4V电池 | 电机电源 |
| VCC | 3.3V | 逻辑电源 |
| GND | GND | 共地 |
| STBY | 3.3V | 常开 |
| AIN1 | PB12 | GPIO输出 |
| AIN2 | PB13 | GPIO输出 |
| PWMA | PA0 (TIM2_CH1) | PWM输出 |
| BIN1 | PB14 | GPIO输出 |
| BIN2 | PB15 | GPIO输出 |
| PWMB | PA1 (TIM2_CH2) | PWM输出 |

## 四、代码实现

```c
#include "stm32f1xx_hal.h"

// 引脚定义
#define AIN1_PORT GPIOB
#define AIN1_PIN  GPIO_PIN_12
#define AIN2_PORT GPIOB
#define AIN2_PIN  GPIO_PIN_13
#define BIN1_PORT GPIOB
#define BIN1_PIN  GPIO_PIN_14
#define BIN2_PORT GPIOB
#define BIN2_PIN  GPIO_PIN_15

extern TIM_HandleTypeDef htim2;

/**
 * 设置电机速度
 * @param left  左轮速度 -100~+100 (正=前进)
 * @param right 右轮速度 -100~+100
 */
void Motor_Set(int left, int right)
{
    // 左轮 (电机A)
    if (left >= 0) {
        HAL_GPIO_WritePin(AIN1_PORT, AIN1_PIN, GPIO_PIN_SET);
        HAL_GPIO_WritePin(AIN2_PORT, AIN2_PIN, GPIO_PIN_RESET);
        __HAL_TIM_SET_COMPARE(&htim2, TIM_CHANNEL_1, left * 10);  // ARR=1000
    } else {
        HAL_GPIO_WritePin(AIN1_PORT, AIN1_PIN, GPIO_PIN_RESET);
        HAL_GPIO_WritePin(AIN2_PORT, AIN2_PIN, GPIO_PIN_SET);
        __HAL_TIM_SET_COMPARE(&htim2, TIM_CHANNEL_1, (-left) * 10);
    }
    
    // 右轮 (电机B)
    if (right >= 0) {
        HAL_GPIO_WritePin(BIN1_PORT, BIN1_PIN, GPIO_PIN_SET);
        HAL_GPIO_WritePin(BIN2_PORT, BIN2_PIN, GPIO_PIN_RESET);
        __HAL_TIM_SET_COMPARE(&htim2, TIM_CHANNEL_2, right * 10);
    } else {
        HAL_GPIO_WritePin(BIN1_PORT, BIN1_PIN, GPIO_PIN_RESET);
        HAL_GPIO_WritePin(BIN2_PORT, BIN2_PIN, GPIO_PIN_SET);
        __HAL_TIM_SET_COMPARE(&htim2, TIM_CHANNEL_2, (-right) * 10);
    }
}

/**
 * 刹车（快速停止）
 */
void Motor_Brake(void)
{
    HAL_GPIO_WritePin(AIN1_PORT, AIN1_PIN, GPIO_PIN_SET);
    HAL_GPIO_WritePin(AIN2_PORT, AIN2_PIN, GPIO_PIN_SET);
    HAL_GPIO_WritePin(BIN1_PORT, BIN1_PIN, GPIO_PIN_SET);
    HAL_GPIO_WritePin(BIN2_PORT, BIN2_PIN, GPIO_PIN_SET);
}

// TIM2配置: PSC=71(72MHz/72=1MHz), ARR=999, PWM频率=1kHz
```

## 五、常见问题

| 问题 | 原因 | 解决 |
|------|------|------|
| 电机不转 | STBY没接高 | STBY接3.3V |
| 电机抖动 | PWM频率太低 | 提高到1kHz以上 |
| 两轮速度不一致 | 电机个体差异 | 加编码器闭环 |
| 电机反转 | 接线反了 | 交换AO1/AO2 |
