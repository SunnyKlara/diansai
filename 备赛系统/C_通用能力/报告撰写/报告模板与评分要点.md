# 电赛报告撰写 —— 模板与评分要点

> 报告占20分（满分120），写好报告是性价比最高的得分手段

## 一、评分标准（通用）

| 项目 | 分值 | 评委看什么 |
|------|------|-----------|
| 系统方案 | 2~4分 | 有没有方案比较？框图清不清晰？ |
| 理论分析与计算 | 5~7分 | 公式推导对不对？参数怎么算的？ |
| 电路与程序设计 | 4~6分 | 原理图完整吗？流程图有没有？ |
| 测试方案与结果 | 3~5分 | 数据完整吗？有没有分析？ |
| 结构与规范性 | 2~3分 | 摘要、图表编号、格式规范 |

## 二、报告结构模板

```
标题：XXXX（题目名称）

摘要（200字以内）
  - 做了什么
  - 用了什么方法
  - 达到了什么指标

一、系统方案
  1.1 方案比较与选择
      方案一：XXX（优点、缺点）
      方案二：XXX（优点、缺点）
      → 选择方案X，理由：...
  1.2 系统总体框图

二、理论分析与计算
  2.1 核心原理分析（公式推导）
  2.2 关键参数计算（电感、电容、频率等）
  2.3 控制策略分析

三、电路与程序设计
  3.1 主电路设计（原理图）
  3.2 控制电路设计
  3.3 程序设计（流程图 + 关键代码说明）

四、测试方案与测试结果
  4.1 测试方案（用什么仪器、怎么测）
  4.2 测试数据（表格）
  4.3 测试结果分析

五、总结

参考文献

附录（完整原理图、关键源代码）
```

## 三、高分技巧

### 摘要（很多评委只看摘要）
```
【好的摘要示例】
本作品设计了一个单相逆变器并联运行系统。采用全桥逆变拓扑，
SPWM调制，STM32G474数字控制。单机运行时输出24V/50Hz/2A，
THD为1.8%，效率89.2%，负载调整率0.15%。双机并联时输出4A，
电流分配比误差小于3%。
```

### 方案比较（必须有，不能只写一个方案）
```
【好的方案比较】
| 方案 | 拓扑 | 优点 | 缺点 |
|------|------|------|------|
| 方案一 | 半桥逆变 | 器件少 | 输出电压低，需要升压 |
| 方案二 | 全桥逆变 | 输出电压高，控制灵活 | 器件多 |
综合考虑，选择方案二。
```

### 测试数据（必须用表格，最好有曲线图）
- 每个基本要求对应一组测试数据
- 数据要完整（多个测试点）
- 最好画成曲线图（Excel即可）
- 要有误差分析

### 图表规范
- 每张图必须有编号和标题：`图1 系统总体框图`
- 每个表必须有编号和标题：`表1 负载调整率测试数据`
- 原理图要清晰可读
- 流程图要规范（开始/结束用椭圆，判断用菱形）

## 四、报告撰写时间安排

| 时间 | 任务 |
|------|------|
| 第2天晚上 | 开始写框架（标题、章节结构） |
| 第3天 | 写方案、理论分析、电路设计部分 |
| 第3天晚上 | 补充测试数据（边测边写） |
| 第4天上午 | 完善测试结果、写摘要和总结 |
| 第4天下午 | 排版、检查、打印 |

## 五、常见扣分点

- ❌ 没有摘要
- ❌ 没有方案比较（只写了一个方案）
- ❌ 没有理论计算（只有结果没有过程）
- ❌ 测试数据不完整（只测了一个点）
- ❌ 图表没有编号
- ❌ 报告超过10页（不含附录）
- ❌ 没有参考文献
