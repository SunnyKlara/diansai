# 2025 F 简易自动接收机 —— 深度审题

## 题目精确

| 项 | 分值 |
|---|---|
| 基本(1) 接收单频 | 25 |
| 基本(2) 自动锁台 | 25 |
| 基本(3) RSSI 显示 | 20 |
| 发挥(1) 多模式 AM/FM | 15 |
| 发挥(2) 全频段扫描 | 15 |
| 报告 | 20 |

## 物理本质
**外差超外差接收机**：射频 → 混频 → 中频 → 解调 → 音频。  
1~30MHz 短波 + 455kHz 中频。

## 方案
- MCU：STM32F407
- 本振：Si5351A（I²C 控制，10kHz~160MHz）
- 混频：NE602（双平衡混频器）
- 中频滤波：455kHz 陶瓷滤波器（CFW455F）
- 检波：AGC + 包络检波

## 复用
| 模块 | 来源 |
|---|---|
| Si5351 驱动 | 已有占位 → 真机扩展 |
| 自动扫频 + RSSI | 与 2023 D 类似（IQ 思路）|
| AM/FM 解调 | 与 2023 D modulation_detect 复用 |

## 风险
| 风险 | 缓解 |
|---|---|
| Si5351 抖动 | 5ppm TCXO 替换默认晶振 |
| NE602 失真 | 输入 < -20dBm |

保底 50 / 期望 80
