## 系统架构：超外差接收机

```
天线(SMA) → [BPF 88~108MHz] → [LNA] → [混频器] → [IF BPF 10.7MHz]
                                          ↑
                                    [本振 PLL]
                                    (98.7~118.7MHz)

→ [IF放大+AGC] → [FM鉴频器] ──→ 音频输出 → [8Ω负载]
                  [AM检波器] ──→ 
                  
→ [RSSI检测] → [MCU: 扫描/识别/控制/显示]
```

## 自动搜索算法

```c
// 从88MHz扫描到108MHz，步进100kHz
// 每个频点停留检测RSSI
// RSSI超过门限 → 锁定该频率 → 尝试解调

#define FREQ_START  88000000UL   // 88MHz
#define FREQ_STOP   108000000UL  // 108MHz
#define FREQ_STEP   100000UL     // 100kHz步进
#define IF_FREQ     10700000UL   // 10.7MHz中频

void auto_search(void) {
    for (uint32_t freq = FREQ_START; freq <= FREQ_STOP; freq += FREQ_STEP) {
        // 设置本振频率 = 信号频率 + 中频
        PLL_SetFreq(freq + IF_FREQ);
        delay_ms(5);  // 等待PLL锁定
        
        // 读取RSSI
        float rssi = read_rssi();
        
        if (rssi > RSSI_THRESHOLD) {
            // 找到信号！
            current_freq = freq;
            
            // 识别调制方式
            ModType_t mod = identify_modulation();
            
            // 切换到对应解调器
            switch_demod(mod);
            
            // 启动AGC
            agc_enable();
            
            display_result(freq, mod);
            return;
        }
    }
    display_string("No signal found");
}
```

## 调制方式识别方法

```
1. 检测IF信号包络变化：
   - 包络恒定 → CW 或 FM
   - 包络有低频变化 → AM

2. 如果包络恒定，检测鉴频器输出：
   - 鉴频器有音频输出 → FM
   - 鉴频器无输出 → CW

判断流程：
   IF信号 → 包络检波 → 低通滤波 → 检测AC分量
   有AC分量 → AM
   无AC分量 → 鉴频器输出检测
              有输出 → FM
              无输出 → CW
```

## 发挥部分关键约束
- **5V单电源，电流≤500mA**（总功耗≤2.5W）
- SA612功耗约6mA，非常适合
- Si5351功耗约30mA
- MCU选低功耗型号或降频运行
- 避免使用大功率运放
