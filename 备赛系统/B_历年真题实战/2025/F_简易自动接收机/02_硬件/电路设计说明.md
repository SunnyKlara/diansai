# 2025F 电路设计说明

## 一、超外差接收机架构

```
天线(SMA) → [BPF] → [LNA] → [混频器] → [IF BPF] → [IF Amp+AGC]
              │                  ↑          │              │
         88~108MHz          [本振PLL]    10.7MHz      [RSSI检测]
         带通滤波         98.7~118.7MHz  陶瓷滤波      AD8310
                           Si5351A
                           
→ [FM鉴频器] ──┐
   MC3361       │
                ├→ [模拟开关] → [音频功放] → [8Ω负载]
→ [AM检波器] ──┘    CD4053      LM386
   1N60+RC          STM32控制
```

## 二、各模块电路

### LNA (SPF5189Z模块)
```
SMA天线 → [DC隔直电容10nF] → SPF5189Z模块 → [输出]
                                  │
                               5V供电
                               
增益: +18dB
噪声系数: 0.6dB
频率范围: 50MHz~4GHz
```
直接用成品模块，不需要自己设计。

### 混频器 (SA612)
```
         VCC(5V)
          │
         [0.1μF]
          │
    ┌─────┴─────┐
    │   SA612   │
RF→ Pin1    Pin4 ← 本振(Si5351)
    │           │
    Pin2    Pin5 → IF输出(10.7MHz)
    │           │
    GND    Pin3=GND
    └───────────┘
    
Pin1: RF输入(LNA输出)，串联10nF隔直
Pin4: 本振输入(Si5351 CLK0)，串联10nF隔直
Pin5: IF输出，接10.7MHz陶瓷滤波器
```

### IF滤波器
```
SA612 Pin5 → [10.7MHz陶瓷滤波器] → IF放大器输入

陶瓷滤波器: 中心频率10.7MHz，带宽约280kHz
两端需要匹配电容(参考滤波器数据手册)
```

### AGC (AD603)
```
IF信号 → AD603输入 → AD603输出 → 分路到解调器
              ↑
         控制电压(0~1V)
              ↑
         STM32 PWM → RC滤波(10kΩ+10μF)
         
AD603增益范围: -11dB ~ +31dB (42dB动态范围)
控制电压0V→最小增益，1V→最大增益
```

### FM鉴频器 (MC3361)
```
IF信号 → MC3361 → 音频输出
         │
      外接10.7MHz陶瓷鉴频器
      (或用455kHz中频方案)
```

### AM检波器
```
IF信号 → [1N60锗管] → [RC滤波(10kΩ+1nF)] → 音频输出
                        │
                     去除IF载波，保留音频包络
```

### 音频功放 (LM386)
```
解调音频 → [10μF] → LM386 Pin3(+) → Pin5(输出) → [220μF] → 8Ω负载
                     Pin2(-) → GND
                     Pin6 ← VCC(5V)
                     Pin1-Pin8之间接10μF → 增益=200(46dB)
```

## 三、RSSI检测 (AD8310)
```
IF信号(分路) → AD8310输入 → AD8310输出(DC电压)
                              │
                           → STM32 ADC(PA0)

AD8310: 对数检波器
输出电压 = 0.024V/dB × (输入功率dBm + 108)
-95dBm → 0.024×13 = 0.31V
-60dBm → 0.024×48 = 1.15V
```

## 四、5V单电源设计
- 所有器件都选5V供电版本
- SA612: 4.5~8V，5V直接供电
- AD603: ±5V双电源 → 需要电荷泵产生-5V
  - 或用AD8367替代(单5V供电)
- LM386: 4~12V，5V直接供电
- STM32: 3.3V，用AMS1117-3.3从5V降压

## 五、PCB设计要点
1. 射频部分用覆铜板手工制作（不用万能板！）
2. 接地面要完整，射频走线50Ω阻抗
3. LNA和混频器之间走线尽量短
4. 数字部分和射频部分分区布局
5. Si5351输出到SA612的走线要短且有地保护
