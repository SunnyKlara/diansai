## 引脚分配
| 引脚 | 功能 | 说明 |
|------|------|------|
| PB6/PB7 | I2C1 | Si5351(0x60) + OLED(0x3C) 共用I2C |
| PA0 | ADC1_CH0 | RSSI检测（AD8310对数检波器输出） |
| PA1 | ADC1_CH1 | AGC反馈（输出峰值检波） |
| PA2 | ADC1_CH2 | 包络检测（AM/FM识别用） |
| PA3 | ADC1_CH3 | 鉴频器输出检测 |
| PA8 | GPIO_OUT | FM/AM解调切换（CD4053模拟开关） |
| PB0 | TIM3_CH3 | AGC控制PWM → RC滤波 → AD603 |
| PC13 | GPIO_IN | 启动按键 |

## 信号链路
```
SMA天线 → SPF5189Z(LNA, +18dB) → SA612(混频器)
                                      ↑
                                Si5351(本振, 98.7~118.7MHz)

→ 10.7MHz陶瓷滤波器 → AD603(VGA, AGC控制) → 分路:
    ├→ MC3361(FM鉴频器) ──┐
    └→ 1N60(AM包络检波) ──┤
                           ↓
                    CD4053(模拟开关, STM32控制选择FM或AM)
                           ↓
                    LM386(音频功放) → 8Ω负载
```

## 自动搜索算法
1. 从88MHz开始，步进100kHz扫描到108MHz
2. 每个频点设置Si5351本振频率 = 信号频率 + 10.7MHz
3. 等待5ms让PLL锁定
4. 读取RSSI（AD8310输出电压与信号功率成对数关系）
5. RSSI超过门限 → 锁定该频率
6. 采集包络和鉴频器输出 → 判断AM/FM/CW
7. 切换对应解调器 → 启动AGC
