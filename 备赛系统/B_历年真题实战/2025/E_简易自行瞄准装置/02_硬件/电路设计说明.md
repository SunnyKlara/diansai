# 2025E 电路设计说明

## 一、系统供电架构

```
锂电池 7.4V (2S)
    │
    ├──[开关1: 小车电源]──→ LM2596(5V) ──→ MSPM0 + 电机驱动 + 传感器
    │
    └──[开关2: 瞄准电源]──→ LM2596(5V) ──→ STM32 + OpenMV + 舵机 + 激光笔
```

**两个独立开关是题目硬性要求！**MSPM0控制板和瞄准模块必须能独立上下电。

## 二、小车底盘电路

### 电机驱动
```
MSPM0 PA0 ──→ TB6612 AIN1
MSPM0 PA1 ──→ TB6612 AIN2  (实际用PWM，AIN1/AIN2用于方向)
MSPM0 TIMA0_C0 ──→ TB6612 PWMA

MSPM0 PA2 ──→ TB6612 BIN1
MSPM0 PA3 ──→ TB6612 BIN2
MSPM0 TIMA0_C2 ──→ TB6612 PWMB

TB6612 VM ← 7.4V电池
TB6612 VCC ← 3.3V
TB6612 STBY ← 3.3V (常开)
```

### 红外传感器
```
5路TCRT5000模块:
  VCC ← 5V
  GND ← GND
  D1~D5 ──→ MSPM0 PA12~PA16 (数字输出，黑线=高)
  
  传感器间距约8mm，总宽约40mm
  安装在小车前端底部，距地面约5~10mm
```

### 编码器
```
左编码器:
  A相 ──→ MSPM0 PB0 (TIMG0输入)
  B相 ──→ MSPM0 PB1
  VCC ← 3.3V
  GND ← GND

右编码器: 同理接PB2/PB3 (TIMG1)
```

## 三、瞄准模块电路

### 云台舵机
```
偏航舵机(MG996R):
  信号线 ← STM32 PA1 (TIM2_CH2, 50Hz PWM)
  VCC ← 5V (独立供电，不要从STM32取)
  GND ← GND

俯仰舵机(SG90):
  信号线 ← STM32 PA2 (TIM2_CH3, 50Hz PWM)
  VCC ← 5V
  GND ← GND
```

**注意**：舵机电流较大(MG996R堵转约2A)，必须从5V电源直接供电，不能从STM32的3.3V取。

### 激光笔开关
```
STM32 PB0 ──→ [1kΩ] ──→ 2N7000(N-MOS)栅极
                          │
                        漏极 ← 激光笔负极
                          │
                        源极 ← GND
                        
激光笔正极 ← 3V (两节纽扣电池或从5V降压)
```

用MOSFET开关控制激光笔通断，STM32 GPIO高电平=激光开。

### OpenMV安装
- OpenMV摄像头朝向靶标方向
- 与激光笔平行安装在云台上
- 随云台一起转动
- 或固定安装（不随云台转），通过坐标变换计算

**推荐固定安装**：避免云台转动时摄像头晃动影响识别。

## 四、整车尺寸控制

题目要求：≤25cm(长) × 15cm(宽) × 25cm(高)

```
侧视图:
         ┌──OpenMV──┐
         │  云台+激光 │  ← 高度≤25cm
    ┌────┴──────────┴────┐
    │    MSPM0 + STM32   │  ← 控制板叠放
    │    电池             │
    └──[轮]────────[轮]──┘
         ← 长度≤25cm →
```

- 电池放在底盘下方或中间
- 控制板叠放节省空间
- 云台装在车顶
- 传感器在车头底部
