## 共享 config.h

```c
#define UART_LINK_BAUD       115200
#define MSG_CAR_READY        0xFF      // 小车 → 瞄准
#define MSG_AIM_DONE         0xAA      // 瞄准 → 小车
```

## 时钟

| MCU | 时钟 |
|---|---|
| 小车 F407 | HSE 8MHz × 21 = 168 MHz |
| 瞄准 F103 | HSE 8MHz × 9 = 72 MHz |

## 启动顺序

1. 瞄准 MCU 先上电（OpenMV 需 30s 启动）
2. 小车上电（看 LED_G 闪烁表示就绪）
3. 按 KEY1 → 小车自动循迹到目标位
4. 到位发 MSG_CAR_READY → 瞄准开始 PID 追踪
5. 命中后发 MSG_AIM_DONE → 小车蜂鸣 + 完成
