# 简易自行瞄准装置 —— 设计报告

## 摘要

本作品设计并制作了一个简易自行瞄准装置，由自动寻迹小车和二维云台瞄准模块两部分组成。小车采用TI MSPM0G3507控制，5路红外传感器循迹，PID算法实现沿100cm正方形轨迹稳定行驶。瞄准模块采用STM32F103控制，OpenMV摄像头检测靶标位置，二维舵机云台控制蓝紫激光笔指向靶心。静态瞄准精度D1≤1.5cm，响应时间≤1.5s。运动中瞄准时激光光斑持续落在靶面上，D1≤2cm。

**关键词**：MSPM0；PID循迹；OpenMV视觉；舵机云台；激光瞄准

## 一、系统方案

### 1.1 方案比较与选择

#### 循迹方案
| 方案 | 传感器 | 优点 | 缺点 |
|------|--------|------|------|
| 方案一 | 红外对管阵列 | 简单可靠，响应快 | 只能检测黑线 |
| 方案二 | 摄像头循迹 | 信息量大 | 题目禁止小车用摄像头 |
| 方案三 | 灰度传感器 | 模拟量，精度高 | 受环境光影响 |

选择方案一。题目明确禁止小车安装摄像头，且红外对管方案成熟可靠。

#### 瞄准方案
| 方案 | 方法 | 优点 | 缺点 |
|------|------|------|------|
| 方案一 | OpenMV视觉+PID | 闭环控制，精度高 | 需要OpenMV |
| 方案二 | 编码器推算+开环 | 不需要视觉 | 累积误差大 |
| 方案三 | IMU+编码器融合 | 不依赖视觉 | 精度有限 |

选择方案一。视觉闭环是实现≤2cm精度的最可靠方案。

## 二、理论分析与计算

### 2.1 循迹PID控制

采用位置式PID控制器，控制量为左右轮速度差：

```
correction = Kp × error + Ki × Σerror + Kd × Δerror
left_speed = base_speed - correction
right_speed = base_speed + correction
```

其中error由5路传感器加权平均计算，范围[-2, +2]。

经实验整定：Kp=25, Ki=0.1, Kd=15, base_speed=60。

### 2.2 瞄准角度计算

已知靶标距离约500mm，OpenMV图像分辨率320×240，视场角约60°。

像素到角度的映射关系：
```
角度偏差 = arctan(像素偏差 × 像素尺寸 / 焦距)
≈ 像素偏差 × 60° / 320 = 像素偏差 × 0.1875°
```

舵机脉宽到角度：约10μs/度。

### 2.3 运动补偿分析

小车运动时，靶标相对小车的方位角持续变化。需要根据小车位置实时计算云台目标角度。

设小车在轨迹上的位置为(x_car, y_car)，靶心位置为(x_target, y_target)：
```
目标偏航角 = arctan((y_target - y_car) / (x_target - x_car))
```

通过编码器里程计估算(x_car, y_car)，实时更新云台角度。
