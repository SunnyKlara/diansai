# 电路模型探究装置 —— 设计报告

## 摘要

本作品设计并制作了一个电路模型探究装置，能够对未知RLC电路进行自主学习建模并推理生成输出信号。装置采用STM32F407微控制器，AD9833 DDS产生扫频信号，双ADC同步采样测量频率响应（增益和相位），存储为查找表。推理阶段对输入信号做FFT，逐频率分量查表应用频率响应，IFFT合成输出信号，通过DAC连续输出。基本要求中，装置能根据已知传递函数H(s)控制RC有源滤波电路的输出电压，误差≤5%。发挥部分中，装置在2分钟内完成对未知电路的学习建模，正确识别滤波器类型，并对正弦波和矩形波输入推理生成与未知电路一致的输出，峰峰值误差≤10%。

**关键词**：频率响应测量；系统辨识；FFT；数字信号处理；DDS

## 一、系统方案

### 1.1 方案比较与选择

对未知电路建模有三种方案：

| 方案 | 方法 | 优点 | 缺点 |
|------|------|------|------|
| A | 参数化建模（拟合传递函数） | 模型紧凑 | 需假设模型阶数，拟合可能不收敛 |
| B | 非参数化建模（频率响应查找表） | 不需假设模型，测量多准就多准 | 需要插值，测量点数有限 |
| C | 时域建模（测量脉冲响应） | 一次测量 | 脉冲信噪比差，卷积计算量大 |

**选择方案B**。理由：
1. 不需要假设电路结构，对任意线性电路都适用
2. 实现简单（FFT→查表→IFFT），MCU上可实时运行
3. 通过自适应扫频密度，在截止频率附近加密采样，精度有保障

## 二、理论分析与计算

### 2.1 频率响应测量原理

对于线性时不变(LTI)系统，频率响应H(f)完全描述了系统的输入输出关系：

$$Y(f) = H(f) \cdot X(f)$$

其中X(f)和Y(f)分别是输入和输出信号的频谱。

测量方法：向系统输入频率为f的正弦信号x(t)=A·sin(2πft)，测量输出y(t)=B·sin(2πft+φ)，则：

$$|H(f)| = B/A, \quad \angle H(f) = \varphi$$

### 2.2 单频DFT提取幅值和相位

对N个采样点，计算频率f对应的DFT系数：

$$X(f) = \sum_{n=0}^{N-1} x[n] \cdot e^{-j2\pi f n / f_s}$$

分解为实部和虚部：

$$\text{Re} = \sum x[n] \cos(2\pi f n / f_s), \quad \text{Im} = -\sum x[n] \sin(2\pi f n / f_s)$$

幅值 $|X| = \sqrt{\text{Re}^2 + \text{Im}^2} \cdot 2/N$，相位 $\angle X = \text{atan2}(\text{Im}, \text{Re})$

### 2.3 同步采样消除频谱泄漏

设计采样率使得N个采样点恰好覆盖M个信号周期：

$$f_s = f \cdot N / M$$

此时信号频率恰好落在DFT的某个bin上，无频谱泄漏，幅值和相位测量精度最高。

### 2.4 推理阶段的频域处理

对输入信号x[n]做FFT得到X[k]，对每个频率bin k：

$$Y[k] = H(f_k) \cdot X[k]$$

其中 $f_k = k \cdot f_s / N$，$H(f_k)$通过查找表插值获得。

对Y[k]做IFFT得到输出信号y[n]。

### 2.5 相位解缠绕

atan2返回[-π, π]，相邻频率点的相位差可能超过π，需要解缠绕：

若 $\varphi[k] - \varphi[k-1] > \pi$，则 $\varphi[k] -= 2\pi$
若 $\varphi[k] - \varphi[k-1] < -\pi$，则 $\varphi[k] += 2\pi$

### 2.6 实时性分析

| 步骤 | 计算量 | 耗时(@168MHz) |
|------|--------|---------------|
| FFT(2048点) | O(N·logN) | 3ms |
| 查表插值(1024点) | O(N·logM) | 1ms |
| IFFT(2048点) | O(N·logN) | 3ms |
| **总计** | | **7ms** |

采集一帧：2048/200kHz = 10.24ms > 7ms → 处理来得及，可实时运行。

### 4.4 测试结果分析

正弦波推理精度较高（误差<5%），因为只涉及单一频率分量。
矩形波推理在低占空比(10%)时误差较大，原因是高次谐波超出了频率响应测量范围，被截断处理。这与实际电路的行为一致——高次谐波本来就会被滤波器衰减。

## 五、总结

本作品成功实现了对未知RLC电路的自主学习建模和推理输出功能。采用频率响应查找表的非参数化建模方法，避免了传递函数拟合的复杂性和不确定性。双ADC同步采样保证了相位测量精度，自适应扫频密度在截止频率附近加密采样提高了关键区域的精度。推理阶段的FFT→查表→IFFT处理链路在STM32F407上实现了实时运行，DAC双缓冲输出保证了波形的连续性。
