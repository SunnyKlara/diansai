# 2025 G 电路模型探究装置 —— STM32F407VET6 引脚分配表

> 平台：STM32F407VET6（最小系统板，¥40 级别）  
> 工具：CubeMX + Keil MDK 5

## 1. 完整引脚一览

| 引脚 | 功能 | 外设 | 备注 |
|---|---|---|---|
| **DDS 信号源 AD9833** ||||
| PA5 | SPI1_SCK | SPI1 → AD9833 SCLK | 18 MHz |
| PA6 | SPI1_MISO | （AD9833 不用，留空）| |
| PA7 | SPI1_MOSI | SPI1 → AD9833 SDATA | |
| PB6 | GPIO | AD9833 FSYNC（CS）| 低有效 |
| **双 ADC 同步采样** ||||
| PA0 | ADC1_IN0 | 参考端电压 V_R | OPA2277 缓冲 |
| PA1 | ADC2_IN1 | 待测端电压 V_DUT | OPA2277 缓冲 |
| **DAC 双通道（X-Y 输出）** ||||
| PA4 | DAC1_OUT1 | 实部信号 X | 给示波器 |
| PA5 | DAC1_OUT2 | 虚部信号 Y | 与 SPI1_SCK 复用，二选一 |
| **OLED I2C** ||||
| PB8 | I2C1_SCL | OLED SCL | 4.7kΩ 上拉 |
| PB9 | I2C1_SDA | OLED SDA | |
| **按键 + LED** ||||
| PE0 | GPIO | KEY_START | 上拉输入 |
| PE1 | GPIO | KEY_MODE | |
| PD12 | GPIO | LED_R | |
| PD13 | GPIO | LED_G | |
| **调试串口** ||||
| PA9 | USART1_TX | UART 调试 | 921600 |
| PA10 | USART1_RX | | |

## 2. 时钟

| 项 | 值 |
|---|---|
| HSE | 8 MHz（板载晶振）|
| PLL_M = 8, PLL_N = 336, PLL_P = 2 | SYSCLK = 168 MHz |
| AHB | 168 MHz |
| APB1 | 42 MHz（DAC, I2C）|
| APB2 | 84 MHz（SPI, USART, ADC）|

## 3. ADC1 + ADC2 同步双模式

| 项 | 值 |
|---|---|
| 模式 | Regular Simultaneous Mode |
| 触发 | TIM2 TRGO（采样率 200 kHz）|
| DMA | DMA2 Stream0, Half-word, Circular |
| 缓冲 | uint16_t s_buf[2][1024] |
| 中断 | DMA2 半完成 + 完成 |

## 4. DAC1（X-Y 输出）

| 项 | 值 |
|---|---|
| 模式 | DAC1 双通道 + DMA + TIM6 触发 |
| 频率 | 与 ADC 同步（如 200 kHz）|
| 输出 | 0~3.3V（接示波器需要中点偏置 1.65V）|

## 5. SPI1（AD9833）

| 项 | 值 |
|---|---|
| Master 模式 | |
| Baud | 18 MHz（系统时钟 / 4） |
| CPOL | 1（空闲高）|
| CPHA | 0 |
| Frame | 16 bit MSB First |

## 6. 中断列表

| 中断 | 优先级 | 用途 |
|---|---|---|
| DMA2_Stream0 (ADC) | 0 | 1024 帧到位 → FFT |
| TIM2 (200kHz) | 1 | 触发 ADC |
| TIM6 (200kHz) | 2 | 触发 DAC |
| EXTI0/1 (KEY) | 3 | 按键 |
| USART1 | 4 | 调试 |

## 7. CubeMX 关键勾选

```
RCC: HSE → Crystal/Ceramic → 8MHz
Clock Config: HCLK = 168MHz
ADC1: IN0 / Trigger = TIM2 TRGO / DMA Continuous
ADC2: IN1 / Trigger = TIM2 TRGO / Multi-Mode = Regular Sim
DAC1: OUT1+OUT2 / Trigger = TIM6 TRGO / DMA Circular
SPI1: Full-Duplex Master / 18MHz / CPOL=1, CPHA=0
TIM2: 内部时钟 / Counter = 168M/200k - 1 / Trigger Update
TIM6: 同 TIM2
I2C1: 100kHz / 4.7k 上拉
USART1: 921600 / Async / NoFlowControl
```
