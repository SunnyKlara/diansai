> 配套：`接线表.md` / `电路设计说明.md` / `材料清单_BOM.md`

## 总体策略

```
┌─────────────────────────────────────────────────────────────────┐
│  策略 1：板分两区 —— 模拟区（左 1/2）+ 数字区（右 1/2）        │
│  策略 2：DDS 输出 + 反馈采样紧靠 MCU，走线 < 3cm                │

```

## 关键模拟链路

```
  AD9833 → 1Vpp 正弦
         ↓
  电压跟随器 (OPA2350) → 0.5Vpp（驱动 DUT）
         ↓
  DUT (R / RC / RL / 二极管)
         ↓
  跨阻放大 (OPA2350, RF=10kΩ) → 把电流转电压
         │
         ├── ADC2 IN1（采电压）
         └── ADC1 IN0（采电流）
         
布局：跨阻 OPA2350 紧靠 STM32 ADC 引脚，走线 < 3 cm
      电源去耦：每个 OPA 旁 100nF（陶瓷）+ 10μF（钽）
```

## 关键间距

| 项 | 距离 |
|---|---|
| AD9833 ↔ STM32 SPI | < 5 cm（SPI ≤ 10MHz 不挑剔）|
| 跨阻 OPA ↔ STM32 ADC | **< 3 cm**（关键，长走线噪声大）|
| DAC 输出 ↔ ADC 输入 | ≥ 2 cm（DAC 高频跳变会串入 ADC）|
| AGND ↔ DGND 桥接点 | 1 处（在 STM32 板地）|

## 接地（双 ADC 同步采样关键）

```
                AGND（模拟地，跨阻/DDS/调理）
                       │
                       │ 0Ω 跳线 / 磁珠（单点）
                       ▼
                DGND（数字地，MCU/SPI/UART）
                       │
                       ▼
                USB 调试 GND
                
注意：磁珠位置必须在 STM32 ADC 引脚附近，**不要**放在板边缘
```

## 走线规则

| 线 | 走法 |
|---|---|
| AD9833 SPI | 双绞，远离 ADC 输入线 |
| ADC IN0 / IN1 | 平行走线、长度等长（同步采样关键）|
| DAC X / Y | 屏蔽线（赛场示波器接 50Ω）|
| 5V → 3V3 LDO | 输入 4.7μF + 输出 1μF |

## 散热

| 元件 | 处理 |
|---|---|
| OPA2350 | 不发热 |
| AD9833 | 不发热 |
| STM32F407 | 100MHz 全速运行 ~80mA，长时间运行温升 5°C，无需散热片 |

## 故障定位 × 布局

| 现象 | TP 检测 |
|---|---|
| DDS 无输出 | TP1 应是 1Vpp 100Hz~100kHz 正弦 |
| 阻抗测量全 0 | TP2 V_ADC 静态 1.65V 摆动幅度 0 → 通道断 |
| 阻抗值不稳定 | TP5 I_ADC 摆动幅度 < 噪声 → 跨阻 RF 不对 |
| LCD 不刷新 | TP6 DAC_X 是否在 1.5~3V 摆动 |

## 5 cm × 8 cm 模板（如手头万用板小）

```
策略：MCU 用 LaunchPad（外置）+ 万用板只焊调理 + DDS + 接口
- 减少焊点 50%
- 布局更宽松
- 缺点：调试时连接线多，需电烙铁定型固定
```

## 测试点验证

| TP | 期望 |
|---|---|
| TP1 DDS_OUT | 1Vpp 1kHz 正弦（赛前用示波器看一眼）|
| TP2 V_ADC | 静态 1.65V，接 1kΩ DUT 时摆动 ±300mV |
| TP3 3V3 | 3.30V ± 0.05V |
| TP5 I_ADC | 静态 1.65V，接 1kΩ 时摆动 ±100mV |
| TP6 DAC_X | 测 X-Y 模式时 0.5~2.5V 摆动（绕圈/直线）|
