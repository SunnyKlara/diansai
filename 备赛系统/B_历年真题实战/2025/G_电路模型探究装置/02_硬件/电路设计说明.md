# 2025G 电路设计说明

## 一、系统信号链路总图

```
【学习阶段】

  AD9833(DDS) ──→ [放大器×3.3] ──┬──→ 被测电路输入
                                  │
                            ADC1(PA0)采样
                                  
  被测电路输出 ──→ [缓冲器(×1)] ──→ ADC2(PA1)采样
  
  
【推理阶段】

  信号发生器 ──→ [缓冲器(×1, Rin≥100kΩ)] ──→ ADC1(PA0)采样
                                                    │
                                              STM32处理
                                              (FFT→查表→IFFT)
                                                    │
                                              DAC1(PA4)输出
                                                    │
                                          [二阶低通滤波器] ──→ 探究装置输出
                                          (fc=100kHz)
```

## 二、DDS输出放大电路

```
                    Rf=22kΩ (0.1%)
                ┌────[===]────┐
                │             │
AD9833 OUT ──[C1=100nF]──┬──[─\     │
                         │    >──┬──┘──→ 输出(约2Vpp)
              Vbias ──[Ri=10kΩ]──[+/
              (1.65V)         │
                             GND
                             
增益 = 1 + Rf/Ri = 1 + 22k/10k = 3.2
AD9833输出0.6Vpp → 放大后约1.9Vpp

Vbias通过Ri叠加直流偏置，使输出在0~3.3V范围内。
```

### 为什么增益选3.2而不是更大？

被测电路的输入阻抗未知（题目说R=1kΩ~10kΩ）。
如果输入信号太大，可能超出运放的线性范围。
1.9Vpp是一个安全的幅度——足够大以保证信噪比，又不会过驱动。

## 三、ADC输入缓冲/偏置电路

```
                    R3=100kΩ (输入阻抗)
信号输入 ──[===]──┬──[─\
                  │    >──┬──→ ADC输入(PA0或PA1)
   Vbias ──[R4=100kΩ]──[+/  │
   (1.65V)            │     │
                     GND   [C2=100pF]
                            │
                           GND
                           
这是一个同相放大器，增益=1（电压跟随器）。
R3提供100kΩ输入阻抗（题目要求≥100kΩ）。
C2是抗混叠滤波电容：fc = 1/(2π×R3×C2) = 1/(2π×100k×100p) ≈ 16kHz
等等，16kHz太低了！50kHz信号会被滤掉。

修正：C2改为10pF → fc ≈ 160kHz，或者去掉C2，用独立的抗混叠滤波器。
```

### 抗混叠滤波器

ADC采样率200kHz，奈奎斯特频率100kHz。
需要在100kHz以上有足够的衰减，防止高频噪声混叠到信号频段。

```
简单方案：一阶RC低通
  R=1kΩ, C=1nF → fc = 1/(2π×1k×1n) = 159kHz
  在200kHz处衰减约-4dB → 不够

更好方案：二阶Sallen-Key低通
  fc=150kHz，在200kHz处衰减约-12dB → 可以接受
  
  用OPA2277的另一半通道实现。
```

## 四、DAC输出滤波电路

```
DAC(PA4) ──[R1=1kΩ]──┬──[R2=1kΩ]──┬──→ 输出
                      │             │
                   [C1=1nF]      [C2=1nF]
                      │             │
                     GND           GND

这是一个二阶RC低通滤波器。
fc = 1/(2π×R×C) ≈ 160kHz（每级）
二阶在200kHz处衰减约-12dB。

更好的方案：用运放做Sallen-Key二阶低通
  可以实现更陡峭的滚降和更平坦的通带。
  但简单RC滤波器在赛场上更快搭建。
```

### 去偏置电路

DAC输出范围0~3.3V，中间值1.65V对应0V信号。
需要去掉1.65V直流偏置，输出纯交流信号。

```
DAC滤波后 ──[C=10μF]──→ 纯交流输出
              │
           隔直电容
           
C=10μF时，对50Hz信号的阻抗 = 1/(2π×50×10μ) = 318Ω
对1kHz信号的阻抗 = 16Ω
对于高阻抗负载（示波器1MΩ），这个阻抗可以忽略。
但对于低阻抗负载，可能需要更大的电容。
```

## 五、已知模型电路（独立电路板）

题目给定传递函数H(s)，需要用RC有源滤波器实现。

假设H(s)是一个二阶低通滤波器（Sallen-Key拓扑）：

```
         R1        R2
Vin ──[===]──┬──[===]──┬──[─\
             │         │    >──┬──→ Vout
            [C1]      [C2]──[+/  │
             │         │         │
            GND       GND       [反馈]
                                 │
                                GND

参数由H(s)的系数决定：
  ω0 = 1/√(R1×R2×C1×C2)
  Q = √(R1×R2×C1×C2) / (C2×(R1+R2))
  
赛场上拿到具体的H(s)后，代入公式计算R1,R2,C1,C2。
建议提前准备好各种阻值的精密电阻和电容，现场组合。
```

## 六、PCB布局要点

本题用万能板手工焊接（4天时间不够画PCB）。

1. **信号线和电源线分开走**：信号线走板子上半部分，电源线走下半部分
2. **模拟地和数字地单点连接**：MCU的GND和运放的GND在一个点汇合
3. **ADC输入线尽量短**：PA0和PA1的走线越短越好，减少噪声耦合
4. **DDS输出线加屏蔽**：如果用杜邦线连接DDS模块，外面套一层铝箔
5. **运放电源引脚旁路电容**：每个运放的VCC和VEE引脚旁边焊100nF陶瓷电容
6. **BNC接口固定牢固**：BNC座焊在板子边缘，方便接线
