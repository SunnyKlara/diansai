## 非线性负载电路（基本要求第1项）

```
uS(24VAC) ──┬── D1 ──┬── D2 ──┬──[RL=20Ω]──┬── IO
             │        │        │             │
             │   KD开关│   D3   │    KC开关    │
             │        │        │     │        │
             │        └── D4 ──┘   [C=?]     │
             │                       │        │
             └───────────────────────┴────────┘

KD断开: 半波整流 (D1单管)
KD闭合: 桥式整流 (D1~D4全桥)
KC断开: 无电容滤波
KC闭合: 有电容滤波 (C=1000~10000μF)
```

4种组合产生4种不同的电流波形和谐波特征：
| KD | KC | 整流方式 | 电流波形 | 主要谐波 |
|----|-----|---------|----------|----------|
| 断开 | 断开 | 半波无滤波 | 半正弦脉冲 | 偶次+奇次 |
| 断开 | 闭合 | 半波有滤波 | 窄脉冲 | 偶次+奇次(更多) |
| 闭合 | 断开 | 全桥无滤波 | 全波脉动 | 奇次为主 |
| 闭合 | 闭合 | 全桥有滤波 | 窄脉冲 | 奇次(含量很高) |
