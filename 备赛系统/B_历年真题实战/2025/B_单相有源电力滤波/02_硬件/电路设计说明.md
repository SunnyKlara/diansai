## 模块 1：非线性负载电路（基本要求 ①，8 分）

```
                       KD
              ┌──────/   ────────┐
              │                  │
uS ─D1─┬──────┤                  ├──── RL (20Ω/10W) ─── IO
       │      │                  │
       │      └─D2/D3/D4─────────┘    ← 全桥
       │           │
       │           │ (KD 闭合时 D2/D3/D4 接入 → 全桥；断开时只 D1 单管 → 半波)
       │
   ────┴────────────── KC ──── C(1000~10000μF)
                       /
```

| KD | KC | 整流方式 | iL 波形 | 主要谐波 |
|---|---|---|---|---|
| 断 | 断 | 半波无滤波 | 正向半正弦脉冲 | 全部偶次 + 奇次 |
| 断 | 闭 | 半波有滤波 | 窄脉冲 | 偶次 + 奇次（更窄）|
| 闭 | 断 | 全桥无滤波 | 双向全波脉动 | 奇次为主 |
| 闭 | 闭 | 全桥有滤波 | 极窄脉冲 | 奇次（含量极高，最难补）|

**器件选择**：
- 整流二极管 **1N5408**（3A/1000V）：余量充足
- RL 取 **20Ω/10W**：满载 IO = 24V/20Ω ≈ 1.2A，配合 KC 后峰值脉冲电流可上 4A，符合"IO ≥ 2A"要求
- 滤波电容 1000/4700/10000 μF 三档准备，测试时切换

## 模块 2：电流互感器与电压调理（基本要求 ②③ 测量精度）

### 2.1 iL 电流采样链路

```
非线性负载主回路 ──┐
                  │
            ┌─────┴─────┐
            │ ZMCT103C  │  变比 1000:1，f₃dB = 10kHz
            │ (穿心)    │
            └─────┬─────┘
                  │ 取样电流 (iL/1000) × 100Ω = ±0.5V 双极性
                  ▼
            ┌──────────────────┐
            │ OPA2277 加偏置   │ Vin × 2.5 + 1.65V  （单电源 3.3V）
            │ + 增益 2.5       │
            └─────┬────────────┘
                  │ 0.4 ~ 2.9 V 单极性
                  ▼
            ADC1 IN1
```

**关键点**：
- 取样电阻 **R = 100Ω 1%** 必须用金属膜（碳膜 ±5% 不够）
- 加偏置必须用**精密基准源**（TL431 + 分压），不要直接 3.3V/2 分压（参考漂移 ≥ 5%）
- 运放选 **OPA2277**：失调 25uV，温漂 0.4uV/°C，远优于 LM358（5mV / 7uV/°C）
- PCB 走线：互感器 → 取样电阻 → 运放在板内**集中布**，长度 < 30mm，避免高频拾取干扰

### 2.2 uS 电压采样链路

```
uS = 24Vac  ──┐
              │
        ┌─────┴─────┐
        │ ZMPT101B  │  变比 1:1，f₃dB = 5kHz
        │ (隔离)    │
        └─────┬─────┘
              │
        ┌─────┴─────┐  R 分压（1MΩ : 100kΩ）
        │ 1/11 衰减 │  ±2.18V → ±0.198V
        └─────┬─────┘
              │
        OPA2277 加偏置 + 增益 → 0.4~2.9V → ADC1 IN0
```

### 2.3 iF（APF 输出电流）采样

iF 在 APF 内部取，用 **ACS712-05B**（霍尔，0.5A 量程精度版）：
- 输出 185 mV/A，零点 2.5V → 直接 ADC2 IN2
- 带宽 80 kHz（远超 7 次谐波 350 Hz）

## 模块 3：APF H 桥主电路（发挥项核心）

```
                   ┌──── M1 ────┐
                   │ (上管)     │
        ┌──────────┤            ├──── L = 1mH ──┬──── 接电网
        │  + 48V   │            │               │
   Vbus ┤   DC     │            │              ─┤─ C = 4.7μF
        │  -       │            │               │
        └──────────┤            ├──── (中点)   ─┤─ (回到电网中性线)
                   │ (下管)     │
                   └──── M2 ────┘
                   ┌──── M3 ────┐
                   │            │
                   │            │
                   ├────────────┤
                   │            │
                   │            │
                   └──── M4 ────┘
                   (反向桥臂，与 M1/M2 互补 PWM 实现单极性 PWM)
```

### 3.1 MOSFET 与驱动

| 元件 | 型号 | 关键参数 |
|---|---|---|
| M1~M4 | **IRF3205** ×4 | Vds 55V, Rds 8mΩ, Qg 146nC |
| 驱动 IC | **IR2110** ×2 | 自举上桥，关闭延迟 ~200ns |
| 自举二极管 | UF4007 | 快恢复 75ns |
| 自举电容 | 1μF / 50V CBB | |
| 栅极电阻 | **22Ω + 22Ω 反并 10V齐纳** | 限制栅极充放电速率 |
| **死区** | **500 ns** | TIM1 BDTR.DTG = 0x54 |

**布板要点**：
- 母线电容（470μF×2）和 H 桥之间走线 **< 20mm，宽 ≥ 2mm**
- 驱动 IC 紧贴 MOSFET 栅极，减小寄生电感
- 加 1nF 高频去耦在每个 MOSFET 漏极对地

### 3.2 输出 LC 滤波器

| 参数 | 值 | 计算依据 |
|---|---|---|
| L | **1 mH** / 5A 饱和 | 用 T130-26 铁粉芯 70 匝 0.8mm 漆包线 |
| C | **4.7 μF** / 100V CBB | X2 安规电容，禁用电解 |
| fc | 2.32 kHz | 1/(2π√LC) |
| 20 kHz 衰减 | 38 dB | 20·log(20/2.32)² |
| 350 Hz 处增益 | -1.4 dB | 7 次谐波几乎无衰减 |

**关键**：fc 必须远高于最大谐波频率（350Hz × 5 = 1.75kHz），又远低于开关频率（20kHz / 5 = 4kHz）。1mH/4.7μF 配合刚好。

## 模块 4：母线预充电（安全关键）

```
+48V ──┬─── 50Ω/10W ────┬─── 母线 ──→ APF
       │                │
       └─── K1（继电器） ─┘
        (软件控制：Vbus > 0.9·48V 时吸合)
```

- K1：松乐 SLA-05VDC-SL-A（10A 触点）
- 软件 ST_PRECHARGE 状态：超时 3 秒未到阈值 → ST_FAULT
- 应急：硬件加 TVS（SMBJ48A）抑制母线浪涌

## 模块 5：辅助电源链

| 输出 | 来源 | 方案 | 用途 |
|---|---|---|---|
| +12V | 48V | LM2596 模块 | 驱动 IC、风扇 |
| +5V  | +12V | AMS1117-5.0 | LCD 背光 |
| +3.3V | +5V | AMS1117-3.3 | MCU、运放、ADC |
| 1.65V 基准 | +3.3V | TL431 + 分压 | ADC 偏置基准 |

## 模块 6：保护电路

| 保护项 | 方案 | 触发后动作 |
|---|---|---|
| 母线过压 (>55V) | 软件 ADC 检测 | 立即 MOE=0，进入 FAULT |
| 母线欠压 (<38V) | 软件 ADC 检测 | 同上 |
| iF 过流 (>6A) | 硬件比较器 LM393 + 软件双重 | 立即 MOE=0 |
| MOSFET 过温 | NTC + 软件查表 | 进入降功率（限制 ramp）|

## 调试外接器材

- 数字示波器（≥ 100 MHz）：观察 uS、iL、iF、iS 四通道
- 频谱仪 / FFT 模式：测谐波 H₂..H₁₀
- 万用表：电压精度 0.1%
- 电流钳：交流 5A 量程

## 与 2023 A 的差异

| 维度 | 2023 A 单相逆变器 | 2025 B APF |
|---|---|---|
| 角色 | 电压源（直接驱动负载）| 电流源（注入电网）|
| 控制环 | 电压外环 + SPWM | 电流跟踪 PI |
| 滤波 | 输出 LC，含负载 | 输出 LC，与电网串联 |
| 保护 | 输出过压保护 | 母线 + iF 过流双重 |

代码层面：H 桥 PWM 驱动、ADC 同步采样、状态机骨架都可复用 2023 A，**只是控制律不同**。
