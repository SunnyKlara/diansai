# 2025 A 能量回馈变流器 —— 故障决策树

> ⚠️ 三相强电更危险，**所有动作前确认母线放完电（< 5V）**。

## 1. 上电类（最常翻车）

### 1.1 上电瞬间烧管
| 步 | 怀疑 | 修复 |
|---|---|---|
| 1 | 死区设错（< 500ns）| HRTIM DTG 重设 0x55 |
| 2 | 上下管 PWM 反向 | 检查 SysConfig 互补关系 |
| 3 | 自举电容失效 | 换 1μF CBB |
| 4 | BKIN 上电瞬间未拉低 | 加上拉电阻 |
| 5 | 直接 36V 上电 | **必须从 12V 开始升**|

### 1.2 三相 PWM 不对称
| 步 | 怀疑 | 修复 |
|---|---|---|
| 1 | HRTIM TIM_A/B/C 周期不一致 | 全部 = 5440000/20000 |
| 2 | 死区设置不一致 | A/B/C 都 DTG=0x55 |
| 3 | 触发源不同步 | Master + 其他 Slave 模式 |

## 2. SVPWM 类

### 2.1 输出三相不平衡
| 步 | 怀疑 | 修复 |
|---|---|---|
| 1 | SVPWM 算法 7 段 vs 5 段不一致 | 重写 svpwm_3phase.c |
| 2 | 调制比超 1.0 | 限幅到 0.95 |
| 3 | dq 旋转坐标系角度 θ 错 | PLL 验证锁相 |

### 2.2 同步整流后效率反而降
| 步 | 怀疑 | 修复 |
|---|---|---|
| 1 | 下管开通时机过晚（错过上管关）| 减小死区时间 |
| 2 | 下管开通时机过早（直通） | 加大死区或加比较器 |
| 3 | Rds_on 太大 | 换 IRFB7434 |
| 4 | 互感器相位偏 | 用 ZMCT103C 或 ACS758 |

## 3. 控制环类

### 3.1 母线电压不稳
| 怀疑 | 修复 |
|---|---|
| Vbus 外环 PI 振荡 | 减 KP_VBUS |
| 负载突变响应慢 | 加前馈 |
| 母线电容容量不够 | 加并 470μF |

### 3.2 PF < 0.95
| 怀疑 | 修复 |
|---|---|
| iq_ref ≠ 0 | 强制 iq_ref = 0 |
| dq 解耦项错 | 检查 ωL·iq 项 |
| PLL 锁相不准 | 加 LPF 平滑 ω |

## 4. 紧急操作

| 现象 | 立即 |
|---|---|
| 烟雾 / 火花 | 断电 + 离开 + 灭火器 |
| MOSFET > 70°C | 断电检查 |
| 母线异常升高 | BKIN 触发，硬件保护 |
| 三相输出异响 | 看是否是变压器铁芯共振，调 fsw |

## 5. 经验

**最大坑**：直接用 36V 上电导致烧 6 个 MOSFET 一齐报销。**严格 12V → 18V → 24V → 36V 渐进升压**！
