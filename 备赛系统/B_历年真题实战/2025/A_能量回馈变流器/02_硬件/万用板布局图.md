# 2025 A 能量回馈变流器 — 万用板布局图

> 配套：`电路设计说明.md` / `材料清单_BOM.md` / `接线表.md`  
> 板型：**3 板分开**：三相功率板（15×20cm）+ 控制板（7×9cm）+ 显示板（5×7cm）  
> **难度：电源天花板** —— 三相 SVPWM + 三相 V/I 采样 + 同步整流，必须严格分板

## 总体策略（能量回馈类核心）

```
┌─────────────────────────────────────────────────────────────────┐
│  策略 1：三板分离 —— 三相功率板 + 控制板 + 显示板               │
│  策略 2：三相桥臂物理对称布局 —— A/B/C 三相 MOS 占等距           │
│  策略 3：三相电感分离 —— 3 颗 1mH 磁芯，间距 ≥ 2cm 防互感         │
│  策略 4：VBUS 大电解集中放置 —— 4×470μF/100V 紧靠桥臂            │
│  策略 5：CAN 信令双机协调（赛中可降级单机模式）                  │
│  策略 6：测试点 12 个（每相 4 个 + 母线 4 个）                   │
└─────────────────────────────────────────────────────────────────┘
```

## 三相对称布局（关键）

```
        三相布局原则：
        ┌─────────┬─────────┬─────────┐
        │  A 相    │  B 相    │  C 相    │
        │  桥臂    │  桥臂    │  桥臂    │
        │  L_A    │  L_B    │  L_C    │
        │  C_A    │  C_B    │  C_C    │
        │  V_A 采样 │  V_B 采样 │  V_C 采样 │
        │  I_A 采样 │  I_B 采样 │  I_C 采样 │
        └─────────┴─────────┴─────────┘
        
为什么要对称：
- 三相不平衡度 < 1% 是基本要求
- 走线长度差 → 导通压降不同 → 三相不平衡
- 互感差 → 相电流相位偏差
```

## 散热（最严苛）

```
散热预算：
- 三相功率：100V × 5A × 95% × 3 相 = 1.4kW
- 损耗：~75W（5%）
- 4 个 MOS / 相 × 3 相 = 12 个 MOS，每个 ~6W

散热方案：
- 12 个 MOS 共享 1 块大铝散热片：200×100×40mm
- 必须加风扇：4cm × 4cm 或 6cm × 6cm，5V/12V
- 导热硅脂 + 螺丝紧固
- 散热片 + 桥臂用绝缘云母片隔离（防短路）

温度限：MOS 温度 < 80°C 工作
        > 80°C 自动降功率到 75%
        > 95°C 强制停机
```

## 三相 ADC 同步采样布局（关键）

```
6 个 ADC 调理电路必须等长走线 + 等距分布：

  V_A 调理 ─┐
  V_B 调理 ─┤
  V_C 调理 ─┤  ← 走线长度差 < 2mm
  I_A 调理 ─┤
  I_B 调理 ─┤
  I_C 调理 ─┘
       │
       ▼
   STM32G4 ADC1+ADC2+ADC3 同步触发
   （HRTIM TRGO 触发，6 路同时采）
```

## 接地策略（最复杂）

```
                  ┌──────────────────────────────────┐
                  │ DC 母线负极（PGND 起点）          │
                  └────────┬─────────────────────────┘
                           │
                           │ 1 处磁珠 / 0Ω 跳线
                           ▼
        ┌───────── ChassisGND（机箱 / 散热片）
        │
        │ 单点合并
        │
        ▼
   DGND（控制板）              AGND（控制板模拟地）
   - STM32 GND                 - 6 个 ADC 调理 GND
   - CAN GND                   - 用磁珠隔离 DGND
   - OLED GND
   
分离原因：
- HRTIM 开关频率 100kHz，单边沿 dV/dt > 5kV/μs
- 任何 PGND 与 DGND 之间的环路都会感应噪声
- ADC 必须用纯净的 AGND
```

## 走线规则（最严苛）

| 线 | 走法 |
|---|---|
| DC 母线（100V） | **2.5mm² 厚铜或者敷锡走线** |
| 三相输出 | 1.5mm² 硅胶线，等长（差异 < 2cm）|
| MOS 栅极 | 0.5mm² + 10~22Ω 串联防振铃 |
| HRTIM PWM | 6 根双绞线（每对 H/L 双绞），≤ 15cm |
| ADC 走线 | 0.2mm²，**等长** ±1mm |
| CAN | 双绞线，120Ω 终端电阻在两端 |

## 故障定位 × 布局（最难定位）

| 现象 | 处理 |
|---|---|
| 上电就跳保险 | 三相中至少一相直通；用万用表测 U/V/W ↔ N 电阻 |
| 三相不平衡 > 5% | TP9~12 测三相电流：哪相小就检查那相调理电路 |
| THD > 5% | 死区太大；TP1 PWM 测占空比，确认是否有死区抖动 |
| 输出失真大 | LC 滤波器电感互感太强；移开 L_A/L_B/L_C 间距 ≥ 3cm |
| 同步整流失败 | CAN 信令丢失；测 CAN_H/CAN_L 应是 ±2.5V 摆动 |
| 母线电压抖动 | 大电解失效或预充电缺失 |

## 测试点（12 个）

```
强电板（功率板）：
TP1 VBUS+      = DC 母线正（应 60V±2V）
TP2 VBUS-      = DC 母线负
TP3 V_A_OUT    = A 相输出 RMS（应 35V±0.05V）
TP4 V_B_OUT    = B 相输出 RMS
TP5 V_C_OUT    = C 相输出 RMS
TP6 I_A_OUT    = A 相输出电流 RMS
TP7 I_B_OUT    = B 相输出电流 RMS
TP8 I_C_OUT    = C 相输出电流 RMS

控制板（弱电）：
TP9 PWM_A_H    = HRTIM 输出
TP10 PWM_A_L
TP11 SYNC_TRG  = HRTIM TRGO 同步信号
TP12 DGND      = 数字地
```

## 安全（最严苛）

```
☐ 100V DC 母线 **绝对** 用接线柱，绝缘 1kV
☐ 4 颗 470μF 大电解 **必须** 装泄放电阻（10kΩ/2W），
   关机后 1 分钟内电压衰减到 < 50V
☐ MOS / 散热片之间用 **绝缘云母片**，杜绝散热片带电
☐ 全程戴绝缘手套 + 防护眼镜
☐ 调试时人 / 板 / 仪器都接地
☐ 测试电压前先万用表测电压**两次** 再触碰
☐ 异常情况立刻按 STOP（红色蘑菇头按钮）
```

## 备份策略

降级路线（如果三相板调不通）：

1. **退到单相 SPWM 模式**（关掉 B/C 相 PWM，只跑 A 相）
   - 仍能拿基本要求 50%
2. **关闭同步整流**（只做 SVPWM 输出，不接收能量回馈）
   - 简化两个核心算法之一
3. **降功率到 30V/1A**
   - 散热问题大幅缓解

## 物理装配

```
最终装配（赛场展示）：
- 三相功率板 +散热片 + 风扇  → 用 4 颗 M3 螺丝固定在机箱底
- 控制板 → 焊在亚克力支架上，立着放在功率板侧面
- 显示板 → 装在机箱前面板（赛评直接看到 OLED 数据）
- 电源 / 输出端子 → 全部走机箱前面板
- CAN 接口 → 后面板（接另一台机）
```
