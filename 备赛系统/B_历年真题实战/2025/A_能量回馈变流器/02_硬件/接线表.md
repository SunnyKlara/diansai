# 2025 A 能量回馈变流器 —— 接线表

> ⚠️ **三相强电**：母线 ±100V DC + 三相输出 28Vac × 3，**比 2023A 危险 2 倍**！  
> 必须：调压器渐进升压 / 戴绝缘手套 / 全程双人操作（一人操作一人监控）。

## 总图
```
[36V DC 电源 + 限流]
        │
        ↓ 50Ω 预充 + 继电器
[母线 ±100V (470μF×2 串联)]
        │
        ├──┬─── 三相全桥 (M1~M6) ───┬──→ A 相 ─→ L+C ─→ 输出 A
        │  │                         │
        │  ├── 同步整流 (M7~M12) ─┼──→ B 相 ─→ L+C ─→ 输出 B  
        │  │                         │
        │  └─────────────────────────┴──→ C 相 ─→ L+C ─→ 输出 C
        │
[电源回馈：能量从负载流回母线 → 36V 电源]
```

## 1. 电源域

| # | 源 | 目的 | 颜色 | 长度 |
|---|---|---|---|---|
| P1 | 36V+ → 50Ω → 母线 + | 母线正极 | 红粗 | 5cm |
| P2 | 36V- → 母线 - | 母线负极 / 共地 | 黑粗 | 5cm |
| P3 | KMR1 触点短接 50Ω | 预充结束 | 红 | 3cm |
| P4 | 母线放电电阻 10kΩ/5W | 跨母线 | 红 | 6cm |

## 2. 三相 H 桥 MOSFET

| # | 元件 | 连接 | 备注 |
|---|---|---|---|
| HA1 | M1 (A 上管) | 漏极→母线+，源极→A 相中点，栅极→IR2110 #1 HO | |
| HA2 | M2 (A 下管) | 漏极→A 相中点，源极→母线-，栅极→IR2110 #1 LO | |
| HB1 | M3 (B 上管) | 同 M1 但接 B 相 | |
| HB2 | M4 (B 下管) | 同 M2 | |
| HC1 | M5 (C 上管) | 同 M1 但接 C 相 | |
| HC2 | M6 (C 下管) | 同 M2 | |

总共 6 个 IR2110（每相 2 个）+ 6 个 IRFB7434 / IRF3205。

## 3. 同步整流（关键 30 分）

> 要点：上下管的 PWM 互补，关比开早 200ns（防直通）。

| # | 内容 | 备注 |
|---|---|---|
| SR1 | M2/M4/M6 下管 PWM | 与上管完全互补，但**死区不对称**：关下管早 200ns |
| SR2 | 控制方式 | HRTIM 自带死区生成，DTG = 0x55 |

## 4. 输出 LC 滤波

| # | 元件 | 值 |
|---|---|---|
| LA | A 相输出电感 | 0.5mH / 5A，**T106-26 70 匝 0.8mm** |
| LB | B 相输出电感 | 同 LA |
| LC | C 相输出电感 | 同 LA |
| CA/CB/CC | 各相输出电容 | 4.7μF / 100V CBB X2 安规 |

## 6. 安全检查

- [ ] 母线电容串联（**禁止单 100V 电容直接挂母线**！）
- [ ] 6 个 MOSFET 全部装散热片
- [ ] BKIN 比较器阈值 = 4A 立即停 PWM
- [ ] 母线放电电阻常驻
- [ ] 所有电容焊点过振动测试

## 7. 接线检查

- [ ] 三相 PWM 时序正确（A → B → C 120° 相位差）
- [ ] 死区 500ns（先示波器看再上电）
- [ ] 同步整流时序：下管关比上管开早 200ns
- [ ] 自举电容 ≥ 1μF / 50V CBB（每相一个）
- [ ] 三相输出对称（万用表测电压差 < 5%）
