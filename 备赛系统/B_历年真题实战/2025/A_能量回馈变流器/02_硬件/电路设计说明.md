# 2025A 电路设计说明

## 一、三相全桥逆变器主电路

```
        Vdc+ (48V)
         │
    ┌────┼────┬────┐
    │    │    │    │
  [Q1] [Q3] [Q5]  │
    │    │    │    │
    ├─a  ├─b  ├─c  │    a,b,c为三相输出
    │    │    │    │
  [Q2] [Q4] [Q6]  │
    │    │    │    │
    └────┼────┴────┘
         │
        Vdc- (GND)

Q1/Q2: A相桥臂 (IR2110_1驱动)
Q3/Q4: B相桥臂 (IR2110_2驱动)
Q5/Q6: C相桥臂 (IR2110_3驱动)
全部使用IRF3205 N-MOSFET
```

## 二、三相LC滤波器

```
a ──[La=1.5mH]──┬── Va_out
                 │
              [Ca=4.7μF]
                 │
                 ├── 星形连接中性点N
                 │
b ──[Lb=1.5mH]──┬── Vb_out
                 │
              [Cb=4.7μF]
                 │
                 ├── N
                 │
c ──[Lc=1.5mH]──┬── Vc_out
                 │
              [Cc=4.7μF]
                 │
                 └── N
```

**三相参数必须一致！** 电感用同一批磁芯、同样匝数绕制。

### 滤波器参数计算
- 截止频率 fc = 1/(2π√(LC)) = 1/(2π√(1.5e-3 × 4.7e-6)) = 1896Hz
- 开关频率 fsw = 20kHz
- 衰减比 = (fsw/fc)² = (20000/1896)² ≈ 111 = 41dB
- 足够滤除开关频率纹波

### 低频(20Hz)注意事项
- 20Hz时电感电流纹波更大（因为占空比变化更慢）
- 需要确认电感在最大电流时不饱和
- T106-26磁芯饱和电流约3A，对2A输出有余量

## 三、IR2110驱动电路（×3套）

每套驱动一个桥臂（高侧+低侧）：
```
STM32 TIM1_CHx  ──→ IR2110 HIN ──→ HO ──→ Q_高侧栅极
STM32 TIM1_CHxN ──→ IR2110 LIN ──→ LO ──→ Q_低侧栅极

VCC = 12V (独立12V电源)
VDD = 3.3V
自举: VB ──[C_boot 1μF]── VS
      VB ──[D_boot UF4007]── VCC
```

## 四、采样电路

### 三相电压采样（每相一路）
```
Va_out ──[100kΩ]──┬──[10kΩ]── 1.65V偏置
                   │
                   └── ADC (PA0)

分压比 1:11，偏置1.65V使交流信号在0~3.3V范围内
```

### 三相电流采样
- 每相一个ACS712-5A模块
- 输出2.5V±185mV/A
- 直接接ADC

## 五、变流器2（整流回馈，发挥部分）

### 方案：三相不控整流+Buck
```
三相输出 → [三相整流桥 MDS100A] → [Buck DC-DC] → 回馈到Vdc母线
```

- 整流桥输出电压 ≈ 1.35 × Vline = 1.35 × 32 = 43.2V
- Buck降压到Vdc(48V以下)回馈
- 或直接整流后并联到母线（如果电压匹配）

### 连接单元
- 可能需要变压器匹配电压
- 或直接连接（如果整流输出电压接近母线电压）
- 具体方案需要根据实际电压关系确定

## 六、PCB布局要点

1. **三相对称布局**：三个桥臂的走线长度和布局尽量一致
2. **母线电容靠近桥臂**：减小寄生电感
3. **驱动信号短走线**：IR2110到MOSFET栅极≤3cm
4. **采样点在滤波器输出端**：不要在桥臂输出端采样
5. **数字地和功率地分开**：单点连接
