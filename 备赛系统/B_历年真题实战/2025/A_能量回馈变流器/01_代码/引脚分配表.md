## 2. SVPWM 时序

| 项 | 值 |
|---|---|
| 频率 fsw | 20 kHz |
| 周期 T | 50 μs |
| 中心对齐 | 7 段式 |
| 死区 | 500 ns（DTG = 0x55） |
| 同步整流死区 | 200 ns（关比开早）|

## 3. 三 ADC 同步双模式

```
ADC1 + ADC2 + ADC3 都跑同一个 HRTIM 触发
=> 7 路同步采样：Ia/Ib/Ic + Va/Vb/Vc + Vbus
```

| 项 | 值 |
|---|---|
| 触发 | HRTIM TRGO（PWM 中点）|
| DMA | DMA1 Stream 0/1/2 |
| 缓冲 | 7 通道 × 2048 点 ring buffer |

## 4. 关键中断

| 中断 | 优先级 |
|---|---|
| HRTIM_TRGO 20kHz | 0 |
| TIM1_BRK BKIN | 0 |
| ADC1 EOS | 1 |
| TIM6 1ms tick | 2 |
| EXTI13 | 3 |
| LPUART1 | 4 |
