# 2025D 代码工程说明

## 文件结构
```
01_代码/
├── main.c           主程序（模式切换、按键、显示、校准流程）
```

## 引脚分配

| 引脚 | 功能 | 说明 |
|------|------|------|
| PB0~PB7 | GPIO_OUT | 端口1 RJ45 8根线（输出） |
| PC0~PC7 | GPIO_IN(上拉) | 端口2 RJ45 8根线（输入） |
| PB8 | GPIO_OUT | 端口1外壳（SFTP检测） |
| PC8 | GPIO_IN | 端口2外壳 |
| PA4 | SPI1_NSS | TDC7200片选 |
| PA5 | SPI1_SCK | TDC7200时钟 |
| PA6 | SPI1_MISO | TDC7200数据输出 |
| PA7 | SPI1_MOSI | TDC7200数据输入 |
| PA3 | GPIO_OUT | TDR脉冲发射 |
| PA2 | GPIO_IN | TDR反射接收(TDC STOP) |
| PA0 | ADC1_IN0 | 电阻测量 |
| PA8 | GPIO_IN(上拉) | 模式切换按键 |
| PA9 | GPIO_IN(上拉) | 启动测量按键 |
| SPI2 | - | LCD显示屏 |

## 测量流程

### 双端检测流程
```
1. 线序检测（GPIO扫描，<10ms）
2. 类型识别（外壳导通检测，<1ms）
3. 电阻测量（恒流源+ADC，<100ms）
4. 显示结果
总时间 < 1s
```

### 单端检测流程
```
1. TDR测距（发脉冲+TDC测时间，×10次取平均，<2s）
2. 短路检测（GPIO扫描，<10ms）
3. 如有短路：TDR测短路位置（<2s）
4. 显示结果
总时间 < 5s
```

## 校准说明
- 开机时需要用已知长度(1m)的SFTP线缆校准
- 校准的是双绞线中的传播速度
- 校准后速度值保存在RAM中（掉电丢失）
- 赛场提供1m SFTP和50m UTP用于校准
