# 2025 D 简易以太网双绞线测试仪 —— 电路设计

## 系统总图
```
RJ45 输入 ─→ [CD4051 8 选 1 模拟开关 ×2] ─→ [TDR 阶跃源 + ADC]
                  ↑                                    ↓
              [STM32 GPIO 选 4 对]               [波形分析]
                                                        ↓
                                                    [OLED 显示]
```

## 关键
- 4 对线轮流测：CD4051 切换
- 每对都做 TDR：阶跃 + 1MSPS ADC + 等效采样
- 数据率：每对 100ms 测完，4 对 < 1s 完成
