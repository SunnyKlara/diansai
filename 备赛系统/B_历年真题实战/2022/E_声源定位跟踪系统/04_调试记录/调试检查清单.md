# 2022E 调试检查清单

## Day1 搭建
- [ ] 麦克风阵列安装（L形或三角形）
- [ ] 麦克风前置放大电路焊接
- [ ] STM32F407多通道ADC配置
- [ ] 云台+激光笔安装
- [ ] LCD显示正常
- [ ] 声源装置制作（蜂鸣器+电池+开关）

## Day2 调试

### 麦克风采样
- [ ] 3路麦克风同步采样（DMA）
- [ ] 采样率≥44.1kHz
- [ ] 声源发声时波形正常

### TDOA时延估计
- [ ] 互相关算法运行正确
- [ ] 两路信号的时延估计合理
- [ ] 声源在正前方时时延≈0

### 定位计算
- [ ] 声源放在已知位置，计算结果接近
- [ ] 距离γ误差测试
- [ ] 角度θ误差测试

### 激光指向
- [ ] 根据定位结果计算云台角度
- [ ] 激光笔指向声源方向 ✓/✗
- [ ] 光点与声源偏差=___cm

## Day3 优化

### 精度优化
- [ ] 多个位置测试定位精度
- [ ] γ误差越小越好
- [ ] θ误差越小越好

### 动态跟踪
- [ ] 声源以0.2m/s移动
- [ ] 激光笔能跟随 ✓/✗
- [ ] 跟踪延迟=___s

## Day4 最终
- [ ] 定位精度达标
- [ ] 激光指向精度达标
- [ ] 动态跟踪（如果完成）
- [ ] 报告打印

## 常见故障

| 问题 | 原因 | 解决 |
|------|------|------|
| 时延估计不准 | 采样不同步 | 确保DMA多通道同时触发 |
| 定位偏差大 | 麦克风位置不准 | 精确测量麦克风坐标 |
| 激光指向偏 | 角度计算错误 | 检查坐标变换公式 |
| 环境噪声干扰 | 背景噪声太大 | 用GCC-PHAT抗噪声 |
| 回声干扰 | 室内反射 | 加吸音材料或算法滤除 |
