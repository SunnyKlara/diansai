## 核心技术：TDOA声源定位

### 原理
```
声源发出声音 → 到达各麦克风的时间不同 → 时间差(TDOA) → 计算位置

至少需要3个麦克风：
  mic1在位置(x1,y1)，到达时间t1
  mic2在位置(x2,y2)，到达时间t2
  mic3在位置(x3,y3)，到达时间t3
  
  Δt12 = t1 - t2 → 声源在以mic1和mic2为焦点的双曲线上
  Δt13 = t1 - t3 → 声源在以mic1和mic3为焦点的双曲线上
  两条双曲线的交点 = 声源位置
```

### 时延估计：互相关法
```c
// GCC-PHAT (广义互相关-相位变换)
// 比普通互相关抗噪声能力更强

void gcc_phat(float* sig1, float* sig2, int N, float* correlation)
{
    // 1. FFT
    arm_cfft_f32(&fft_inst, sig1_complex, 0, 1);
    arm_cfft_f32(&fft_inst, sig2_complex, 0, 1);
    
    // 2. 互功率谱
    for (int i = 0; i < N; i++) {
        // G12 = X1 × conj(X2)
        float re = sig1_complex[2*i]*sig2_complex[2*i] + sig1_complex[2*i+1]*sig2_complex[2*i+1];
        float im = sig1_complex[2*i+1]*sig2_complex[2*i] - sig1_complex[2*i]*sig2_complex[2*i+1];
        
        // PHAT加权：除以幅度
        float mag = sqrtf(re*re + im*im);
        if (mag > 1e-10f) {
            correlation[2*i] = re / mag;
            correlation[2*i+1] = im / mag;
        }
    }
    
    // 3. IFFT
    arm_cfft_f32(&fft_inst, correlation, 1, 1);  // 逆变换
    
    // 4. 找峰值位置 = 时延(采样点数)
}
```

### 位置计算
```c
// 已知3个麦克风位置和两组时延差
// 解非线性方程组（最小二乘法）

typedef struct {
    float x, y;
} Point_t;

Point_t locate_source(float tdoa_12, float tdoa_13,
                       Point_t mic1, Point_t mic2, Point_t mic3,
                       float sound_speed)
{
    // 距离差
    float d12 = tdoa_12 * sound_speed;
    float d13 = tdoa_13 * sound_speed;
    
    // 线性化求解（Chan算法或Taylor展开）
    // 这里用简化的网格搜索法
    Point_t best = {0, 0};
    float min_error = 1e10f;
    
    for (float x = -150; x <= 150; x += 1) {
        for (float y = 50; y <= 350; y += 1) {
            float r1 = sqrtf((x-mic1.x)*(x-mic1.x) + (y-mic1.y)*(y-mic1.y));
            float r2 = sqrtf((x-mic2.x)*(x-mic2.x) + (y-mic2.y)*(y-mic2.y));
            float r3 = sqrtf((x-mic3.x)*(x-mic3.x) + (y-mic3.y)*(y-mic3.y));
            
            float err = fabsf((r1-r2) - d12) + fabsf((r1-r3) - d13);
            if (err < min_error) {
                min_error = err;
                best.x = x;
                best.y = y;
            }
        }
    }
    return best;
}
```

## 关键提醒
- 麦克风阵列的几何布局影响定位精度
- 推荐L形或三角形布局
- 采样率≥44.1kHz，多通道同步采样
- 声速约340m/s，受温度影响
- 环境噪声和回声是最大干扰源
