# 2022 C 小车跟随行驶系统 —— 电路设计说明

## 系统总图（×2 套，领头/跟随）
```
[MSP432P401R] ─PWM→ [TB6612] → [JGA25-370 ×4]
       ↑                ↑
[5×IR 红外循迹]      [编码器×2]
       ↑
[HC-SR04 超声波] (跟随车前向，0~3m 范围)
       ↑
[NRF24L01+PA 心跳 50ms]
       ↑
[起动按键 + LED + 蜂鸣器]
```

## 关键

- 领头车：循迹 + 速度规划 + 等停检测，**有按键启停**
- 跟随车：循迹 + 间距 PID + 跟领头车节奏，**只有上电开关**
- 间距 20cm ± 6cm

## 方向决策与超车
- 外圈跟随：跟随车保持 20cm 距离
- 内圈超车：跟随车走更短的内圈，跟领头车异线

## 复用
- motor / encoder / ir_line 与 2021 F 完全一致
- 间距 PID 从 0 开始（独有）
