# 2022 C 小车跟随行驶系统 —— 接线表

> 双车系统：领头 / 跟随 各 1 辆，**仅跟随车**装超声波。

## 总图（领头 / 跟随通用骨架）
```
              [4WD 底盘]
              ┌──────────────────────┐
              │ ┌──┐  ┌──────────┐  │
              │ │L1│  │ MSP432    │  │
              │ └──┘  └────┬──────┘  │
              │            │         │
              │   [TB6612] [NRF24]   │
              │       │              │
              │   [4 电机]            │
              │ ┌──┐                ┌──┐
              │ │L2│  [电池 2S]    │R2│
              │ └──┘                └──┘
              └──────────────────────┘
              [HC-SR04（仅跟随）] [5 路 IR]
```

## 1. 电源

| # | 源 | 目的 | 颜色 | 长度 |
|---|---|---|---|---|
| P1 | 电池 + (7.4V) | TB6612 VM | 红粗 | 8cm |
| P2 | 电池 - | TB6612 GND | 黑粗 | 8cm |
| P3 | TB6612 5V out | MSP432 VCC | 红 | 5cm |
| P4 | MSP432 3.3V | NRF24 VCC | 红 | 6cm |
| P5 | MSP432 3.3V | OLED VCC | 红 | 8cm |
| P6 | MSP432 5V | HC-SR04 VCC（跟随车）| 红 | 12cm |

## 4. 5 路 IR

| # | MSP432 | IR | 颜色 |
|---|---|---|---|
| I1~I5 | P5.4/5.5/5.6/5.7/P4.7 | IR1~IR5 OUT | 紫蓝绿黄灰 |

## 5. HC-SR04（仅跟随车装）

| # | MSP432 | HC-SR04 | 颜色 | 备注 |
|---|---|---|---|---|
| U1 | P6.4 | TRIG | 黄 | 5V GPIO 输出 |
| U2 | P6.5 | ECHO | 蓝 | 5V → 3.3V 加分压（10k+20k）|

> **HC-SR04 是 5V 模块**，ECHO 输出 5V，必须分压到 3.3V 喂给 MSP432，否则烧 GPIO。

## 6. NRF24

| # | MSP432 | NRF24 | 颜色 |
|---|---|---|---|
| N1 | P1.5 | SCK | 蓝 |
| N2 | P1.6 | MOSI | 橙 |
| N3 | P1.7 | MISO | 黄 |
| N4 | P5.2 | CSN | 白 |
| N5 | P5.3 | CE | 紫 |

## 7. OLED

| # | MSP432 | OLED |
|---|---|---|
| O1 | P6.5 | SCL（与 ECHO 冲突，看实际选其一）|
| O2 | P6.4 | SDA（与 TRIG 冲突）|

> **冲突解决**：OLED 仅在领头车上装；跟随车专心做超声波。

## 8. 接线检查

- [ ] 跟随车：HC-SR04 ECHO 必须分压到 3.3V
- [ ] 双车：NRF24 channel 一致（76）
- [ ] 5 路 IR 阵列居中
- [ ] 跟随车前向超声波装在车头中心，**距地 5 cm**

## 9. 双车区分

| | 领头 (LEAD) | 跟随 (FOLLOW) |
|---|---|---|
| 编译 | `-DCAR_LEAD` | `-DCAR_FOLLOW` |
| HC-SR04 | 不装 | 装 |
| OLED | 装 | 可不装 |
| LED | RED | GREEN |
