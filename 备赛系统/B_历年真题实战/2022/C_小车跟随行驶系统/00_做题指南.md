## 系统架构

```
┌──────────────┐     NRF24L01     ┌──────────────┐
│  领头小车     │ ←──无线通信──→  │  跟随小车     │
│  TI MCU      │                  │  TI MCU      │
│  循迹+测速   │                  │  循迹+测速   │
│  启动按键    │                  │  仅上电开关   │
│  速度控制    │                  │  间距控制    │
│  等停检测    │                  │  跟随/超车   │
└──────────────┘                  └──────────────┘
```

### 关键约束
- 跟随小车**只有上电开关**，无其他按键
- 领头小车一键启动控制全程
- 两车间距保持20cm±6cm
- 不得有车外遥控

## 核心技术

### 1. 间距保持（20cm）
- **超声波测距**：HC-SR04测量两车距离
- **PID控制**：误差=20cm-实际距离 → 调节跟随车速度
- 弯道时距离会变化，需要滤波

### 2. 双车通信协议
```c
// 领头车发送的数据包
typedef struct {
    uint8_t header;      // 0xAA
    uint8_t cmd;         // 命令: START/STOP/SPEED/OVERTAKE
    int16_t speed;       // 当前速度 (mm/s)
    int16_t position;    // 当前位置 (编码器计数)
    uint8_t lap_count;   // 当前圈数
    uint8_t track_type;  // 内圈/外圈
    uint8_t checksum;
} LeaderPacket_t;

// 每50ms发送一次
```

### 3. 超车策略
- 领头车走外圈，跟随车走内圈（路径更短→超车）
- 内圈BFD段：小车发出灯光指示
- 超车后角色互换
- 需要精确的路径切换时机

### 4. 等停检测
- 路径上有"等停标识"（两条平行横线，间隔5cm）
- 领头车检测到后停车5s
- 跟随车也要同步停车（通过通信或自主检测）

## 场地制作
- 白纸铺设，黑色胶带(1cm宽)画路径
- 外圈：圆角矩形，直线段+圆弧弯道
- 内圈：外圈内侧的较短路径
- A点标志线：2cm宽×5cm长黑色横线
- 等停标识：两条2cm×10cm平行横线，间隔5cm
