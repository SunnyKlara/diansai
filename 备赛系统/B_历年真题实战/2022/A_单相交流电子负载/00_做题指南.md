## 核心原理

### 什么是交流电子负载？
- 传统负载（电阻）只能消耗能量
- 电子负载可以**模拟**不同类型的负载特性
- 通过控制输入电流的幅度和相位来模拟R/L/C负载

### 功率因数模拟
```
电阻性: 电流与电压同相      cosφ = 1.0
电感性: 电流滞后电压        cosφ < 1.0 (滞后)
电容性: 电流超前电压        cosφ < 1.0 (超前)
```

### 实现方法：PWM整流器
- 用全桥H桥作为可控整流器
- 通过控制开关管的导通时序，控制输入电流波形
- 电流波形 = 设定幅度 × sin(ωt + φ)
- φ>0: 电流超前→电容性
- φ<0: 电流滞后→电感性
- φ=0: 电流同相→电阻性

### dq坐标变换法
```
id = I × cosφ  (有功电流分量)
iq = I × sinφ  (无功电流分量)

设定cosφ → 计算id和iq → dq逆变换 → 电流参考波形 → 电流跟踪控制
```

## 难点分析

1. **功率因数精确可调**：需要精确的相位控制
2. **电流跟踪精度**：PI控制器带宽要足够
3. **能量回馈**：ΔP越小=回馈效率越高
4. **容性/感性切换**：需要平滑过渡

## 与2025B的关系
- 2022A是"模拟负载"，2025B是"补偿谐波"
- 都用到H桥+电流控制
- 2022A控制基波相位，2025B控制谐波分量
- 掌握2022A的电流控制技术对2025B很有帮助
