# 2022D 蔡氏电路设计 —— 深度工程分析

> 这不是代码文件，而是比代码更重要的设计文档。
> 混沌电路的难点不在编程，而在电路参数的精确设计。
> MCU只负责切换参数，真正的信号产生完全由模拟电路完成。

## 一、为什么蔡氏电路能产生混沌？

### 混沌的数学本质
蔡氏电路的状态方程：
```
C1 × dV1/dt = (V2 - V1)/R - g(V1)
C2 × dV2/dt = (V1 - V2)/R + iL
L  × diL/dt = -V2
```
其中 g(V1) 是蔡氏二极管的非线性特性（分段线性负阻）。

这是一个**三阶非线性自治系统**。混沌产生的条件：
1. 至少三阶（三个状态变量：V1, V2, iL）
2. 至少一个非线性元件（蔡氏二极管）
3. 参数在特定范围内

### 关键洞察：R是唯一的"调谐旋钮"
改变R的值，系统会经历：
```
R大 → 稳定平衡点（不振荡）
R减小 → Hopf分岔 → 单周期振荡
继续减小 → 倍周期分岔 → 双周期
继续减小 → 四周期 → 八周期 → ... → 混沌
继续减小 → 单涡旋混沌
继续减小 → 双涡旋混沌（经过"混沌合并"）
R太小 → 又变成周期振荡（混沌窗口之外）
```

**这就是题目要求的5种相图的产生路径！**
不需要改变电路拓扑，只需要精确调节R。

## 二、参数设计（这是最关键的部分）

### 经典蔡氏电路参数（低频版，kHz级）

| 元件 | 值 | 说明 |
|------|-----|------|
| C1 | 10nF | 决定频率和分岔行为 |
| C2 | 100nF | C2/C1的比值影响混沌类型 |
| L | 18mH | 与C1C2共同决定频率 |
| R | 1.2kΩ~2.0kΩ | **可调！这是唯一的控制参数** |

### C2/C1比值的深层含义

**这是很多参考资料不会告诉你的：**

C2/C1 = β，这个比值决定了混沌的"丰富程度"：
- β = 7~10：经典蔡氏混沌，倍周期路线完整
- β < 5：混沌窗口很窄，难以观察到完整的倍周期序列
- β > 15：混沌区域很宽，但单周期和双周期区域很窄

**我的选择**：β = 10（C1=10nF, C2=100nF）
理由：倍周期路线最完整，5种相图都容易找到。

### 蔡氏二极管的参数

蔡氏二极管的V-I特性：
```
        i
        │    /
        │   / m0 (内区斜率，负值)
        │  /
   ─────┼─/────────── V
        │/  Bp (断点电压)
       /│
      / │ m1 (外区斜率，负值)
     /  │
```

关键参数：
- m0 = -1/R0_inner ≈ -0.5mS（内区负导纳）
- m1 = -1/R0_outer ≈ -0.8mS（外区负导纳）
- Bp ≈ 1V（断点电压）

### 用运放实现蔡氏二极管

```
经典双运放实现：

         R3(220Ω)
Vin ──[R1(3.3kΩ)]──┬──[─\
                    │    >──┬── 
             ┌──────┤──[+/  │
             │      │       │
            [R2     [R4     [R5
            (22kΩ)] (22kΩ)] (2.2kΩ)]
             │      │       │
            GND   -Vsat   +Vsat

第二个运放结构相同但参数不同，两个并联。

m0 = -1/R1 + 1/R3 = -1/3300 + 1/220 = -0.303mS + 4.545mS
等等...这个计算很容易搞错。
```

**更好的方法：用仿真确定参数！**

在LTspice中搭建蔡氏电路，扫描R值，观察分岔图。
确认5种相图对应的R值范围后，再焊实际电路。

## 三、R值的精确控制方案

### 方案A：手动电位器（最简单但不可靠）
- 10kΩ多圈电位器
- 手动旋转切换5种模式
- 问题：比赛时手抖，很难精确定位

### 方案B：数字电位器（推荐）
- AD5206：6通道，256步，10kΩ
- SPI接口，MCU控制
- 分辨率：10000/256 = 39Ω/步
- 问题：39Ω的步进可能不够精细（某些分岔点对R非常敏感）

### 方案C：数字电位器 + 固定电阻串并联（最优）
```
R_total = R_fixed + R_digital

R_fixed选在混沌区域的中心值（约1.5kΩ）
R_digital用AD5206的一个通道（0~2kΩ范围）
总范围：1.5kΩ~3.5kΩ，分辨率约8Ω/步

这样在关键的分岔区域有足够的分辨率。
```

### MCU控制代码
```c
// 5种模式对应的R值（需要实际标定！）
// 这些值是通过仿真+实测确定的
typedef struct {
    const char* name;
    uint8_t digi_pot_value;  // AD5206设定值(0~255)
    float r_total_ohm;       // 对应的总电阻
} ChuaMode_t;

const ChuaMode_t modes[5] = {
    {"Period-1",  180, 1920},  // 单周期：R较大
    {"Period-2",  150, 1800},  // 双周期：R稍小
    {"Period-3",  140, 1760},  // 三周期：R更小（窗口很窄！）
    {"Scroll-1",  100, 1600},  // 单涡旋混沌
    {"Scroll-2",   60, 1440},  // 双涡旋混沌：R最小
};

// 注意：三周期的窗口非常窄（可能只有几Ω宽）！
// 这是最难调的一种模式。
// 如果数字电位器分辨率不够，可能需要手动微调。
```

## 四、40MHz带宽的挑战（发挥部分30分）

### 为什么低频蔡氏电路只有kHz级带宽？
- 频率 ≈ 1/(2π√(LC)) 
- L=18mH, C=10nF → f ≈ 12kHz
- 混沌信号的带宽约为基频的10~50倍 → 约120kHz~600kHz
- 远远不到40MHz

### 提高到40MHz需要什么？
- f = 40MHz → LC = 1/(2π×40M)² = 1.6×10⁻¹⁷
- 如果C1=1pF → L = 16nH（这已经是PCB走线的寄生电感量级！）
- 运放GBW需要 > 200MHz（AD8065的GBW=145MHz，勉强）

### 现实的方案
1. **用分立晶体管替代运放**实现蔡氏二极管
   - 晶体管的fT可达几GHz
   - 但电路设计和调试难度大幅增加
   
2. **用Colpitts振荡器的混沌版本**
   - Colpitts振荡器天然工作在高频
   - 通过调节偏置进入混沌状态
   - 更适合40MHz目标

3. **折中：先做低频版拿基本分，高频版作为加分项**

## 五、无电感方案（发挥部分20分）

### 回转器（Gyrator）原理
用运放+电容模拟电感：
```
        C_gyrator
Vin ──[R_g]──┬──[─\
              │    >──┬── Vout
       ┌──────┤──[+/  │
       │      │       │
      [C_g]  GND     [R_g2]
       │              │
      GND            GND

等效电感 L_eq = R_g × R_g2 × C_g
```

例：R_g=R_g2=10kΩ, C_g=180pF → L_eq = 10k×10k×180p = 18mH ✓

**注意**：回转器需要2个运放，加上蔡氏二极管的2个，总共4个。
题目要求≤6个运放，还有2个余量（可以用于缓冲或其他功能）。
