## 物理本质

### 蔡氏电路（Chua's circuit）
最经典混沌振荡器。R + 2 C + L + 蔡氏二极管 → 双涡卷吸引子。

### 数值仿真 vs 模拟实现
**数值方案**（推荐）：MCU 跑 RK4 积分 → DAC 输出 X/Y。优势：参数全数字，易切系统。  
**模拟方案**：用 OPA + ZNR 二极管搭电路。劣势：参数靠物理元件精度，难调节。

## 方案选择
**数值方案**（与 2024F 磁悬浮一样的"实时仿真"思路）：
- MCU：STM32F407 168MHz
- 控制率：50 kSPS（覆盖混沌频谱 1~10kHz）
- 输出：DAC1+DAC2 双通道 → 示波器 X-Y 模式

## 蔡氏方程（无量纲）
```
x' = α(y - x - g(x))
y' = x - y + z
z' = -β y - γ z
g(x) = m1·x + 0.5·(m0-m1)·(|x+1| - |x-1|)
α=15.6, β=28, γ=0, m0=-1.143, m1=-0.714  → 双涡卷
```
