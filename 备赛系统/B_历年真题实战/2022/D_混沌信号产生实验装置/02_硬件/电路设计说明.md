# 2022 D 混沌信号产生实验装置 —— 电路设计

## 系统总图
```
[STM32F407 168MHz]
   ├── DAC1 ──→ [LM6172 缓冲] ──→ X 通道（示波器 X-Y 模式）
   └── DAC2 ──→ [LM6172 缓冲] ──→ Y 通道
   
   主循环：50 kSPS 数值 RK4 积分蔡氏方程 → DAC 输出
```

## 关键
- 数值实现 vs 模拟实现：数值实现可调参数全在软件，方便切多种混沌系统
- DAC 50 kSPS 足够覆盖混沌信号 1~10kHz 频谱
- LM6172 高速运放（GBW=100MHz）保证 DAC 输出无失真
