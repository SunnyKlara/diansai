## 核心电路：蔡氏电路(Chua's Circuit)

```
     ┌──[L]──┬──[R]──┬──[NR(蔡氏二极管)]──┐
     │       │       │                     │
    [C2]    [C1]    GND                   GND
     │       │
    GND     GND
```

### 蔡氏二极管（用运放实现）
```
        R3
Vin ──[R1]──┬──[─\
            │    >──┬── Vout
     ┌──────┤──[+/  │
     │      │       │
    [R2]   [R4]    [R5]
     │      │       │
    GND   -Vsat   +Vsat

分段线性特性：
  Vout = m0×Vin (|Vin|<Bp)
  Vout = m1×Vin + (m0-m1)×Bp (|Vin|>Bp)
  
其中m0<0, m1<0 (负阻特性)
```

### 参数调节
- 通过改变R的值，可以在周期和混沌之间切换
- R减小：单周期→双周期→四周期→...→混沌
- 用数字电位器(AD5206)实现程控调节

## 40MHz带宽挑战
- 普通蔡氏电路频率在kHz级
- 提高频率：减小L和C
- 40MHz需要：L≈100nH, C≈10pF
- 运放GBW需要>200MHz（如AD8065 GBW=145MHz，可能不够）
- 可能需要用分立晶体管代替运放

## 无电感方案
- 用运放实现有源电感（回转器）
- 回转器：将电容"变换"为等效电感
- 需要额外2个运放
