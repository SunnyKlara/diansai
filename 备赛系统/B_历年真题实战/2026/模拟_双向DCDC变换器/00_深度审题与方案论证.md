# 2026 模拟 双向 DC-DC 变换器 —— 深度审题与方案论证

> **本题为 2026 模拟出题**，用于检验方法论 + 脚手架的实战可用性。
> 题目设定参考电赛历年电源类常见模式：双向能流 + 高效率 + 精确稳压。

---

## 第一层：题目设定（模拟）

### 题目要求

设计一台双向 DC-DC 变换器，连接 24V 电池组（B 端）与 12V 直流母线（A 端），具备：

| 项 | 要求 | 分值 |
|---|---|---|
| (1) Buck 模式（24V → 12V）| Vout = 12V ± 0.1V，Iout 最大 5A | 25 |
| (2) Boost 模式（12V → 24V）| Vout = 24V ± 0.2V，Iout 最大 2.5A | 25 |
| (3) 模式自动切换 | 切换瞬态 < 50ms，不停机 | 20 |
| (4) 效率 | 满载效率 ≥ 90% | 20 |
| (5) 过流 / 过压保护 | 故障 100ms 内停机 | 5 |
| (6) 显示 + 旋钮调电压 | OLED + 旋钮编码器 | 5 |
| 报告 | | 20 |
| **合计** | | **120** |

MCU：**必须 TI 系（MSP430 / MSPM0 / C2000 等）** —— 模拟规则按近年趋势设定。

### 隐含约束

**约束 1**：双向同步整流  
"切换瞬态 < 50ms 不停机" 暗示用同步整流（4 个 MOS 全控），而非加二极管。

**约束 2**：电流采样必须双向  
Buck 模式电流方向 A→B，Boost 反向。霍尔电流传感器（如 ACS712-30A）天然双向。

**约束 3**：效率 90% 是硬指标  
开关损耗 + 导通损耗 + 磁性损耗 ≈ 10%。MOS 选 Rds(on) < 10mΩ 的 SiC，电感选低 DCR 高磁导率铁硅铝。

## 第二层：误差预算 + 效率预算

### 控制精度链

```
Vbus 采样 → ADC → 数字 PI → PWM 占空比 → 实际 Vbus
   ↓         ↓       ↓            ↓             ↓
分压精度  量化   稳态误差   死区时间    负载阶跃响应
```

| 误差源 | 量级 | 处理 |
|---|---|---|
| 分压电阻精度 | 0.5%（普通） / 0.1%（精密） | 用 0.1% 精密 |
| ADC 量化（12bit） | 0.024% | 可忽略 |
| PI 稳态残差 | 取决于 Ki | Ki 大但要稳，仿真先调 |
| 死区电压跌落 | 死区 200ns × fsw 100kHz × Vbus = 0.2% × Vbus | 控制在 ±0.1% |
| 总误差 | < 0.5% × 12V = 60mV | < 0.1V 容差 ✓ |

### 效率预算（满载 5A @ 12V = 60W）

| 损耗源 | 量级 | 占比 |
|---|---|---|
| MOS 导通损耗（Rds(on)=10mΩ × 5A² × 2 个 MOS） | 0.5W | 0.83% |
| MOS 开关损耗（fsw=100kHz, V=24V, I=5A） | 1.5W | 2.5% |
| 电感铜损（DCR=20mΩ × 5A²） | 0.5W | 0.83% |
| 电感铁损（高磁导率铁硅铝） | 0.5W | 0.83% |
| 控制电路 + 驱动 | 1.0W | 1.67% |
| **合计** | **4.0W** | **6.67%** |

效率 ≈ 1 - 6.67% = **93.3%**，达 90% 要求有 3.3% 余量。

## 第三层：MCU + 拓扑选型

### MCU 选 STM32G474（TI 系替代品 MSPM0G3507）

> 注：本题模拟"必须 TI MCU"，实际选型走 MSPM0G3507。

| 候选 | 主频 | PWM 互补 | ADC | 适合度 |
|---|---|---|---|---|
| MSPM0G3507 | 80MHz | ✅ HRPWM | 4 MSPS 12bit | ✅ 推荐 |
| MSP430FR5994 | 16MHz | ❌ | 200kSPS | ✗ 主频不够 |
| C2000 F28379D | 200MHz | ✅ | 3.5MSPS 16bit | △ 性能 OK 但开发难 |

### 拓扑：同步 Buck-Boost（4 MOS 半桥）

```
B 端 (24V 电池) ──[L]── A 端 (12V 母线)
     │                       │
   M1/M2                    M3/M4
   半桥                      半桥
   PWM_AB                   PWM_CD
```

- Buck 模式：M3/M4 作为同步整流，M1/M2 主控
- Boost 模式：M1/M2 作为同步整流，M3/M4 主控
- 切换：通过软件状态机，不停 PWM 直接切换占空比

### 关键器件

| 器件 | 型号 | 关键参数 |
|---|---|---|
| MOS | IRFR3711 / IRLR8113 | Rds(on) < 10mΩ, Vds 40V |
| 电感 | 22uH 铁硅铝 | DCR < 20mΩ, Isat > 8A |
| 电流传感器 | ACS712-30A | 双向，灵敏度 66mV/A |
| 驱动 | UCC27210 | 半桥驱动，3A 峰值 |

## 第四层：软件架构

### 双闭环控制

```
外环（电压环，10kHz）：
  Vref - Vbus → PI → I_ref（电流参考）

内环（电流环，100kHz，与 PWM 同步）：
  I_ref - I_meas → PI → 占空比 D
```

### 状态机

```
ST_BOOT ──→ ST_PRECHARGE（预充电，限制涌入电流）
                ↓
         ST_BUCK_RUN ←──→ ST_BOOST_RUN
                ↓                 ↓
              ST_FAULT（过压 / 过流 / 过温）
```

### 关键算法

1. **双闭环 PI**：复用 5 题里的 `snippets/pid_controller.py`
2. **平均电流采样**：100kHz PWM 中点采样 → 12-tap 滑动平均
3. **方向自动检测**：Vbus_ref > Vbat 时进入 Boost，否则 Buck

## 第五层：硬件难点

### 难点 1：双向同步整流的死区时间（关键！）

**症状**：死区太短直通烧管；太长则二极管续流损耗大。

**对策**：
- 测试期从 500ns 起步
- 用示波器测互补 PWM 上升 / 下降沿
- 找到 "上沿到上沿" 的临界值，加 100ns 余量
- 典型最终值：200~300ns

### 难点 2：电流采样的相位

ACS712 输出有 ~5μs 延迟。100kHz PWM 周期 10μs → 50% 相位偏移。

**对策**：用电感电流上升斜率的中点（即 PWM 中央时刻）采样。

### 难点 3：模式切换瞬态

Buck → Boost 切换时，PI 积分项是上一模式的，会引入冲击。

**对策**：切换时 PI 积分项重置为 "新模式的稳态前馈值"。

## 风险登记

| 风险 | 概率 | 影响 | 缓解 |
|---|---|---|---|
| 死区调不好烧管 | 高 | 致命 | 先用 500ns 跑半小时，再缩短 |
| 模式切换不平滑 | 中 | 高 | 仿真先验证 PI 积分重置 |
| 满载效率不达 | 中 | 20 分 | MOS 选 SiC，电感低 DCR |
| 模式切换震荡 | 中 | 中 | 加 0.5V 滞回 |
