## 常见故障应对

| 现象 | 可能原因 | 解决 |
|---|---|---|
| 上电瞬间 MOS 烧 | 死区太短 | 死区调到 500ns 起步 |
| 输出电压抖 | PI 振荡 | KP 调小 |
| 切换有过冲 | 积分没重置 | reset_for_mode_switch |
| 满载效率 80% | MOS 选错 / 电感 DCR 大 | 换 IRLR8113 / 检查 DCR |
| ADC 噪声大 | 模拟地数字地没分割 | 检查 PCB |
| OLED 闪 | I2C 上拉太小 | 改 4.7kΩ |
