# 2026 模拟 双向 DC-DC 变换器 — 电路设计说明

## 一、系统拓扑

```
B 端 (24V 电池 / 12V 电池) ──┬── M1 ──┬── L (22uH) ──┬── M3 ──┬── A 端 (12V/24V 母线)
                              │         │              │         │
                              M2        │              M4        │
                              │         │              │         │
                              GND       └─ Cmid ──     GND       └─ Cout ──
                                          (470uF)                  (470uF)
```

- M1/M2 半桥（B 端 / 输入侧）
- M3/M4 半桥（A 端 / 输出侧）
- L 电感串联在中间

### Buck 模式（B → A，24V → 12V）

- M1/M2 主控（PWM, D = Vout/Vin = 0.5）
- M3 = ON（同步整流，与 M2 互补）
- M4 = OFF
- 实际上 M3/M4 也是高频开关（M3 与 M2 同相，M4 与 M1 同相）

### Boost 模式（B → A，12V → 24V）

- M3/M4 主控（PWM, D = 1 - Vin/Vout = 0.5）
- M1 = ON（同步整流）
- M2 与 M3 同相
- M1 与 M4 同相

### 双向同步整流

四个 MOS 全部高频开关，由 MCU 决定哪对作为"主控"哪对作为"同步整流"，模式切换通过软件改 PWM 占空比方向即可，**硬件不切换**。

## 二、电压采样

### Vbus 采样（A 端，0~30V → 0~3.0V）

```
A 端 ──[10kΩ]──┬──[2.2kΩ]── GND
                │
                ├── [1kΩ + 100nF RC]── ADC IN1
                │   (fc = 1/(2π·1k·100n) = 1.6 kHz)
```

分压比 = 2.2 / (10 + 2.2) = 0.18  
0~30V → 0~5.4V → 经 OPA2333 缓冲后限幅到 3.0V

### Vbat 采样（B 端，0~30V → 0~3.0V）

同上结构。

## 三、电流采样（双向）

```
功率回路 ──[ACS712-30A 通道]──┬── 输出
                               │
                          1.65V 偏置中心
                               │
                          灵敏度 66 mV/A
                               │
                          [1kΩ + 100nF RC]── ADC IN3
                               │
                          MCU 端：i = (adc - 2048) × 3.3/4095 / 0.066
```

ACS712 BW = 80kHz，对 100kHz PWM 有 -6dB 衰减。**必须**用电感电流"中点采样"补偿（在 PWM 中央时刻触发 ADC，此时电流是平均值）。

## 四、PWM 死区

- 实测起步：500ns（保险）
- 调到："上升沿到上升沿"刚好不重叠 + 100ns 余量
- 典型最终值：200~300ns
- 太短：直通，瞬间 1000A 烧管
- 太长：续流损耗，效率掉

## 五、自举电源

UCC27210 的 HS 自举：

```
Vcc (+12V) ──[1N4148]──┬── BOOT
                        │
                     [100nF]
                        │
                        └── HS (MOS 源极)
```

每个半桥一套。**MOS 长时间 ON 时（如 Buck 模式 D=1.0），自举电容会放完**，所以最大占空比限到 0.95。

## 六、保护电路

### 过流（硬件）

电流采样输出 → 比较器 → MCU 急停引脚（Break Input）

阈值：8A（过载 1.6×）→ 比较器跳变 → MCU 200ns 内关 PWM。

### 过压（软件）

ADC 采样 Vbus，超 28V（Boost 模式）或 14V（Buck 模式）→ 故障状态机。

### 过温（软件）

NTC 测 MOS 散热片，超 70°C → 降低 PWM 限幅到 0.5。

## 七、PCB 布局要点

1. **功率回路最短**：M1-M2 半桥的 D-S 走线 + 电容回流路径要尽量短，**面积越小越好**
2. **驱动信号包地**：MOS 栅极走线下面铺 GND 平面，减少 EMI
3. **采样地分割**：模拟地（运放 / ADC 输入）与功率地单点连接
4. **电感放中央**：避免磁场耦合到 ADC 信号
5. **散热片电气浮空**：通过云母片或导热硅胶绝缘

## 八、调试顺序（保命）

> ⚠️ 永远先在低压低电流下测试，验证 OK 再放开。

1. **干灯测试**（不上电）：用万用表蜂鸣档查 D-S 通断、栅源短路
2. **3.3V 控制电源**：先只上 3.3V，验证 MCU + OLED 工作
3. **12V 驱动电源**：上 12V 驱动，验证 PWM 输出（示波器看栅极波形）
4. **5V 母线 + 100mA 限流**：开环 PWM = 0.5 (恒定)，看是否能稳定输出
5. **逐步升压**：5V → 12V → 24V，每档跑 5 分钟，**摸 MOS 是否烫**
6. **加负载**：1A → 3A → 5A，每档测稳态精度 + 效率
7. **闭环**：开 PI，从轻载到满载
8. **模式切换**：低压验证切换瞬态
9. **故障注入**：模拟过流 → 验证保护
