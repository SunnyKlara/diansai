## 通用调参三步走（所有 PID 适用）

```
Step 1: 只 P，找临界振荡 KP_crit
        现象 1：波形持续等幅振荡
        实测时机：用串口画 setpoint 与 measured 曲线
        
Step 2: KP = 0.5 × KP_crit
        加入 KD = 0.1 × KP（典型）
        看是否过冲 < 20%
        
Step 3: 加 KI = 0.01 × KP（很小）
        慢慢翻倍直到稳态误差消失
        但 KI 太大会震荡
```

**禁忌**：
- 永远不要先调 KI（积分饱和炸）
- 永远不要 KP > 临界值（持续振荡 → 烧管 / 失控）
- 永远不要在不稳定系统（如磁悬浮）上加大 KI

| 收敛时间 | < 50ms | | |

### 4.2 PI 起步参数从哪来

```
开环带宽 = ωc = 2π × 100 = 628 rad/s
KP × 母线/L = ωc → KP = 628 × 1.5e-3 / 36 = 0.026
为留余量取 0.05~0.10。
```

### 4.3 调试节奏

```
Step 1: 空载，KP=0.05, KI=0
        阶跃设定值 12V → 24V，看响应
        预期：无振荡，但稳态误差 1V

Step 2: KP=0.10, KI=0
        稳态误差仍 0.5V

Step 3: KP=0.10, KI=50
        稳态误差消失，但 50ms 收敛

Step 4: KP=0.10, KI=150
        稳态 < 10ms，无振荡 ✅
```

### 5.3 dq 解耦项（前馈）

```
ud_decoup = +ω·L·iq
uq_decoup = -ω·L·id
```

> ω L 在 G474 内用 FMAC 协处理器 0.5μs 算一次，**不要再 CPU 算**。

### 5.4 调参顺序（强约束）

```
1. 先开环：固定调制比 0.5，看三相输出对称
2. 加电流环：先 KP_ID 后 KI_ID
3. 加 dq 解耦：观察 iq 是否稳定在 0
4. 加母线外环：先 KP_VBUS 后 KI_VBUS
5. 加同步整流：观察效率提升
```

## 7. 调试常见问题

| 现象 | 大概率原因 | 修复 |
|---|---|---|
| 系统持续振荡 | KP 太大 | KP × 0.7 |
| 稳态误差不消 | KI 太小 | KI × 2 |
| 启动过冲 > 30% | KI 过大 / 积分饱和 | 加积分限幅 |
| 响应慢 > 100ms | KP 太小 | KP × 1.5 |
| 高频抖动 | KD 噪声放大 | D 项加 LPF |
| 突然失稳 | 积分饱和（windup）| 加 anti-windup |

## 8. 通用 anti-windup 代码

```c
integ += err * Ki * dt;
if      (integ > integ_max) integ = integ_max;
else if (integ < integ_min) integ = integ_min;

float u = Kp * err + integ;

if      (u > u_max) u = u_max;
else if (u < u_min) u = u_min;

/* 反计算 anti-windup */
if      ((integ > 0) && (u >= u_max - 0.01f)) integ -= err * Ki * dt;
else if ((integ < 0) && (u <= u_min + 0.01f)) integ -= err * Ki * dt;
```
