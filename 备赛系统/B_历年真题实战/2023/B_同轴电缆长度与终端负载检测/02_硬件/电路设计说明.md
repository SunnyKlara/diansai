# 2023 B 同轴电缆长度与终端检测 —— 电路设计

## 系统总图
```
[STM32F407 TIM1] ──→ [GPIO 阶跃发生器（74HC04 反相器组阶跃）] ──→ [50Ω 串联匹配] → 同轴电缆 → 终端 (开/短/R)
                                                                                                            
                              [ADC IN0 ←───────────────────────] 输入端电压（含入射 + 反射）
                                  ↑
                           [STM32 ADC 1MSPS]
                                  ↑
                           [等效采样 100MSPS]
```

## 关键
- 同轴 50Ω 必须严格匹配，否则源端二次反射干扰
- 阶跃边沿 < 5ns（74HC04 ~ 4ns）
- ADC 输入端加 LC 衰减网络，避免反射电压超过 ADC 范围
- 触发 - 采样延迟用 TIM1 ARR / OC 编码（步进 6ns @ 168MHz）
