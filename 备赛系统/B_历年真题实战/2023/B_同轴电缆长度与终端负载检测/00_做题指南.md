## 核心思路：等效采样 TDR

1. 阶跃发生器（74HC04 反相器组）→ 同轴电缆 → 终端反射
2. STM32 ADC 1MSPS + TIM1 编程延迟 → 100 次延迟扫描合成等效 100MSPS
3. 找入射峰 + 反射峰 → 双程时间 → 长度
4. 反射系数 ρ = V反 / V入 → R = Z₀(1+ρ)/(1-ρ)

## 关键参数

- Vp（RG58）= 2c/3 = 2×10⁸ m/s
- 1m 双程 = 10ns，50m = 500ns
- Z₀ = 50Ω（同轴）
- 阶跃边沿 < 5ns
