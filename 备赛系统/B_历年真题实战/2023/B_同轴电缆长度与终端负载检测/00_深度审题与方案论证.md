# 2023 B 同轴电缆长度与终端负载检测装置 —— 深度审题

## 题目精确

| 项 | 范围 / 要求 | 分值 |
|---|---|---|
| 测电缆长度 | 1m~50m，误差 ≤ 5% | 25 |
| 测终端开/短/匹配 | 三态识别 | 15 |
| 终端任意 R 测量 | 50~200Ω, 误差 ≤ 10% | 30 |
| 报告 | — | 20 |

## 物理本质：TDR (Time Domain Reflectometry)
发射阶跃 / 脉冲 → 终端阻抗失配反射 → 测反射时间得长度，测反射幅值得阻抗。

## 关键参数
- 同轴电缆 RG58: Vp ≈ 2c/3 = 2×10⁸ m/s
- 1m 电缆双程时间 = 2/(2×10⁸) = 10ns
- 50m 双程时间 = 500ns
- ⇒ ADC 采样率必须 ≥ 100MSPS（看清 10ns）

## 方案

### 方案 A: 高速 ADC + STM32H7
AD9226 (12bit 65MSPS) 或 ADS54J60 (250MSPS)；STM32H7 480MHz

### 方案 B: 等效采样（推荐）
触发信号 N 次重复发射，每次 ADC 在不同延迟点采样 → 等效高速。  
STM32F407 ADC 1MSPS 通过 100 次重复 + 0..100ns 延迟 → 等效 100MSPS。

### 方案 C: 时间数字转换 (TDC)
用 LMX2581 或 MCU 内置 TDC 测过零时刻，精度 ps 级。

**选 B**：硬件成本低，对 1MSPS ADC 友好，1m 精度足够（10ns / 100 = 0.1ns 等效精度）。

## 控制
- TIM1 触发激励 + ADC 同步采样
- 软件改延迟扫 100 次构建波形
- 找首个反射峰 → 长度
- 反射系数 = V_反射 / V_入射 → 终端阻抗 R = Z₀(1+ρ)/(1-ρ)

## 风险
| 风险 | 缓解 |
|---|---|
| 等效采样需信号严格周期 | TIM1 高精度触发 |
| 反射多次叠加 | 触发周期 > 50m 双程 1μs |
| Z₀ 不准 | 用已知 50Ω 终端反推 Z₀ |

保底 50 / 期望 80 / 冲刺 95
