# 2023B TDR同轴电缆检测 —— 独到经验总结

## 一、TDR的"盲区"问题——发挥部分(3)的20分

### 什么是盲区？
发射脉冲本身有一定宽度（比如10ns）。
在脉冲持续期间，如果反射波已经回来了，
发射脉冲和反射脉冲会重叠，无法区分。

盲区距离 = 脉冲宽度 × 传播速度 / 2

10ns脉冲 → 盲区 = 10ns × 1.98×10^8 / 2 = 0.99m ≈ 100cm

**这就是为什么发挥部分(3)要求"减小盲区至L≤100cm"——
它本质上是在考你能不能产生足够窄的脉冲。**

### 如何突破100cm盲区？

**方案1：更窄的脉冲**
- 目标：脉冲宽度<5ns → 盲区<50cm
- STM32 GPIO翻转约5ns@168MHz → 刚好
- 但GPIO的上升沿不够陡（约2ns），会影响TDC触发精度

**方案2：脉冲压缩（Chirp + 匹配滤波）**
- 发送线性调频脉冲（chirp），接收后做匹配滤波
- 等效脉冲宽度可以做到亚纳秒级
- 但实现复杂度大幅增加

**方案3：频域法（VNA原理）**
- 不发脉冲，而是扫频测量S11参数
- 对S11做IFFT得到时域反射波形
- 分辨率取决于扫频带宽，不受脉冲宽度限制
- 但需要精确的幅度和相位测量

**我的建议**：方案1（窄脉冲），用74AC04反相器链产生3~5ns脉冲。
简单可靠，足以将盲区降到50cm以下。

## 二、负载参数测量——发挥部分(2)的15分

### 电阻值测量

反射系数 ρ = (R-50)/(R+50)

已知ρ → R = 50×(1+ρ)/(1-ρ)

**关键**：ρ的测量精度直接决定R的精度。

ρ的测量方法：
1. 先测开路时的反射幅度 V_open（ρ=1）
2. 再测接电阻时的反射幅度 V_load
3. ρ = V_load / V_open

**误差分析**：
- 题目要求R在10~30Ω范围
- R=10Ω → ρ = (10-50)/(10+50) = -0.667
- R=30Ω → ρ = (30-50)/(30+50) = -0.250
- ρ的变化范围：-0.667 ~ -0.250
- 要求误差≤10% → ρ的测量精度需要约±0.04
- 这对ADC精度要求不高（12bit足够）

### 电容值测量

电容负载的反射波形是指数曲线：
V_reflected(t) = V0 × (1 - 2×exp(-t/τ))
其中 τ = Z0 × C = 50 × C

测量τ → C = τ / 50

**关键**：需要看到反射波形的时间演变，不能只测一个时间点。

方法：等效采样
1. 第1次测量：在反射到达后0ns采样
2. 第2次测量：在反射到达后10ns采样
3. 第3次测量：在反射到达后20ns采样
...
N. 第N次测量：在反射到达后(N-1)×10ns采样

拼接出完整的反射波形，然后拟合指数曲线得到τ。

**时间范围**：
- C=100pF → τ = 50×100p = 5ns（非常快！）
- C=300pF → τ = 50×300p = 15ns
- 需要在5~50ns范围内采样，步进约1~2ns
- 等效采样需要约30次测量 × 10ms/次 = 300ms

## 三、电缆自备的策略

题目说"被测同轴电缆由参赛队自备"。

**这是一个隐藏的优势！**

你可以：
1. 提前精确测量自备电缆的长度（用卷尺，精度±1cm）
2. 提前测量电缆的传播速度（用已知长度校准）
3. 提前测量电缆的特性阻抗（用LCR表或网络分析仪）
4. 准备多种长度的电缆（1m, 5m, 10m, 20m）

**赛场上的校准流程**：
1. 接上自备的已知长度电缆
2. 运行校准程序，标定传播速度
3. 校准完成后开始正式测量

## 四、比较器电路的设计——TDC STOP信号

TDC7200的STOP引脚需要一个上升沿来标记反射波到达时刻。

**方案**：高速比较器（TLV3501，传播延迟4.5ns）

```
反射信号 → [耦合电容] → [比较器(+)] → TDC STOP
                          [阈值电压(-)]
```

**阈值设置的学问**：
- 阈值太低：噪声触发，产生虚假STOP
- 阈值太高：弱反射信号检测不到
- 最优：略高于噪声底（约50mV）

**另一个陷阱**：比较器的传播延迟(4.5ns)会引入固定误差。
但这个误差是固定的，可以在校准时消除。
