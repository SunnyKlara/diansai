## 2. 时钟

```
HSE: 8 MHz
PLL × 9 → SYSCLK 72 MHz
APB1: 36 MHz（TIM2 倍频 72 MHz）
APB2: 72 MHz
```

## 3. PWM 50Hz（舵机）

```
TIM2 PSC = 71 → 1 MHz 计数
ARR = 19999 → 50 Hz 周期（20 ms）
CCR = 500 → 0.5 ms（0°）
CCR = 1500 → 1.5 ms（90°）
CCR = 2500 → 2.5 ms（180°）
```

每度 ≈ (2500-500)/180 = 11.11 个 CCR 单位。

## 4. 引脚映射 → 屏幕坐标

```
屏幕距离 D = 1 m（KT 板）
yaw_deg = arctan(x_m / D) × 180/π
pitch_deg = arctan(y_m / D) × 180/π

舵机 CCR = 1500 + yaw_deg × 11.11
舵机限位: yaw ±45°, pitch ±30°
```
