# 2023 E 运动目标追踪 —— 接线表

> 红色源 / 绿色追踪两套独立装置，**完全无通信**（赛规约束）。

## 总图

```
[KT 板屏幕 80×80cm，距云台 1m]
     ↑                          ↑
     │ 红色光斑                  │ 绿色光斑（追红色）
     │                          │
[红色云台]                  [绿色云台]
[红激光]                    [绿激光 + OpenMV]
   ↑                            ↑
[STM32F103]                 [STM32F103]
[预设路径]                   [视觉 + PID]
```

## 1. 电源

| # | 源 | 目的 | 颜色 |
|---|---|---|---|
| P1 | 5V/3A 适配器 | MG996R × 2 VCC | 红粗 |
| P2 | GND | MG996R × 2 GND | 黑粗 |
| P3 | MCU 3.3V | OpenMV 3.3V | 红 |
| P4 | 5V | OpenMV VIN | 红 |
| P5 | MCU 3.3V | 激光头继电器线圈 + | 红 |
| P6 | 共地 | 全部 GND | 黑 |

> **舵机不能用 MCU 板载 3.3V**——电流不够，必须独立 5V 适配器。

## 2. 二轴云台舵机

| # | MCU | 舵机 | 颜色 |
|---|---|---|---|
| S1 | PA0 (TIM2_CH1) | Yaw 舵机 信号 | 黄 |
| S2 | 5V | Yaw 舵机 + | 红 |
| S3 | GND | Yaw 舵机 - | 黑 |
| S4 | PA1 (TIM2_CH2) | Pitch 舵机 信号 | 橙 |
| S5 | 5V | Pitch 舵机 + | 红 |
| S6 | GND | Pitch 舵机 - | 黑 |

## 3. 激光头控制

| # | MCU | 激光 | 备注 |
|---|---|---|---|
| L1 | PA8 | 继电器线圈 + | 经 1kΩ + 8050 NPN 三极管 |
| L2 | GND | 继电器线圈 - | |
| L3 | 继电器触点常开 | 激光头 + | |
| L4 | 5V | 激光头电源 | 5mW 红/绿激光（赛规上限）|

## 4. OpenMV（仅绿色追踪 MCU 用）

| # | MCU | OpenMV | 颜色 |
|---|---|---|---|
| O1 | PA10 (RX) | OpenMV P5 (TX) | 黄 |
| O2 | PA9 (TX) | OpenMV P4 (RX) | 蓝 |
| O3 | GND | GND | 黑 |
| O4 | 5V | VIN | 红 |

> OpenMV 安装在绿色云台**正上方**，镜头朝向 KT 屏幕，距屏幕 1m。

## 5. A4 旋转角拨码（仅红色源 MCU）

| # | MCU | 拨码 |
|---|---|---|
| R1 | PB0 | bit0 |
| R2 | PB1 | bit1 |
| R3 | PB2 | bit2（4 位编码 0/15/30/45/60/75/90°）|
| R4 | PB3 | bit3 |

## 6. 按键 / LED

| # | MCU | 元件 |
|---|---|---|
| K1 | PA11 | KEY_START |
| K2 | PA12 | KEY_RESET |
| L1 | PB12 | LED_RED + 1kΩ |
| L2 | PB13 | LED_GREEN + 1kΩ |
| L3 | PC13 | 板载 LED |

## 7. 接线检查

- [ ] 舵机用独立 5V 电源，**不要用核心板 3.3V**
- [ ] 激光不要直接接 MCU GPIO（5mW 不能直驱），用三极管 / 继电器
- [ ] OpenMV 安装时镜头水平且不晃动
- [ ] 屏幕距云台 = 1.0m ± 5cm
- [ ] 黑色 KT 板背景（白板会反光干扰）

## 8. 现场调试

1. 红色源单独通电 → 看激光打在屏幕中心
2. 设 0/45/90° 拨码 → 看路径轨迹符合 A4 旋转
3. 绿色追踪通电 → OpenMV 启动后能识别红色光斑
4. 同时上电 → 验证追踪
