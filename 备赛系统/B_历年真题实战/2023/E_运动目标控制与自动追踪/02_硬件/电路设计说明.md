# 2023 E 电路设计说明

## 系统总图（×2 套：红 / 绿）
```
[STM32F103] ─PWM(50Hz)─→ [Yaw 舵机 MG996R]
                          [Pitch 舵机 MG996R]
                          [激光头 5V GPIO 控制]

绿色另加：
[OpenMV H7] ──UART──→ [STM32F103]
   ↑
[摄像头 OV7725]
```

## 关键
- 屏幕 KT 板 80×80cm，距云台 1m
- 红激光 650nm ≤5mW，绿激光 532nm ≤5mW
- 舵机 5V/3A 适配器（不要用 MCU 供电）
- 信号地与电源地分开走线（避免抖动）
