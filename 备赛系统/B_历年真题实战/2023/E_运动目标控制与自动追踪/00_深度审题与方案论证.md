# 2023 E 运动目标控制与自动追踪系统 —— 深度审题

## 题目精确

| 项 | 分值 |
|---|---|
| 基本(1) 红色复位 ≤2cm | 10 |
| 基本(2) 红色边线 30s 一圈 | 10 |
| 基本(3) 红色 A4 胶带 | 15 |
| 基本(4) A4 旋转后重做 (3) | 15 |
| 发挥(1) 绿色 2s 内追上红色 | 10 |
| 发挥(2) 红色运动绿色追踪 | 35 |
| 报告 | 20 |
| **合计** | **115** |

## 强约束
红/绿系统**完全独立**，绿色只能视觉获取红色信息。  
评分标准：误差 < 3cm 算追踪成功；> 5cm 完全失分。

## 方案
- 红色：STM32F103 + 二轴 MG996R 云台 + 红激光，**预设路径 + 角度查表**
- 绿色：STM32F103 + 二轴云台 + 绿激光 + OpenMV H7
- OpenMV：颜色阈值 + find_blobs() → cx/cy → UART → MCU

## 控制
- 红色：t → 边线参数方程 → (x,y) → arctan(x/D) → 舵机角
- 绿色：cx,cy → 像素误差 → PID → 舵机增量

## 关键参数
- 屏幕 1m, 边线 50×50cm, A4 21×29.7cm
- 舵机分辨率：MG996R ±45° / 1024 = 0.088° → 屏幕分辨 1.5mm @ 1m
- OpenMV 帧率 30fps，60ms 延迟

## 风险

| 风险 | 缓解 |
|---|---|
| 红/绿激光阈值漂移 | OpenMV 启动时取 ROI 自校 |
| 舵机抖动 | PID 加 D 项 + 死区 |
| OpenMV 帧延迟 | PID Kp 不能太大（< 0.1）|
| A4 旋转角识别 | 加按键 0/15/30/45/60° 手动设置 |

保底 60 / 期望 90 / 冲刺 105
