# 2023 D 信号调制方式识别 —— 深度审题

## 题目精确

| 项 | 范围 | 分值 |
|---|---|---|
| 识别 AM/FM/2ASK/2FSK/2PSK 5 种 | SNR ≥ 15dB | 50 |
| 显示载频、调制度 | 误差 < 5% | 30 |
| 报告 | — | 20 |

## 物理本质
不同调制方式在**包络变化**、**瞬时频率**、**瞬时相位**三个维度上有不同特征：
- AM / 2ASK：包络变 → σ_a 大
- FM / 2FSK：瞬时频率变 → σ_f 大
- 2PSK：瞬时相位跳变 180° → σ_phi 双峰

## 方案
**特征统计 + 决策树**（不用机器学习，规则简单）：
1. 软件 IQ 下变频到基带
2. 求 a, df/dt, dφ
3. σ_a > 0.3 → AM 类；σ_a > 0.6 → 2ASK
4. σ_a < 0.3 + σ_f > 0.5 → 2FSK；σ_f > 0.2 → FM
5. σ_a < 0.3 + σ_f < 0.2 → 2PSK

## 风险

| 风险 | 缓解 |
|---|---|
| 阈值随 SNR 漂移 | 多帧投票 + 校准模式 |
| 2PSK 与 2FSK 混淆 | 两个阈值都要稳过 |

保底 50 / 期望 75 / 冲刺 95
