## 调制方式识别算法

### 第一步：区分AM和FM/CW
```
计算信号包络的方差：
  包络方差大 → AM
  包络方差小 → FM 或 CW
```

### 第二步：区分FM和CW
```
计算瞬时频率的方差：
  频率方差大 → FM
  频率方差小 → CW
```

### 第三步（发挥）：区分数字调制
```
2ASK: 包络有两个离散电平
2FSK: 瞬时频率有两个离散值
2PSK: 相位有两个离散值(0和π)
```

### 代码框架
```c
typedef enum {
    MOD_AM, MOD_FM, MOD_CW,
    MOD_2ASK, MOD_2FSK, MOD_2PSK,
    MOD_UNKNOWN
} ModType_t;

ModType_t identify_modulation(float* signal, int len) {
    // 1. 计算包络
    float* envelope = hilbert_envelope(signal, len);
    float env_var = calc_variance(envelope, len);
    
    // 2. 计算瞬时频率
    float* inst_freq = calc_inst_freq(signal, len);
    float freq_var = calc_variance(inst_freq, len);
    
    // 3. 判决
    if (env_var > ENV_THRESHOLD) {
        // 包络变化大
        if (is_continuous(envelope, len))
            return MOD_AM;      // 连续包络变化
        else
            return MOD_2ASK;    // 离散包络变化
    } else {
        // 包络恒定
        if (freq_var > FREQ_THRESHOLD) {
            if (is_continuous(inst_freq, len))
                return MOD_FM;
            else
                return MOD_2FSK;
        } else {
            // 频率也恒定，检查相位
            float* phase = calc_inst_phase(signal, len);
            float phase_var = calc_variance(phase, len);
            if (phase_var > PHASE_THRESHOLD)
                return MOD_2PSK;
            else
                return MOD_CW;
        }
    }
}
```

## 关键技术挑战

### 2MHz载波采样
- 奈奎斯特：≥4MHz
- 实际推荐：≥10MHz
- STM32F4片内ADC最高约2.4MSPS → 不够！
- 方案：外部ADC(AD9226 65MSPS)或带通采样

### 带通采样（推荐）
- 2MHz载波，带宽约100kHz
- 带通采样率只需≥200kHz（远低于奈奎斯特）
- 但需要精确的采样时钟和抗混叠滤波
