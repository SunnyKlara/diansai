# 2023D 调制方式识别 —— 独到经验总结

## 一、这道题的本质是什么？

表面上是"通信题"，本质上是**模式识别/信号分类问题**。

和机器学习的分类任务一样：
1. 特征提取（从原始信号中提取有区分度的特征）
2. 分类器设计（根据特征判断类别）
3. 参数估计（确定类别后估计具体参数）

区别在于：电赛不能用GPU跑神经网络，必须在MCU上实时运行。
所以特征要精心选择，分类器要简单高效。

## 二、6种调制方式的"指纹"

| 调制方式 | 包络 | 瞬时频率 | 瞬时相位 | 频谱 |
|----------|------|----------|----------|------|
| CW | 恒定 | 恒定 | 线性增长 | 单一谱线 |
| AM | **连续变化** | 恒定 | 线性增长 | 载波+对称边带 |
| FM | 恒定 | **连续变化** | 非线性 | 宽带，Carson带宽 |
| 2ASK | **离散跳变(2电平)** | 恒定 | 线性增长 | 载波+sinc包络 |
| 2FSK | 恒定 | **离散跳变(2值)** | 分段线性 | 双峰 |
| 2PSK | 恒定 | 恒定 | **π跳变** | 类似2ASK但无载波 |

**关键区分特征**：
- AM vs 2ASK：包络是连续变化还是离散跳变 → 用**峰度**区分
- FM vs 2FSK：瞬时频率是连续变化还是离散跳变 → 用**峰度**区分
- CW vs 2PSK：相位是否有π跳变 → 用**相位差分直方图**区分

## 三、最容易混淆的两对

### AM vs 2ASK
- 当AM调幅度ma很大(>0.8)时，包络接近矩形 → 像2ASK
- 当2ASK的码元速率很低时，包络变化很慢 → 像AM
- **区分方法**：包络的峰度
  - AM包络是正弦调制 → 峰度≈1.5（均匀分布的峰度）
  - 2ASK包络是矩形 → 峰度≈1.0（双峰分布）
  - 阈值约1.2~1.3

### CW vs 2PSK
- 两者的包络都恒定，瞬时频率都恒定
- 唯一区别：2PSK在码元边界有相位π跳变
- **区分方法**：
  1. 计算信号的平方：s²(t)
  2. CW的平方仍是正弦（频率翻倍）
  3. 2PSK的平方消除了相位跳变（因为π×2=2π=0）
  4. 对s²(t)做FFT，如果在2fc处有尖锐谱线 → 2PSK
  5. 如果在fc处有尖锐谱线 → CW

这个方法叫**平方律检测**，是2PSK识别的经典方法。

## 四、2MHz载波的采样方案

### 为什么不能直接用4MSPS ADC采样？

理论上可以，但：
1. STM32F4片内ADC最高2.4MSPS → 不够
2. 外部ADC(AD9226 65MSPS)可以，但数据量巨大
3. 4096点×65MSPS = 63μs的数据，可能不够一个调制周期

### 我的方案：混频到中频后采样

```
2MHz信号 → [混频器(SA612)] → [455kHz中频滤波器] → [1MSPS ADC]
                ↑
           [本振 2.455MHz]
           (Si5351产生)
```

优势：
- 1MSPS的ADC对455kHz信号有2.2倍过采样，足够
- 455kHz中频滤波器是标准器件，便宜好买
- 所有调制信息完整保留在中频信号中
- STM32F4片内ADC就够用，不需要外部ADC

### 带通采样方案（备选）

如果不想加混频器，可以用带通采样：
- 信号中心频率2MHz，带宽约20kHz
- 带通采样率只需≥40kHz
- 但需要精确的抗混叠带通滤波器（中心2MHz，带宽20kHz）
- 这个滤波器不好做

## 五、10秒时间限制的分配

题目要求"按下启动键10秒后仍无法显示结果，该项不得分"。

| 步骤 | 耗时 | 说明 |
|------|------|------|
| ADC采样 | 4ms | 4096点@1MSPS |
| Hilbert变换(FFT+IFFT) | 10ms | 4096点FFT×2 |
| 包络/频率/相位提取 | 5ms | 遍历4096点 |
| 特征计算(均值/方差/峰度) | 2ms | |
| 分类判决 | <1ms | 几个if-else |
| 参数估计 | 2ms | |
| 显示 | 50ms | LCD刷新 |
| **总计** | **<75ms** | **远小于10s** |

时间非常充裕。可以做多次采样取平均提高可靠性。
建议：采样5次，每次独立分类，取多数投票结果。
5次×75ms = 375ms，仍然远小于10s。
