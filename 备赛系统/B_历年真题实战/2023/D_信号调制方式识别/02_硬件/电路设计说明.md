# 2023 D 信号调制方式识别 —— 电路设计

## 系统总图
```
[RF 输入] → [混频器 NE602] → [基带滤波] → [双 ADC IQ 采样]
                                              ↓
                                     [STM32F407 + DSP]
                                              ↓
                                          [OLED 显示]
```

## 关键
- 中频 IF = 25kHz, IQ 双通道下变频
- 双 ADC 同步采样 IQ 通道
- 200 kSPS 采样足够覆盖 10 kHz 调制带宽
