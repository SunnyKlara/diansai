# 2023 A 单相逆变器并联 —— 接线表

> ⚠️ **强电警示**：母线 36~48V DC + 输出 24V AC，**人身安全第一**！  
> 调试时务必：  
> 1. 用调压器 0V → 36V 慢慢升  
> 2. 戴绝缘手套（10kV 标准）  
> 3. 单手操作，留另一只手备用  
> 4. 一次只调一个变量

## 总图
```
[电源 36V DC（限流 1A）]
    │   ↓ 通过预充
    │ [50Ω/10W + 继电器 KMR1（高拉吸合短路）]
    │   │
    └───┴─→ [母线 470μF×2]
                │
                ↓
        ┌───────H 桥───────┐
        │  M1 上 / M2 下   │
        │  M3 上 / M4 下   │
        └────────┬─────────┘
                 │
            [L 1.5mH + C 4.7μF]
                 │
                 ↓ 24V AC 50Hz
            [负载 12Ω 5W]
                 │
            [ZMCT103C 互感器]
                 │
                 ↓
            [OPA2277 调理]
                 │
                 ↓ ADC1 IN1
        [STM32G474]
```

## 2. H 桥 MOSFET（IRF3205 × 4 + IR2110 × 2）

| # | 连接 | 备注 |
|---|---|---|
| HB1 | 母线 + → M1 漏极 / M3 漏极 | 上管接母线正 |
| HB2 | M2 漏极 / M4 漏极 → 母线 - | 下管接母线负 |
| HB3 | M1 源极 = M2 漏极 → A 中点 → 滤波电感 L1 一端 | 半桥 A |
| HB4 | M3 源极 = M4 漏极 → B 中点 → 滤波电感 L2 一端 | 半桥 B |
| HB5 | M1/M3 栅极 → IR2110 HO 输出（上管驱动）| 通过 22Ω 栅极电阻 |
| HB6 | M2/M4 栅极 → IR2110 LO 输出 | 同上 |

**栅极电阻 22Ω + 反并 10V 齐纳**（防过驱）。

## 6. 安全与保护

- [ ] 母线电容必须有放电电阻（10kΩ/5W 并联）
- [ ] H 桥外壳金属一定要接地
- [ ] 调试时**人在两米外**操作
- [ ] 换 MOSFET 前必须断电 + 短路母线放电（用 100Ω 跨母线）
- [ ] 第一次上电用 12V 试，**绝对不要直接 36V**

## 7. 接线检查表

- [ ] M1~M4 漏源未短路
- [ ] IR2110 HIN/LIN 与 MCU 对应正确
- [ ] 自举电容容量 ≥ 1μF
- [ ] 死区 ≥ 800ns（先示波器看再上电）
- [ ] 母线电容极性
- [ ] L 与 C 串并联接法
- [ ] BKIN 信号（PA12）有效（保护链路）
