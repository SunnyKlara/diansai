# 2023 A 单相逆变器并联 — 万用板布局图

> 配套：`电路设计说明.md` / `材料清单_BOM.md`  
> 板型：**3 块板分开**：功率板（10×15cm）+ 控制板（7×9cm）+ 显示板（5×7cm）  
> 关键约束：**强电（48V 母线 + 24V/2A 输出）+ 互补 PWM 死区控制**，散热与隔离是核心

---

## 总体策略（电源类核心）

```
┌─────────────────────────────────────────────────────────────────┐
│  策略 1：3 板分开 —— 功率板 + 控制板 + 显示板，物理隔离         │
│  策略 2：功率板用厚铜（2oz），最差也要 1oz；走线宽 ≥ 2mm         │
│  策略 3：MOS 周围加散热片，并联用导热硅脂                        │
│  策略 4：控制板用排针接到功率板，**严禁** 强弱电共板             │
│  策略 5：测试点 8 个，强电板与弱电板各 4 个                      │
└─────────────────────────────────────────────────────────────────┘
```

## 关键 MOS 布局（功率板）

```
       散热片 1（A 桥臂）         散热片 2（B 桥臂）
       ┌──────────────┐           ┌──────────────┐
       │  上管 IRFP260 │           │  上管 IRFP260 │
       │              │           │              │
       │  下管 IRFP260 │           │  下管 IRFP260 │
       └──────────────┘           └──────────────┘
       
散热片选型：100×60×30mm 铝散热片 ×2，导热硅脂涂抹
工作时 MOS 温升预估：48V × 2A × 95% 效率 = 5W 损耗 / 4 个 MOS = 1.25W/MOS
                    + 死区 800ns × 20kHz = 1.6% 损耗 → 共 ~7W
                    100×60mm 散热片 7W → 温升约 25°C，可接受
```

## 互补 PWM 信号布局

```
控制板 → 功率板 信号（4 根）：
- A 上管 PWM    → 灰
- A 下管 PWM    → 紫（互补）
- B 上管 PWM    → 蓝
- B 下管 PWM    → 绿（互补）
+ GND 一根（控制板地）

走线：用 4 根 0.3mm² 杜邦线，**双绞每对（上管+下管）**，长度 < 15cm
关键：上下管 PWM 不能搞反！实测：用万用表笔点 IR2104 输入端，10ms 量
       一次（运行时），上管应是 PWM，下管应是反 PWM（赛场调试常踩）
```

## 接地策略（电源板核心）

```
                      ┌──────────────────────────────┐
                      │ 48V 电池负极（PGND 起点）     │
                      └──────────┬───────────────────┘
                                 │ 1 处单点合并
              ┌────────────────────┴─────────────────────┐
              │                                         │
              ▼                                         ▼
       PGND（功率板）                           DGND（控制板）
       - 48V 母线 GND                           - STM32 GND
       - MOS 源极 GND                           - ADC GND
       - LC 滤波 GND                            - OLED GND
       
分离原因：MOS 开关瞬间地反弹电压可达数十 V，
         如果 ADC 与 MOS 共地，ADC 数据完全乱
```

## 走线规则（功率板）

| 线 | 截面积 | 走法 |
|---|---|---|
| 48V 母线 | **2mm² 厚铜（敷锡线）** | 直线最短，避免折角 |
| MOS 漏极/源极 | 1.0mm²（焊厚锡） | 短直线 |
| 栅极驱动 | 0.5mm² | 短粗，加 10Ω 串联电阻防振铃 |
| 输出 LC | 0.5mm² 实焊 | L 后 C 前，紧凑布局 |
| LC 输出 | 1.0mm² 硅胶线 | 出板到外部端子 |

## 散热（电源板必备）

```
☐ 4 个 MOS 全部加散热片（导热硅脂 + 螺丝固定）
☐ 散热片用 100×60×30mm 铝型材 ×2
☐ 长时间运行（>30min）建议加 5V 风扇（4cm × 4cm × 1cm）
☐ MOS 温度 > 60°C 时降功率到 75%（PI 输出限幅在 0.7）
```

## 故障定位 × 布局（电源题专用）

| 现象 | 用万用板布局快速定位 |
|---|---|
| 上电就过流断电 | 测 48V 母线对地：< 1Ω 一定是上下管直通；> 1MΩ 是开路 |
| MOS 一上电就烧 | IR2104 的死区 + 自举电容缺失 |
| 输出 0V | 测 PWM 输出端子：是否有 PWM 信号 |
| 输出有但不稳 | TP2 VOUT 静态测：稳态有 PI 抖动？ADC 反馈通道有问题 |
| OLED 不亮但其他正常 | 控制板的 5V 是否被功率板拉低（不应共用 5V）|

## 测试点（8 个）

```
功率板（强电）：
TP1 VDC      = 母线电压（应 36~55V）
TP2 VOUT     = 输出 RMS 电压（应 24V±0.05V）
TP3 PGND     = 功率地
TP4 12V_AUX  = 12V 辅助电源

控制板（弱电）：
TP5 V_ADC_IN = 电压采样（静态 1.65V，运行时 1.65±1V 摆）
TP6 I_ADC_IN = 电流采样（静态 1.65V，运行时 1.65±0.4V 摆）
TP7 PWM_A_H  = A 上管 PWM（应有 20kHz 脉冲）
TP8 DGND     = 数字地
```

## 安全（48V 强电板必备）

```
☐ 48V 母线**严禁** 直接焊在排针，必须用接线柱（红黑端子）
☐ 大电解 470μF 必须焊正确极性，反接会爆
☐ 上电前先用万用表测大电解两端 < 1Ω 是否短路
☐ 每次改 PWM 死区参数后**重启** 测试，禁止热改
☐ 不戴防护眼镜不上 48V（电解爆喷）
☐ 测试 3 分钟后断电摸 MOS 散热片：> 60°C → 检查死区/驱动
```
