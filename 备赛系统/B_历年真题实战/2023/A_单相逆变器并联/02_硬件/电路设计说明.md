# 2023A 电路设计说明

## 一、系统框图

```
                    ┌─────────────────────────────────────┐
                    │          逆变器1 (独立控制器)         │
 直流电源1 ──→ [母线电容] → [全桥H桥] → [LC滤波] ──┬──→ 负载RL
                    │    STM32G474 控制              │
                    └─────────────────────────────────┘
                                                     │ (并联)
                    ┌─────────────────────────────────┘
                    │          逆变器2 (独立控制器)         │
 直流电源2 ──→ [母线电容] → [全桥H桥] → [LC滤波] ──┘
                    │    STM32G474 控制              │
                    └─────────────────────────────────┘
```

## 二、全桥逆变主电路

### 原理图描述
```
        Vdc (36V~48V)
         │
    ┌────┴────┐
    │         │
  [Q1]      [Q3]        Q1,Q3: IRF3205 (高侧)
    │         │          Q2,Q4: IRF3205 (低侧)
    ├── a     ├── b      IR2110_1 驱动 Q1/Q2
    │         │          IR2110_2 驱动 Q3/Q4
  [Q2]      [Q4]
    │         │
    └────┬────┘
         │
        GND

    a ──[L=1.5mH]──┬── Vout+
                    │
                [C=4.7μF]
                    │
    b ──────────────┴── Vout-
```

### IR2110驱动电路
```
                Vdc
                 │
            [D_boot UF4007]
                 │
    VB ──[C_boot 1μF]── VS (连接到a点或b点)
                 
    STM32 PWM_H ──→ HIN ──→ HO (驱动高侧Q1或Q3)
    STM32 PWM_L ──→ LIN ──→ LO (驱动低侧Q2或Q4)
    
    VCC = 12V (独立12V电源)
    VDD = 3.3V (逻辑电平)
```

### 关键设计参数

| 参数 | 值 | 计算依据 |
|------|-----|---------|
| 母线电压 | 36~48V | 需要>24×√2=33.9V |
| 开关频率 | 20kHz | 平衡效率和THD |
| 死区时间 | 800ns | IRF3205 tdon+trise≈200ns，留余量 |
| 滤波电感 | 1.5mH | 截止频率约2kHz |
| 滤波电容 | 4.7μF | 与电感配合 |
| 母线电容 | 470μF×2 | 纹波电流承受能力 |

## 三、采样电路

### 输出电压采样
```
Vout ──[R1=100kΩ]──┬──[R2=10kΩ]── GND
                    │
                    └── ADC_CH1 (PA0)
                    
分压比 = 10k/(100k+10k) = 1/11
ADC输入范围: 0~3V 对应 Vout: 0~33V
```

### 输出电流采样
```
使用ACS712-5A模块:
- 输出: 2.5V ± 185mV/A
- 接到 ADC_CH2 (PA1)
- 量程: ±5A
```

## 四、PCB布局要点

1. **功率回路最小化**: Q1→L→C→Q2 的回路面积尽量小
2. **驱动信号短走线**: IR2110到MOSFET栅极≤3cm
3. **采样点位置**: 电压采样在LC滤波器输出端
4. **数字地和功率地**: 单点连接，避免地回路干扰
5. **散热**: MOSFET加散热片，留足散热空间

## 五、注意事项

- IR2110的VCC(12V)和VDD(3.3V)必须分开供电
- 自举电容充电需要低侧先导通，启动时注意时序
- 死区时间不能太小（防止直通）也不能太大（影响THD）
- 母线电容要靠近H桥，减小寄生电感
