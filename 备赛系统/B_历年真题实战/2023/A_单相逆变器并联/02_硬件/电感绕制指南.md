# 电感绕制指南

## 为什么要自己绕电感
- 电赛用的滤波电感（1~2mH/3~5A）市面上不好买
- 自己绕可以精确控制电感量
- 成本低（磁芯+漆包线总共20元以内）

## 材料准备
| 材料 | 规格 | 数量 | 说明 |
|------|------|------|------|
| 铁粉芯磁环 | T106-26 (黄白色) | 2个 | 备用1个 |
| 漆包线 | 0.8mm (AWG20) | 3米 | 多备一些 |
| 万用表 | - | 1个 | 测电感量 |
| 砂纸 | 细砂纸 | 1张 | 刮漆包线绝缘层 |

## T106-26磁芯参数
- 外径: 26.9mm
- 内径: 14.5mm
- 高度: 11.1mm
- AL值: 36.1 nH/匝² (每匝²的电感量)
- 材料: -26 铁粉芯（黄白色标识）

## 匝数计算

目标电感量 L = 1.5mH = 1500000 nH

```
N = √(L / AL) = √(1500000 / 36.1) = √41551 ≈ 204匝
```

**实际绕制约200匝**（会有误差，绕完后实测调整）

## 绕制步骤

### 1. 准备漆包线
- 剪取约3米漆包线（200匝 × 磁环平均周长约15mm = 3m）
- 一端留10cm引线，用砂纸刮掉绝缘层（露出铜线）

### 2. 绕制
```
绕制方向：始终同一方向，不要反绕

    ┌──────┐
    │ 磁环 │  ← 漆包线从内孔穿过，绕在磁环上
    └──────┘     每穿一次 = 1匝
    
均匀分布：尽量让线圈均匀覆盖磁环一周
不要重叠：单层绕满后再绕第二层
拉紧：每匝都要拉紧贴合磁环
```

### 3. 固定
- 绕完后另一端也留10cm引线，刮掉绝缘层
- 用热缩管或胶带固定线圈，防止松散
- 两根引线焊上接线端子

### 4. 测量
- 用万用表的电感档测量（如果有）
- 或用LCR表测量
- 如果没有电感表：接入电路后通过谐振频率反算
  ```
  接一个已知电容C，测谐振频率f
  L = 1 / (4π²f²C)
  ```

### 5. 调整
- 电感偏大：拆掉几匝
- 电感偏小：加绕几匝
- 每次调整后重新测量

## 注意事项
- 漆包线绝缘层不能破损（否则匝间短路，电感量骤降）
- 绕制时不要用力过猛折弯漆包线（会断）
- 引线焊接前必须用砂纸彻底刮掉绝缘层
- 绕好的电感通大电流时会发热，注意散热
- 多绕一个备用，比赛时坏了可以直接换

## 常见问题
| 问题 | 原因 | 解决 |
|------|------|------|
| 电感量偏差大 | 匝数不准或磁芯参数偏差 | 实测后增减匝数 |
| 电感发热严重 | 线径太细或磁芯饱和 | 换粗线或换大磁芯 |
| 电感量随电流变化 | 铁粉芯在大电流下饱和 | 正常现象，选余量大的磁芯 |
