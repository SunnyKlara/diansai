# 2023 A 单相逆变器并联运行系统 —— 深度审题与方案论证

> 本题是 2025 A 三相能量回馈变流器的"前身"：单相 + 不带回馈但带并联。  
> 5 大方向中**电源类**最经典的一道，吃透后 2025 A、2025 B、2024 A 都能复用框架。  
> 评判维度：电源工程师、控制工程师、报告评审、安全审核员、架构师 五视角。

---

## 第一层：题目的精确解读

### 题目要求的精确数字

**单机部分（基本要求）**

| 项 | 要求 |
|---|---|
| 输出线电压有效值 | **24 V ± 0.5 V**（容差 ±2%） |
| 输出额定电流 | **2 A** 有效值（纯阻负载）|
| 频率 | 50 Hz |
| THD | **≤ 2%** |
| 效率 η | **≥ 88%**（满载 2A 时） |
| 负载调整率 | **≤ 0.2%**（48 mV 容差，比 2025 A 还严）|
| 母线电压 | 36~48 V DC（题目典型给定）|

**并联部分（发挥）**

| 项 | 要求 |
|---|---|
| 双机并联输出 | 4 A 有效值，电压仍 24V |
| 并网（电流源模式）电流 | 1~3 A 可设 |
| 并网电流比 | 0.5~2.0 可调（双机比例分配）|

### 隐含约束（题目没明说但必须遵守）

**约束 1**：负载调整率定义比 2025 A 严  
2025 A 是 0.3%，本题 **0.2%（48 mV）** → 闭环 PI 必须更精细。  
开环（无 PI）下，死区 + 输出阻抗导致的跌落约 **2~3V**（30~50%），**没有闭环 0 分**。

**约束 2**：效率 88% 是关键得分点  
2A × 24V = 48W 输出，88% 效率意味着 P_loss ≤ 6.5W。  
4 个 MOSFET（IRF3205, Rds_on=8mΩ）导通损耗 ≈ 4 × 2² × 8mΩ = 0.13W → 可忽略。  
**最大损耗源是开关损耗 + 电感铜损 + LC 谐振**。

**约束 3**：双机并联 ≠ 简单并联接线  
两台逆变器输出端直接连接需要：
- 频率严格同步（PLL 锁相）
- 相位偏差 < 0.5°（否则环流大）
- 幅值偏差 < 1%（否则负载分配不均）
- **下垂控制 droop control** 是工业级标准方案

**约束 4**：并网电流可设  
"电流可设"意味着逆变器从**电压源**变为**电流源**模式：
- 电压源模式：输出电压恒定，电流由负载决定
- 电流源模式：输出电流恒定，电压由电网决定
- 控制环路从"电压外环"变为"电流内环 + 锁相外环"

### 评委会怎么测试？

1. **基本要求**：接电阻负载 → 测电压 / THD / 效率 / 负载调整率
2. **并联**：两台并联接同一电阻负载 → 测合成电流（应为 4A）
3. **并网电流**：连到工频电源 → 测注入电流（应为设定值）

**关键观察**：
- 基本要求 50 分 + 报告 20 分 = 70 分（赛区一等奖线）容易拿
- 发挥 45 分中**并联（15 分）**比"并网（15+15=30）"难得多吗？**反过来！** 并联只需要两台同步，不连电网；并网需要 PLL 锁相 + 电流环，复杂得多。  
- **稳拿 70 分，争取 80 分（基本 50 + 并联 15 + 报告 18）**

## 第二层：误差与效率预算

### 负载调整率 0.2%（48 mV 容差）

输出阻抗带来的电压跌落：
- 死区 800ns / 50μs 周期 = 1.6% 等效输出阻抗 → Rdz = 24V × 1.6% / 2A ≈ 192 mΩ → 满载 2A 跌落 0.38V ❌
- 滤波电感等效 ESR ≈ 50 mΩ → 跌落 0.1V ❌
- 不闭环 → 总跌落 ≈ 0.5V → 调整率 2.1%，**远超 0.2%**

**必须电压闭环 PI**。闭环带宽 100Hz（远小于开关 20kHz，不会振荡），稳态误差消除到 < 30 mV。

### 效率 88% 预算

满载 P_out = 24V × 2A = 48W。  
P_loss ≤ 48W × (1/0.88 - 1) = **6.55 W**。

| 损耗源 | 估算 |
|---|---|
| 4 × MOSFET 导通（IRF3205, Rds_on=8mΩ）| 4 × 2² × 8mΩ = 0.13W |
| 4 × MOSFET 开关（Vds=48V, Id=2A, ts=100ns, fsw=20kHz）| 4 × ½ × 48 × 2 × 100ns × 20kHz = 0.77W |
| 电感铜损（DCR ≈ 80 mΩ）| 2² × 80mΩ = 0.32W |
| 电感铁损（@ 20 kHz / 0.5mT）| ~ 0.5W |
| 滤波电容 ESR | < 0.1W |
| 死区阶段二极管 body diode 损耗 | 4 × 0.7V × 2A × 1.6% = 0.09W |
| 驱动 IC 静态 | 2 × 100mA × 12V = 2.4W ❌ |
| **合计** | **~ 4.3W** |
| **效率** | **48 / 52.3 = 91.8%** ✅ 余量 4% |

**关键发现**：IR2110 驱动 IC 自身静态电流不可忽略！优化方向：
- 用低功耗驱动 IC（如 EG3002，静态 < 5mA）
- 或者用 MOSFET 集成驱动方案（如 LM5106）

## 第三层：MCU 与器件选型

### MCU：STM32G474RE（与 2025 A 同款）

| 优势 | 说明 |
|---|---|
| HRTIM 184ps 分辨率 | 死区 / SPWM 的精度天花板 |
| 双 ADC 同步 | 电压电流相位测量精确 |
| FMAC 协处理器 | PI 控制器硬件加速 |
| 170 MHz | 闭环带宽充足 |

### MOSFET 选型

| 型号 | Vds | Rds_on | Qg | 价格 |
|---|---|---|---|---|
| IRF3205（已用） | 55V | 8 mΩ | 146 nC | ¥2 |
| IRFB7430 | 40V | 1.3 mΩ | 250 nC | ¥8 |
| IRFB7434 | 40V | **1.4 mΩ** | 81 nC | ¥6 |

**结论**：基本要求用 IRF3205（够用）；冲刺效率（>92%）换 IRFB7434。

### 滤波电感选型

L = 1.5 mH，I = 2 A × √2 = 2.83 A 峰值  
铁粉芯 T106-26（黄白色）：
- Al ≈ 93 nH/N²
- N = √(1500000 / 93) ≈ **127 匝**
- 0.8mm 漆包线
- 饱和电流 ~ 3A（对 2.83A 峰有 6% 余量，**勉强够，建议升 T130-26**）

### 滤波电容

C = 4.7 μF / 100V CBB
- LC fc = 1/(2π√(1.5e-3 × 4.7e-6)) = **1896 Hz**
- 20 kHz 处衰减 = 20 log(20000²/1896²) = **41 dB** ✅

## 第五层：硬件难点

### 难点 1：母线电容预充电

48V 母线 + 470μF × 2 = 940μF。  
直接上电瞬间充电电流 = V × C / dt，可达 100A 量级 → 烧二极管 / 烧 MOSFET 体二极管。

**对策**：
- 加 100Ω 限流电阻 + 继电器并联
- 软件 ST_PRECHARGE 状态等 Vbus > 0.9 × Vdc 才启动 PWM

### 难点 2：死区时间与 THD 平衡

死区太短（< 500ns）：上下管直通烧管。  
死区太长（> 2μs）：输出畸变 → THD 升 → 失分。

**经验值**：800 ns（DTG = 136 @ 170MHz / 2 = 85MHz）。  
若用 IR2110 驱动 IC（关闭延迟约 200ns），死区 < 600ns 就有风险。  
保守起步 1.5μs，逐步降到 800ns 验证 THD。

### 难点 3：双机并联的同步

发挥 (1) 要求两台并联输出 4A。**最简单的实现**：
- 两台 MCU 都跑同一段 SPWM 代码，理论上频率 ± 0.01%（晶振精度）
- 输出端通过电感串接（每台 0.5 mH）增加阻抗，降低环流敏感性
- **不做下垂控制**，依赖晶振精度（实测一般够 50Hz ± 0.005Hz）

冲刺级方案：
- 一台为主（产生 50Hz 时基），另一台作从（PLL 锁相到主）
- 工业级 droop control 算法

### 难点 4：MOSFET 散热

满载损耗每管 ~ 0.2W。  
TO-220 + 散热片热阻 ~ 5°C/W → 1°C 升温（很安全）。  
但要注意：**不能直接焊到板子**，必须有散热片，否则 8°C/W 自身热阻下温度升 8°C × 4 = 32°C。

## 风险登记

| 风险 | 概率 | 影响 | 缓解 |
|---|---|---|---|
| 母线预充缺失 → 上电烧管 | 中 | 高 | 限流电阻 + 继电器 |
| 死区设置过短 → 直通 | 低 | 极高（烧 4 管）| 1.5μs 起步逐步降 |
| 闭环 PI 振荡 | 中 | 中 | 先 KP 后 KI，慢慢加 |
| 电感饱和 | 中 | 中 | T130-26 替代 T106-26 |
| 双机并联环流过大 | 高（基本要求不需要）| 中 | 串接电感 + 同步 |
