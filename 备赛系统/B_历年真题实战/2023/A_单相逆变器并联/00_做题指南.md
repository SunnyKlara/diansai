### 关键代码段

#### SPWM正弦表生成（Python预计算）
```python
import math
N = 400  # 20kHz/50Hz = 400点/周期
table = [math.sin(2 * math.pi * i / N) for i in range(N)]
# 输出为C数组格式
print("const float sine_table[400] = {")
for i in range(0, N, 10):
    line = ", ".join(f"{table[j]:.6f}f" for j in range(i, min(i+10, N)))
    print(f"    {line},")
print("};")
```

#### 电压有效值计算
```c
float calc_vrms(uint16_t* adc_buf, uint16_t len) {
    float sum_sq = 0;
    for (uint16_t i = 0; i < len; i++) {
        float v = (adc_buf[i] / 4096.0f * 3.3f - 1.65f) * VOLTAGE_SCALE;
        sum_sq += v * v;
    }
    return sqrtf(sum_sq / len);
}
```

## 第五步：调试流程

### Day2 调试检查清单
- [ ] PWM波形正确（示波器看两路互补PWM）
- [ ] 死区时间合适（500ns~1μs）
- [ ] 空载输出正弦波形正常
- [ ] 输出电压有效值可调
- [ ] 闭环稳压到24V±0.2V
- [ ] 加2A负载，电压稳定
- [ ] 测THD（目标≤2%）
- [ ] 测效率（目标≥88%）

### 常见问题速查
| 问题 | 原因 | 解决 |
|------|------|------|
| 输出波形畸变 | 死区太大 | 减小死区或软件补偿 |
| 效率低于85% | 开关损耗大 | 检查驱动波形、换管子 |
| 负载调整率差 | 控制带宽不够 | 增大PI增益 |
| 输出有直流偏置 | 两桥臂不对称 | 校准PWM占空比 |
