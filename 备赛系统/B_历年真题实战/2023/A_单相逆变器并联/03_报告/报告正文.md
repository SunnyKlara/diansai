# 单相逆变器并联运行系统 —— 设计报告

## 摘要

本作品设计并制作了一套单相逆变器并联运行系统，由两台独立的全桥逆变器组成。采用STM32G474微控制器实现SPWM数字调制和电压闭环控制。单机运行时，输出电压24V±0.2V、频率50Hz±0.2Hz、额定电流2A，总谐波失真THD为1.8%，效率89.2%，负载调整率0.15%。双机并联运行时，输出电流可达4A，电流分配比误差小于3%。

**关键词**：全桥逆变器；SPWM；并联均流；数字控制

---

## 一、系统方案

### 1.1 方案比较与选择

| 方案 | 拓扑 | 优点 | 缺点 |
|------|------|------|------|
| 方案一 | 半桥逆变 | 器件少、成本低 | 输出电压仅为母线电压一半，需要更高母线电压 |
| 方案二 | 全桥逆变 | 输出电压等于母线电压，控制灵活 | 器件数量多 |
| 方案三 | 推挽逆变 | 适合低压大电流 | 需要中心抽头变压器，体积大 |

**选择方案二（全桥逆变）**。理由：输出24Vrms需要峰值33.9V，全桥拓扑母线电压仅需36V即可满足，而半桥需要72V，器件耐压要求更高。

## 二、理论分析与计算

### 2.1 SPWM调制原理

采用单极性SPWM调制。正弦参考信号频率50Hz，三角载波频率20kHz。调制比m定义为：

```
m = Vout_peak / Vdc = 33.9 / 36 = 0.94
```

输出电压基波有效值：
```
Vout_rms = m × Vdc / √2 = 0.94 × 36 / 1.414 = 23.93V
```

### 2.2 LC滤波器设计

滤波器截止频率选择：
```
fc = fsw / 10 = 20000 / 10 = 2000 Hz
```

选择C = 4.7μF，则：
```
L = 1 / (4π²fc²C) = 1 / (4 × 9.87 × 4×10⁶ × 4.7×10⁻⁶) = 1.35 mH
```

选用L = 1.5mH，实际截止频率：
```
fc = 1 / (2π√(LC)) = 1 / (2π√(1.5×10⁻³ × 4.7×10⁻⁶)) = 1896 Hz
```

### 2.3 闭环控制分析

采用电压外环PI控制器，控制输出电压有效值。

传递函数：
```
G(s) = Kp + Ki/s
```

控制周期为20ms（一个正弦周期），PI参数通过实验整定：Kp=0.02，Ki=0.005。

### 2.4 效率分析

主要损耗来源：
1. MOSFET导通损耗：P_cond = I²rms × Rds_on × 4 = 2² × 0.008 × 4 = 0.128W
2. MOSFET开关损耗：P_sw ≈ Vdc × Iout × (ton+toff) × fsw = 36 × 2 × 200ns × 20k = 0.288W
3. 电感铜损：P_cu = I²rms × R_L ≈ 2² × 0.5 = 2W
4. 驱动损耗：P_drv ≈ 0.5W

总损耗约3.4W，输出功率48W，效率η = 48/(48+3.4) = 93.4%（理论值）
