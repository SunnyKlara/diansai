# 2023 A 单相逆变器并联 —— STM32G474RE 引脚分配表

> 平台：STM32G474RE Nucleo（NUCLEO-G474RE，¥45）  
> 选 G474 因为 HRTIM 184ps 高分辨率，SPWM 死区精确到 ns 级。

## 1. 完整引脚

| 引脚 | 功能 | 外设 | 备注 |
|---|---|---|---|
| **H 桥 SPWM 互补输出（HRTIM_A）** ||||
| PA8 | HRTIM_CHA1 | 上管 PWM A | TIM1_CH1 也可用 |
| PB13 | HRTIM_CHA1N | 下管 PWM A（互补 + 死区）| |
| PA9 | HRTIM_CHA2 | 上管 PWM B | |
| PB14 | HRTIM_CHA2N | 下管 PWM B | |
| **死区** | 800 ns | TIM1 BDTR.DTG = 0x88 | 见 SysConfig |
| **MOSFET 故障保护** ||||
| PA12 | TIM1_BKIN | 硬件比较器输出 | 过流即关 PWM |
| **ADC 同步双通道** ||||
| PA0 | ADC1_IN1 | V_out 输出电压 | OPA2277 调理 |
| PA1 | ADC2_IN2 | I_out 输出电流 | ZMCT103C |
| PC0 | ADC1_IN6 | V_bus 母线电压 | 高阻分压 |
| **OLED I2C** ||||
| PB8 | I2C1_SCL | OLED | 4.7k 上拉 |
| PB9 | I2C1_SDA | OLED | |
| **按键 + LED** ||||
| PC13 | GPIO | 板载用户按键 | 启动逆变器 |
| PA5 | GPIO | 板载 LED | 状态 |
| **预充继电器** ||||
| PB10 | GPIO | 继电器 KMR1 | 高有效，吸合短路限流电阻 |
| **调试串口** ||||
| PA2 | LPUART1_TX | ST-Link VCP | 115200 |
| PA3 | LPUART1_RX | | |

## 2. 时钟

| 项 | 值 |
|---|---|
| HSE | 24 MHz Nucleo 板 |
| PLL | × 200/48 = 100 → ÷? |
| SYSCLK | 170 MHz |
| HRTIM 时钟 | 170 × 32 = 5.44 GHz（虚拟，184ps 分辨率）|

## 3. HRTIM SPWM 配置

| 项 | 值 |
|---|---|
| 模式 | TIM_A 主控 / 中心对齐 PWM |
| 频率 | 20 kHz |
| 周期 | 5440000 / 20000 = 272 ticks（HRTIM 内部）|
| 死区 | DTG = 800ns × 32 / 1ns = 25600 → 但实际寄存器 0x88 |
| 互补 | 自动反相（CHA1 ↔ CHA1N）|

## 4. ADC 同步

| 项 | 值 |
|---|---|
| 触发 | HRTIM_TIM_A 比较事件 |
| 同步采样点 | PWM 中点（避免开关纹波）|
| DMA | DMA1 Stream0 双通道 |

## 5. 中断

| 中断 | 优先级 | 用途 |
|---|---|---|
| HRTIM TRGO（20kHz） | 0 | SPWM 角度推进 + 控制环 |
| ADC1 EOS | 1 | 推 V/I 入控制 |
| TIM1_BRK_TIM15（BKIN）| 0 | **硬件保护，最高优先级**|
| Periodic 1ms | 2 | 系统 tick |
