# 2023 A 单相逆变器并联 — 独到经验总结

> 单相逆变器是电源类入门题，但 0.2% 负载调整率 + 88% 效率不容易。
> 8 个赛前必知的坑全部踩一遍后还能拿国二，避开 5 个就能国一。

---

## 一、赛前必知的 8 个坑

### 坑 1：220V 类比安全 — 36~48V 母线一样会咬人

虽然不是市电级别，但母线电容（470μF × 2）储能足以损坏元件甚至烧手。

**对策**：
- 调试时**不要徒手碰板子的功率部分**
- 改硬件接线 → 拔母线插头 → **等 30 秒**（电容放电）→ 万用表测 < 5V → 才动手
- 加 10kΩ 放电电阻让电容 < 30 秒自然放电

### 坑 2：负载调整率 0.2% — 必须电压闭环

开环时死区 + 输出阻抗导致 2~3V 跌落，远超 48 mV 容差。  
**没闭环 = 14 分（基本要求 4）直接没**。

调参经验：
1. 关闭 I 项（KI=0），调 KP 到稳态误差减半（KP 从 0.005 翻倍）
2. 出现轻微振荡时把 KP 退一档
3. 加 I 项（KI 从 KP/100 起步）
4. 满载切空载收敛 < 1s 即达标

### 坑 3：效率 88% — IR2110 静态电流是头号 killer

IR2110 自身静态电流约 100mA × 12V = 1.2W，2 颗就 2.4W。  
满载损耗预算 6.5W 中**不知不觉占了 37%**！

**对策**：
- 用低功耗驱动 IC（EG3002, IR2117, LM5106 等，静态 < 5mA）
- 或用 N+P 互补对（不需要自举，但驱动复杂）
- IR2110 + 优化驱动电源（VCC 用 10V 而不是 15V）→ 损耗减半

### 坑 4：死区时间 800ns 是经验值，不是绝对值

死区太短（< 500ns）：上下管直通烧管。  
死区太长（> 2μs）：输出畸变 → THD 升 → 失分。

**调参流程**：
1. 起步 1.5μs，验证 THD ≈ 5%（不达标）
2. 减到 1.0μs，THD ≈ 3%（仍不达标）
3. 减到 800ns，THD ≈ 1.5%（达标 + 余量 0.5%）
4. 再减到 600ns 风险高，**不建议**

### 坑 5：母线预充电

48V 母线 + 940μF 电容，上电瞬间充电电流可达 100A。  
**没预充电 = 烧二极管 / 烧 MOSFET 体二极管 / 跳保险丝**。

**对策**：
- 加 100Ω 限流电阻 + 5A 继电器，限流电阻预充 → 继电器旁路
- 软件 ST_PRECHARGE 状态等 Vbus > 0.9 × Vdc 才闭合继电器 + 启动 PWM

### 坑 6：滤波电感饱和

T106-26 黄白色铁粉芯，1.5mH @ 127 匝时饱和电流 ~ 3A。  
满载 2A × √2 = 2.83A 峰值 → 余量 6%（接近饱和！）

**症状**：饱和后输出畸变 + 电感发烫 + 过流报警。

**对策**：选 **T130-26**（饱和电流 5A，余量充足）。**多花 ¥3 救命**。

### 坑 7：双机并联同步

简单做法（赛区一等奖够）：
- 两台 MCU 都跑同一段 SPWM 代码
- 各自有独立晶振，频率 ± 0.01% 偏差
- 实测同步误差 < 0.5°，环流可接受
- 输出端串接 0.5mH 电感增加阻抗，进一步降低环流敏感性

工业级做法（国一冲刺）：
- 一台为主，另一台 PLL 锁相
- 下垂控制（droop control）按下式：  
  `f = f0 - kd · P`（频率下垂）  
  `V = V0 - kv · Q`（电压下垂）
- **赛前 1 个月开始练**，否则现场调不出来

### 坑 8：闭环带宽与开关频率耦合

电压闭环 KP/KI 设大了 → 闭环带宽接近开关频率 → 振荡。  
**对策**：
- 闭环带宽设为开关频率的 1/100（20 kHz / 100 = 200 Hz）
- KP 不要超过 0.1
- KI 不要超过 0.05

## 三、链路验证标准流程

### Level 0：单模块离线验证

| 模块 | 验证 | 通过判据 |
|---|---|---|
| MCU 闪灯 | 编译烧录 | LED 闪 |
| 单 PWM 通道 | 占空比 50% | 万用表测电压 = Vdc/2 |
| 双互补 PWM | 死区 800ns | 示波器看上下沿不重叠 |
| ADC 单通道 | 接已知直流 | 数值与万用表对应 |
| ADC 双通道 | 接同一信号源 | 两通道读同值 |
| OLED | 显示文字 | 显示正常 |

### Level 1：开环全链路验证

1. 设 m = 0.7 / f = 50 Hz 开环
2. 接 12Ω 负载（≈ 2A 电流）
3. 测线电压 RMS（万用表交流档）= **20~22V**（开环理论值）
4. 测 THD（用功率分析仪或 FFT 软件）= **5~10%**（开环理论值）

### Level 2：闭环单机验证

1. 启用电压闭环 → Vrms 收敛到 24.0V ± 0.05V
2. 加载 0~2A → 电压波动 < 50mV（满足 0.2%）
3. 测 THD（满足 ≤ 2%）
4. 测效率（功率分析仪测 P_in / P_out）

### Level 3：双机并联验证

1. 两台都收敛到 24.0V 单机模式
2. 输出端通过电感串接连一起
3. 接 6Ω 负载（4A 电流）
4. 用电流钳分别测两台输出电流，应各为 2A ± 0.2A
5. 振荡 → 退回单机模式

## 四、保命级安全规则

⚠️ **48V 母线虽不致命但易烧元件**。

1. 通电时**手不接触功率部分**
2. 测电压用**夹子探头**，不要用手扶
3. 改硬件 → 拔插头 → 等 30 秒 → 万用表测确认 0V → 才动手
4. 备用熔断器 1A，烧熔断器不烧主板
5. 使用 RCD 漏电保护插座（防止市电意外接入）
