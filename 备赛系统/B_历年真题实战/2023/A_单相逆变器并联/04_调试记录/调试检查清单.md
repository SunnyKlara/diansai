# 2023A 调试检查清单

## Day1 焊接完成后检查

- [ ] 万用表检查电源对地无短路
- [ ] IR2110 VCC(12V)和VDD(3.3V)电压正确
- [ ] STM32核心板供电正常，能下载程序
- [ ] MOSFET栅极到驱动IC连线完好
- [ ] 自举电容和二极管方向正确
- [ ] LC滤波器焊接牢固
- [ ] 采样电路分压电阻值正确

## Day2 软件调试

### 第一步：PWM输出验证（不接功率级）
- [ ] 示波器看PA8/PA9的PWM波形
- [ ] 确认互补输出，死区时间约800ns
- [ ] 确认SPWM调制波形（占空比正弦变化）
- [ ] 确认输出频率50Hz（一个正弦周期20ms）

### 第二步：开环功率级测试
- [ ] 接上直流电源（先用低电压如12V）
- [ ] 空载看LC滤波后输出波形
- [ ] 确认输出为正弦波，无明显畸变
- [ ] 逐步升高母线电压到36V
- [ ] 测量输出电压有效值

### 第三步：ADC采样验证
- [ ] 用万用表测量采样点电压，与ADC读数对比
- [ ] 电压采样缩放系数校准
- [ ] 电流传感器零点校准（无电流时应为2.5V）
- [ ] RMS计算结果与万用表读数对比

### 第四步：闭环调试
- [ ] 使能PI控制器
- [ ] 设定目标24V，观察输出是否稳定
- [ ] 如果振荡：减小Kp
- [ ] 如果稳态误差大：增大Ki
- [ ] 加载2A，观察电压跌落和恢复
