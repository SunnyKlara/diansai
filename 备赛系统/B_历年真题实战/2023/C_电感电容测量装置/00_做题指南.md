## 测量方案：I-V法（推荐）

```
DDS信号源 → [已知电阻R0] → [被测元件Zx] → GND
              │                │
           ADC_CH1(Vr)      ADC_CH2(Vx)

Zx = R0 × Vx / (Vr - Vx)  (复数运算)
```

### 电容测量
```
Zx = 1/(jωC) = -j/(2πfC)
|Zx| = 1/(2πfC)
C = 1/(2πf|Zx|)
D = Re(Zx)/|Im(Zx)| = ESR × 2πfC
```

### 电感测量
```
Zx = jωL + R_dc = j2πfL + R_dc
|Zx| = √((2πfL)² + R_dc²)
L = Im(Zx)/(2πf)
Q = Im(Zx)/Re(Zx) = 2πfL/R_dc
```

### 关键：相位测量
D值和Q值本质上是**相位测量**。需要精确测量Vr和Vx的相位差。

方法：对Vr和Vx同时采样，做DFT提取幅值和相位。

```c
// DFT提取基波幅值和相位
void dft_extract(uint16_t* data, uint16_t N, float* amplitude, float* phase)
{
    float sum_sin = 0, sum_cos = 0;
    for (int i = 0; i < N; i++) {
        float angle = 2.0f * PI * i / N;
        float val = (float)data[i];
        sum_sin += val * sinf(angle);
        sum_cos += val * cosf(angle);
    }
    *amplitude = 2.0f * sqrtf(sum_sin*sum_sin + sum_cos*sum_cos) / N;
    *phase = atan2f(sum_sin, sum_cos);
}
```
