# 2023 C 电感电容测量装置 —— 电路设计

## 系统总图
```
[AD9833 DDS] → 10kHz 正弦 → [R_ref 1kΩ 0.1%] → [DUT (L 或 C)] → GND
                                  ↑              ↑
                            [OPA2277 缓冲]    [OPA2277 缓冲]
                                  ↓              ↓
                              [ADC1 IN0]     [ADC2 IN1]  ← STM32F103 双 ADC 同步
                                  ↓
                              [DFT @ 10kHz]
                                  ↓
                              [OLED]
```

## 关键
- R_ref **必须 0.1% 精密**（金属膜或电阻箱），否则测量误差直接传递
- 4 线开尔文连接（电流端 + 电压端分开）消除接触电阻
- DUT 端 OPA2277 输入阻抗 > 1MΩ，避免分流
