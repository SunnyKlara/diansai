# 2023 C 独到经验

## 1. 单频 DFT 比 FFT 划算
只关心一个频点，单频 DFT 复杂度 O(N) vs FFT O(N log N)，实测 1024 点 STM32F103：
- FFT: 18 ms
- 单频 DFT: 0.4 ms
50 倍速度差！而且不需要存 FFT 实例。

## 2. 接触电阻是 1Ω 量级
普通鳄鱼夹接触电阻 0.5~1Ω，测 10mΩ 级电感铜阻直接淹没。**4 线开尔文**必须做。

## 3. DDS 谐波抑制
AD9833 输出含 3 次谐波（约 -10dBc）→ DUT 上 10kHz 与 30kHz 同时存在。**对策**：DDS 输出加 LC 低通 fc=20kHz。
