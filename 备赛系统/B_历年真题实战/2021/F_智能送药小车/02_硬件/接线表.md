# 2021 F 智能送药小车 —— 接线表

> 双车系统：每辆车独立装配，**编号 1 / 2 区分**，软件通过编译宏 `CAR_ID=1` / `CAR_ID=2` 切换。

## 8. 接线检查表

- [ ] 电池极性正确
- [ ] STBY 万用表测 = 3.3V（启动后软件拉高）
- [ ] NRF24 接 3.3V，**不要 5V**
- [ ] OpenMV 安装位置不晃动
- [ ] 5 路 IR 阵列居中对称
- [ ] 共地连通（万用表二极管挡测）
- [ ] 双车 NRF24 通道一致（Channel 76）

## 9. 双车区分

|  | 车 1（送药）| 车 2（接药）|
|---|---|---|
| 编译 | `-DCAR_ID=1` | `-DCAR_ID=2` |
| 颜色标识 | 红 LED 常亮 | 绿 LED 常亮 |
| NRF24 地址 | RX=`0xCAR1`, TX=`0xCAR2` | 反过来 |

## 10. 常见错误

| 现象 | 原因 |
|---|---|
| 上电瞬间小车冲出 | STBY 上电默认低 → 软件需立即拉高，且 PWM 默认 0 |
| OpenMV 串口丢字 | 波特率不匹配 / 共地缺失 |
| NRF24 收不到包 | 双车地址写反 / 通道不一致 / VCC 抖动 |
| 编码器只有左有数 | TIM3/4 都要配编码器模式 |
| 数字识别误判 | 镜头脏 / 光线不一致 / 阈值未调 |
