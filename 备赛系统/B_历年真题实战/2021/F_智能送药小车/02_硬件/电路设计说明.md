# 2021 F 智能送药小车 —— 电路设计说明

## 系统总图

```
[OpenMV H7] ─UART─→ [STM32F103] ─PWM─→ [TB6612] ─→ [电机×4]
                       ↑           ↑
                  [5×IR 红外]   [编码器×2]
                       ↑
                  [NRF24L01 与对方车通信]
                       ↑
                  [LED 红/绿/黄] + [蜂鸣器]
                       ↑
                  [微动开关 装载/卸载]
```

## 关键模块

### 红外循迹板
- ITR9606 × 5
- 每路 100Ω 上拉，OPA2350 比较器
- 阈值 1.5V 比较器中心

### 视觉模块（OpenMV H7）
- 80×60 子画面，Haar Cascade 数字分类器
- UART 1 Mbps → STM32

### TB6612 电机驱动
- 双通道，1.2A/通道
- VCC = 7.4V（2S 锂电）
- BIN1/BIN2 PWM 控制方向

### NRF24L01+PA
- 2.4GHz, 250kbps
- 32 字节包，每 50ms 心跳

## 复用关系
- 与 2024 H 自动小车：90% 复用 motor / encoder / ir_line / pid
- 与 2022 C 小车跟随：100% 复用此骨架
