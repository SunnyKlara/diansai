## Day2 调试

### 单车循迹（红色引导线）
- [ ] 红外传感器能区分红线和白色地面
- [ ] PID循迹沿红线稳定行驶
- [ ] 能识别T字路口（转弯点）

### 数字识别
- [ ] OpenMV能识别数字1~8
- [ ] 识别时间≤20s
- [ ] 识别准确率≥90%

### 路径规划
- [ ] 根据病房号选择正确路径
- [ ] 到达目标病房门口停车
- [ ] 红灯亮，等待卸载

### 装载/卸载检测
- [ ] 放上药品→微动开关触发→自动启动
- [ ] 取下药品→微动开关释放→自动返回

## Day3 联调

### 基本要求
- [ ] 送药到1号病房≤20s ✓/✗
- [ ] 送药到中部病房(随机)≤20s ✓/✗
- [ ] 送药到远端病房≤20s ✓/✗
- [ ] 返回药房≤20s ✓/✗

### 发挥部分（双车协同）
- [ ] 小车1送药，小车2等待
- [ ] 小车1卸载后返回，通知小车2
- [ ] 小车2继续送药
- [ ] 全程无碰撞

## Day4 最终
- [ ] 基本要求3项全部通过
- [ ] 指示灯状态正确（红/绿/黄）
- [ ] 报告打印
- [ ] 场地布置完成

## 常见故障

| 问题 | 原因 | 解决 |
|------|------|------|
| 红线循迹不稳 | 传感器对红色不敏感 | 调灵敏度或换灰度传感器 |
| 数字识别错误 | 光线变化 | 调OpenMV阈值 |
| T字路口走错 | 路口检测逻辑有误 | 加编码器计距辅助判断 |
| 双车碰撞 | 通信延迟 | 加安全距离检测 |
