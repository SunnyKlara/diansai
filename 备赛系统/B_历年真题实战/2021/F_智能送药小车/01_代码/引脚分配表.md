## 2. 引脚冲突说明

STM32F103C8 引脚紧张，**必须避免以下冲突**：
- 5 路 IR + UART2（OpenMV）：IR 用 ADC1_IN2~5（PA2/3/4/5），UART2 也在 PA2/3 → **改用 UART3 (PB10/PB11)** 或外部 ADC（MCP3008）
- SPI1（NRF24）+ 编码器：都用 PA5~7 → **NRF24 改用 SPI2 (PB12/13/14/15)**

**最终方案**：
| 模块 | 接口 | 引脚 |
|---|---|---|
| 5 路 IR | ADC1 | PA2/3/4/5/8 |
| OpenMV | UART3 | PB10/PB11 |
| NRF24 | SPI2 | PB12/13/14/15 |
| 编码器 | TIM3/4 | PA6/7 + PB6/7 |
| 调试 | UART1 | PA9/10 |
| 电机 | TIM2/PWM | PA0/1 |

## 3. 时钟配置

| 项 | 值 |
|---|---|
| HSE | 8 MHz 板载晶振 |
| PLL × 9 | SYSCLK = 72 MHz |
| AHB | 72 MHz |
| APB1 | 36 MHz（TIM 倍频后 72 MHz）|
| APB2 | 72 MHz（ADC / TIM1 / USART1）|

## 4. PWM 周期推导

```
PWM 频率 = 1 kHz（推荐用于直流电机）
TIM2 时钟 = 72 MHz
PSC = 71 → 计数频率 = 1 MHz
ARR = 999 → 1 MHz / 1000 = 1 kHz 周期
占空比分辨率 1000（CCR 0..999）
```

## 5. 中断列表

| 中断 | 优先级 | ISR | 用途 |
|---|---|---|---|
| TIM6 (10ms) | 1 | `Tick10ms_ISR` | 主控制环 |
| USART3 RX | 2 | `USART3_IRQHandler` | OpenMV 数据接收 |
| EXTI3 (NRF24 IRQ) | 2 | `EXTI3_IRQHandler` | 双车通信 |
| EXTI11/12 (KEY) | 4 | `EXTI15_10_IRQHandler` | 按键 |
| USART1 RX | 5 | `USART1_IRQHandler` | 调试 |

## 6. 调试节奏

1. 单测电机 PWM（万用表测 PA0/PA1 应有 1kHz 方波）
2. 单测编码器（手转轮子，串口打印计数）
3. 单测 5 路 IR（ADC 读数 0~4095）
4. 单测 OpenMV UART（串口模拟接收 0xAA + 数字）
5. 单测 NRF24（双板对发对收）
6. 集成主循环
