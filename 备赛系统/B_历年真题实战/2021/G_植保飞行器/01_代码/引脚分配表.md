## 2. MAVLink 通信

PX4 默认 TELEM2 输出 MAVLink v2 / 921600 波特率。任务 MCU 解析以下消息：
- `HEARTBEAT` (#0)：飞控状态
- `GLOBAL_POSITION_INT` (#33)：GPS 经纬高
- `MISSION_ITEM` (#39)：航点
- 自定义命令通过 `COMMAND_LONG` (#76)

## 3. 时钟
```
HSE 8 MHz × 21 = 168 MHz
APB1 42 MHz, APB2 84 MHz
```

## 4. 中断优先级

| 中断 | 优先级 |
|---|---|
| USART4 (Pixhawk MAVLink) | 1（高频心跳）|
| USART1 (OpenMV) | 2 |
| USART2 (LoRa) | 3 |
| EXTI (KEY) | 4 |
| TIM6 (10ms tick) | 1 |

## 5. 飞控决策

**重要**：任务 MCU **不直接控飞机**，而是通过 MAVLink 给 Pixhawk 发指令。Pixhawk 跑 PX4 完成稳定 + 导航 + 安全 fallback。

任务 MCU 只做：
1. 接收 OpenMV 视觉目标
2. 决定航点 / 喷洒时机
3. 通过 LoRa 上报状态

**自研飞控不推荐**——太难，6 个月做不完。
