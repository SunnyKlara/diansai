# 2021 G 植保飞行器 —— 接线表

> 飞控 + 任务 MCU 双层架构：飞控 Pixhawk 4 跑 PX4，任务 MCU 跑视觉 + 喷洒 + 通信。

## 总图

```
[Pixhawk 4 飞控（已烧 PX4）]
   │ MAVLink (UART4)
   ↓
[STM32F407 任务 MCU]
   ├── UART1 ← OpenMV H7（顶视目标识别）
   ├── UART2 ← LoRa 433MHz（地面站通信）
   ├── GPIO  → 蠕动泵 12V（经继电器）
   └── GPIO  → 状态 LED + 蜂鸣器

[Pixhawk]──→ [4 ESC]──→ [无刷电机 ×4]
[Pixhawk]──→ [GPS 模块] (M9N)
[Pixhawk]──→ [遥控接收机 SBUS]
[Pixhawk]──→ [电池 4S 5200mAh]
```

## 1. 飞控接线（Pixhawk 4 标准接法）

按 PX4 官方文档接线，**不重复造轮子**：
- POWER：电池 + 限流模块
- MAIN OUT 1~4：4 个 ESC 信号
- TELEM2：MAVLink 输出（接任务 MCU）
- GPS1：M9N
- RC IN：遥控接收机

## 2. 任务 MCU ↔ 飞控（MAVLink）

| # | 任务 MCU | Pixhawk TELEM2 | 颜色 |
|---|---|---|---|
| F1 | PA0 (USART4 TX) | RX | 黄 |
| F2 | PA1 (USART4 RX) | TX | 蓝 |
| F3 | GND | GND | 黑 |

> Pixhawk TELEM2 默认 921600 波特率，**MAVLink 协议**。

## 3. OpenMV H7（顶视）

| # | F407 | OpenMV | 颜色 |
|---|---|---|---|
| O1 | PA9 (USART1 TX) | P5 (RX) | 黄 |
| O2 | PA10 (USART1 RX) | P4 (TX) | 蓝 |
| O3 | GND | GND | 黑 |
| O4 | 5V (从飞控 BEC) | VIN | 红 |

> OpenMV 装机身**正下方**，镜头朝地，30~50cm 高度看得清植物。

## 4. LoRa 433MHz

| # | F407 | LoRa（如 RA-02）| 颜色 |
|---|---|---|---|
| R1 | PA2 (USART2 TX) | RX | 黄 |
| R2 | PA3 (USART2 RX) | TX | 蓝 |
| R3 | 3.3V | VCC | 红 |
| R4 | GND | GND | 黑 |

> LoRa 模块功耗大，VCC 加 100μF 电解电容稳压。

## 5. 蠕动泵控制

| # | F407 | 元件 | 备注 |
|---|---|---|---|
| W1 | PB0 | 1kΩ + 8050 NPN | 三极管驱动 |
| W2 | 三极管 C | 5V 继电器线圈 - | |
| W3 | 3.3V | 继电器线圈 + | |
| W4 | 继电器触点常开 | 蠕动泵 + | |
| W5 | 12V (从飞控)| 蠕动泵 + 电源 | WP-200 12V/0.3A |
| W6 | GND | 蠕动泵 - | |

## 6. 状态指示

| # | F407 | 元件 |
|---|---|---|
| L1 | PD12 | LED_R 红（任务运行）|
| L2 | PD13 | LED_G 绿（GPS 锁定）|
| L3 | PD14 | LED_B 蓝（喷洒进行）|
| L4 | PE5 | 蜂鸣器（完成提示）|

## 7. 接线检查

- [ ] 飞控 + 任务 MCU **共地**
- [ ] OpenMV 镜头清洁，固定不晃
- [ ] 蠕动泵 12V 路独立，不能用 3.3V 直驱
- [ ] LoRa 天线远离飞控 GPS（互扰）
- [ ] 电池 4S 满电 16.8V，**不要超充**

## 8. 软件准备清单

- [ ] Pixhawk 烧 PX4 v1.14（开源）
- [ ] QGroundControl 地面站调参
- [ ] OpenMV IDE 烧颜色检测脚本
- [ ] 任务 MCU 烧 STM32 工程（本仓库代码）
- [ ] LoRa 模块烧默认 433MHz / 9600 配置

## 9. 启动顺序

1. **室内**调试：拆桨 + 飞控 USB 连 QGC，校准 + 调 PID
2. **场外**起飞前：GPS 锁定 ≥ 8 颗，电池 ≥ 16.0V
3. **遥控试飞**：先手动飞 30s 检查无异常
4. **任务起飞**：上传航点，自主飞行 + 视觉识别 + 喷洒
