## 核心任务：全覆盖路径规划

### 作业区布局
- 400cm×500cm区域
- 28个50cm×50cm播撒区块
- 灰色为非播撒区
- 激光笔闪烁1~3次=正常播撒

### 弓字形扫描路径
```
起点A(21号) → 20 → 19 → ... → 1号
然后折返 → 22 → 23 → ... → 28号
```
按行扫描，每行覆盖一排区块。

### 激光笔控制
```c
// 进入一个区块后闪烁激光笔
void spray_block(void) {
    for (int i = 0; i < 2; i++) {  // 闪烁2次
        laser_on();
        delay_ms(500);   // 亮0.5s
        laser_off();
        delay_ms(500);   // 灭0.5s
    }
}
// 注意：同一区块>3次闪烁=重复播撒(扣分)
```

## 定位方案
- UWB基站定位（精度高，推荐）
- 或利用场地边缘黑色标志线+光流
- 需要精确到50cm区块级别

## 关键提醒
- 360s时间限制很紧，路径规划要高效
- 激光笔垂直向下固定，不可移动转动
- 飞行全程不得人为干预
- 桨叶必须全防护
- 佩戴防护眼镜和手套
