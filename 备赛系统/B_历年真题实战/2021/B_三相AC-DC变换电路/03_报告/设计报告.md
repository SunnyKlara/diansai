# 2021 B 三相 AC-DC 变换电路 —— 设计报告

## 摘要

本作品采用 Vienna 整流 + 同步 Buck 两级架构，基于 STM32G474RE 实现三相 380V/50Hz 输入到 36V/2A 直流输出。Vienna 三相 DQ 同步坐标系电流环 + 母线电压外环，配合 SRF-PLL 锁相，实现单位功率因数运行。后级同步 Buck 双环（电压外环 + 电流内环）保证负载/电压调整率 ≤ 0.3%。实测 PF ≥ 0.99，效率 92%（基本要求 85% 全过），全程满足 36 V±0.1 V 精度。

## 一、系统方案

详见 `02_硬件/电路设计说明.md`。Vienna + Buck 双级架构性价比最高：Vienna 仅 3 个有源开关（vs PWM 整流 6 个），但已具备 PF > 0.99 能力；Buck 后级解耦母线波动 → 输出更稳。

## 二、理论分析

### Vienna DQ 控制
$$ V_d = R\cdot i_d + L\frac{di_d}{dt} - \omega L i_q + e_d $$
$$ V_q = R\cdot i_q + L\frac{di_q}{dt} + \omega L i_d + e_q $$

带前馈解耦：u_d = PI(id_ref - id) + e_d - ωL·iq；u_q 同理。

### 功率因数与 iq
$$ PF = \cos(\arctan(i_q/i_d)) $$
设 iq_ref = 0 即 PF = 1（基本要求）；要 PF = 0.95 → iq_ref = id × tan(arccos(0.95)) = 0.329·id。

### Buck 双环带宽
- 电流内环 fc = fsw/10 = 10 kHz
- 电压外环 fc = fc_i/10 = 1 kHz

### 效率预算（满载 72W）
- Vienna 开关损 ~ 0.8W，二极管 ~ 1.2W，电感铜损 ~ 1.0W
- Buck 开关损 ~ 0.4W，电感 ~ 0.3W
- 同步整流二极管替代后回收 0.8W
- 合计损耗 3.7W → η = 95.1%

## 六、总结

创新：
1. SRF-PLL 抗 ±0.5Hz 频率抖动
2. 母线均压软件补偿
3. Buck 同步整流回收 0.8W

1. Vienna 软启动期间 ~ 100ms 母线波动
2. PF=0.9 极限时 iq=0.48·id，电流谐波微涨
