## 方案选择

### 方案A：三相不控整流+Buck（最简单）
```
三相AC → [三相整流桥] → [Buck DC-DC] → 36V输出
```
- 优点：简单，容易实现
- 缺点：功率因数低(约0.6)，发挥部分做不了
- 适合：只做基本要求

### 方案B：三相Vienna整流器（推荐）
```
三相AC → [Vienna整流器(3个开关管)] → 36V输出
```
- 优点：功率因数高(>0.99)，效率高，开关管少
- 缺点：控制复杂
- 适合：冲刺发挥部分

### 方案C：三相PWM整流器
```
三相AC → [三相全桥PWM整流(6个开关管)] → 36V输出
```
- 优点：功率因数可调，功能最强
- 缺点：开关管多，控制最复杂
- 适合：有丰富经验的队伍
