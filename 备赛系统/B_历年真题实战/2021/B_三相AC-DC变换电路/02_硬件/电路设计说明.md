# 2021 B 三相 AC-DC 变换电路 —— 电路设计说明

## 系统总图
```
三相 380V/50Hz ──→ [LC 输入滤波] ──→ [Vienna 整流: 3 开关 + 6 二极管]
                                              ↓ ±350V 分裂母线
                                      [均压电容 470μF×2]
                                              ↓
                                      [同步 Buck 100kHz]
                                              ↓
                                      36V/2A 输出
```

## 模块 1：Vienna 整流主电路

```
La─────M1上 ─┬─ Da+ ──→ +350V
             │
             M1中（中点钳位双向开关）
             │
La回流 ──── Da- ──→ -350V

（Mb/Mc 同结构）
```

| 元件 | 型号 | 数量 |
|---|---|---|
| 中点开关 MOSFET | IPW60R070C7 | 3 |
| 上下二极管 | STTH8R06D | 6 |
| 网侧电感 | 1mH/8A 磁粉芯 | 3 |
| 母线电容 | 470μF/450V | 2（串联）|
| 均压电阻 | 100kΩ/10W | 2 |

## 模块 2：Buck 后级

| 元件 | 型号 |
|---|---|
| 主开关 | IRF540N |
| 同步开关 | IRF540N（替代肖特基）|
| 电感 | 100μH/10A 磁环 |
| 输出电容 | 470μF/63V × 3 |

## 模块 3：采样与控制

- 三相相电压：差分放大 + 1/100 衰减
- 三相相电流：CT 互感器 1000:1 + 100Ω 取样 + OPA2277 调理
- 母线 ±350V：高阻分压 1MΩ:10kΩ + 隔离
- 输出 36V/IL：OPA2277 + ACS712-5A
- MCU：STM32G474RET6
- ADC：4 通道双 ADC 同步 + DMA

## 难点与对策

| 难点 | 对策 |
|---|---|
| 母线均压 | 软件叠加 (V+ - V-)/(V+ + V-) 修正项到 dq 输出 |
| 软启动 | 限流电阻 + 继电器，预充到 0.92·V_bus |
| EMI | 输入加 X2 共模电感 + Y 电容 |
| PF 调整 | 改 iq_ref 即可改 PF（设 iq_ref = id_tan(arccos(PF))） |
