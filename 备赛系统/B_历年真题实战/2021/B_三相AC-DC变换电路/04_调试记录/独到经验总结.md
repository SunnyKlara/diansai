# 2021 B 三相 AC-DC 变换电路 —— 独到经验总结

## 1. Vienna 上电序列

绝对不要直接 380V 上电！正确序列：
1. 三相 120V 慢速上电（调压器）→ 测各路 PWM 波形
2. 测母线 ±100V 是否对称（不对称 → 均压电阻烧）
3. 升压到 220V → 看母线 ±200V，测 PF（应 > 0.95）
4. 升到 380V 满载

跳过任何一步 → MOSFET 烧。

## 2. PLL 锁相必须用 SRF-PLL（DQ 框架）

简单过零检测 PLL 在频率扰动 ±0.5Hz 时收敛 > 1s，期间电流环失稳。  
SRF-PLL（基于 vq → 0 闭环）收敛 < 50ms，鲁棒得多。

## 3. 母线均压的软件补偿

电容容差 ±20%，硬件均压电阻无法做完美。软件方法：
```
v_balance = (V+ - V-) / (V+ + V-)
md_corrected = md - 0.5 · v_balance
mq 不变
```
这个修正只占 dq 输出的 < 5%，但能把 V+/V- 不对称度从 5% 降到 < 0.5%。

## 4. Buck 同步整流回收效率的玄机

不用 SR 时，下管二极管压降 0.6V × 2A = 1.2W 损耗。  
用 SR（同步 MOSFET）时：Rds × 2A² = 8mΩ × 4 = 0.032W。**回收 1.17W，效率 +1.6%**。

代价：必须保证 SR 关断时刻 < 上管开通时刻（防直通），死区 200ns。

## 5. PF = 0.99 vs PF = 1.00 的微妙

PF 1.00 = id 全为有功，但**整流瞬态会让 iq 短暂非零**，PA1900 测仪取平均后只有 0.998。  
PF 0.99 已是常人的"边界"，**不要为差 0.01 调到崩溃**。
