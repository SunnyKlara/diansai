# 2021 J 周期信号波形识别及参数测量装置 —— 电路设计

## 系统总图
```
[输入] ─┬─[÷4 衰减 R 网络]──┐
         │                   │
         └─[直通]            ├──[CD4053 3 选 1 模拟开关]──[OPA2277 缓冲]──[ADC]
                             │
        ┌─[×10 放大 OPA2277]─┘
        │
        └────────────────────┘

       [STM32F407 + DMA + FFT(1024)]
              ↓
        [OLED 0.96"]：显示波形类型 + f + Vpp + 占空比
```

## 量程档位（CD4053 控制）
| 档 | 输入范围 | 通道 |
|---|---|---|
| HIGH | 2V~10V | ÷4 |
| MID  | 200mV~2V | 直通 |
| LOW  | 50mV~200mV | ×10 |

## 自动量程切换
```
ADC 峰值 > 80% 满量程 → 切到 HIGH
ADC 峰值 < 10% 满量程 → 切到 LOW
否则 MID
```
