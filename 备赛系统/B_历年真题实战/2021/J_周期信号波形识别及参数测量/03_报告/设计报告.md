# 2021 J 周期信号波形识别及参数测量装置 —— 设计报告

## 摘要
基于 STM32F407 + CD4053 自动量程切换 + 1024 点 FFT 实现波形识别 + 参数测量。波形识别用 H3/H1 谐波比阈值法（正弦 < 0.05, 三角 0.05~0.2, 矩形 > 0.2）。频率用过零+线性插值（精度 < 0.5%）。Vpp 多周期取最大最小（精度 < 1%）。占空比用中值阈值统计（精度 < 1%）。覆盖 50mV~10V Vpp 与 1Hz~50kHz 频率范围。

## 二、理论分析

### 谐波比阈值
正弦：H3=0；三角：H3/H1=1/9=0.111；矩形：H3/H1=1/3=0.333。

### 频率精度
过零线性插值，每周期插 1 点 → 精度 = fs/(N·f) × 100%。fs=100kHz, N=10 周期, f=1kHz → 误差 0.01%。

### Vpp 误差
ADC 12bit, 50mV 信号 ×10 放大到 500mV → 620 LSB → 1 LSB = 0.16% → 满足 1% 余量充足。

### 量程切换
3 档自动切换，CD4053 + OPA2277 增益 0.25 / 1 / 10。
