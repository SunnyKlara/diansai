## 这道题真正在考什么？

### 表面上考的
- 波形识别（正弦/三角/矩形）
- 参数测量（频率/峰峰值/占空比）

### 实际上考的
**你对信号的理解有多深。**

一个只会调库函数的人，可以用FFT算出频谱，但不知道为什么正弦波的频谱只有一根线而矩形波有很多根线。

一个真正理解信号的人，知道：
- 正弦波是"纯净"的——只有基波，没有谐波
- 矩形波是"粗暴"的——含有大量奇次谐波(3,5,7,9...)，幅度按1/n衰减
- 三角波介于两者之间——也含奇次谐波，但幅度按1/n²衰减（比矩形波衰减快得多）

**这个区别就是识别的关键。** 不需要什么机器学习，不需要什么神经网络，就是看谐波的衰减速度。

## 波形识别的核心原理（不是套路，是物理）

### 三种波形的傅里叶级数

**正弦波**：
```
x(t) = A × sin(2πft)
频谱：只有基波，没有谐波
谐波比：H3/H1 = 0, H5/H1 = 0
```

**矩形波**（50%占空比）：
```
x(t) = (4A/π) × [sin(ωt) + sin(3ωt)/3 + sin(5ωt)/5 + ...]
频谱：只有奇次谐波，幅度按1/n衰减
谐波比：H3/H1 = 1/3 = 0.333, H5/H1 = 1/5 = 0.200
```

**三角波**：
```
x(t) = (8A/π²) × [sin(ωt) - sin(3ωt)/9 + sin(5ωt)/25 - ...]
频谱：只有奇次谐波，幅度按1/n²衰减
谐波比：H3/H1 = 1/9 = 0.111, H5/H1 = 1/25 = 0.040
```

### 识别判据

看3次谐波与基波的比值 R3 = H3/H1：

| 波形 | R3理论值 | 实际范围（考虑噪声和失真） |
|------|----------|--------------------------|
| 正弦波 | 0 | < 0.05 |
| 三角波 | 0.111 | 0.05 ~ 0.20 |
| 矩形波 | 0.333 | > 0.20 |

**就这么简单。** 一个比值就能区分三种波形。

但如果只用一个比值，鲁棒性不够。加上5次谐波比值 R5 = H5/H1 做双重验证：

| 波形 | R3 | R5 | R5/R3 |
|------|-----|-----|-------|
| 正弦波 | ≈0 | ≈0 | 不定义 |
| 三角波 | 0.111 | 0.040 | 0.36 |
| 矩形波 | 0.333 | 0.200 | 0.60 |

R5/R3的比值也能区分三角波和矩形波：三角波约0.36，矩形波约0.60。

### 为什么不用时域特征？

很多人想用时域方法：
- "矩形波有平顶" → 检测平坦段
- "三角波有尖角" → 检测斜率突变
- "正弦波最圆滑" → 检测曲率

**问题**：这些方法对噪声极其敏感。
信号发生器输出的矩形波不是理想矩形——有上升沿、过冲、振铃。
三角波的尖角也不是理想尖角——有圆角。
时域方法很容易被这些非理想因素误导。

频域方法（看谐波比）对这些非理想因素天然免疫，因为上升沿和过冲只影响高次谐波（>10次），不影响3次和5次谐波的比值。

## 参数测量的核心原理

### 频率测量

**方案A：等精度测频法（推荐）**

原理：用被测信号作为计数器的门控信号，同时用高频标准时钟计数。
```
被测信号周期内，标准时钟计数N次
频率 = 标准时钟频率 × 被测信号周期数 / N
```

优点：在整个频率范围内精度一致（"等精度"的含义）
缺点：需要额外的硬件（比较器+定时器捕获）

**方案B：FFT测频（简单但精度受限）**

原理：FFT后找最大峰值对应的bin，频率 = bin × fs/N
精度 = fs/N（频率分辨率）

例：fs=100kHz, N=1024 → 精度=97.6Hz
对于1kHz信号：误差=97.6/1000=9.76% → **不达标！**（要求≤1%）

**改进**：抛物线插值
在峰值bin附近3个点做抛物线拟合，精度提升约10倍。
97.6Hz/10 ≈ 10Hz → 对1kHz信号误差1% → 刚好达标。

**方案C：过零检测（最简单）**

和我之前在2021A中写的一样，加线性插值可以达到0.1%精度。

**我的选择**：方案C（过零检测+插值）用于基本要求，方案A（等精度测频）用于发挥部分。
理由：过零检测纯软件实现，不需要额外硬件；等精度测频精度更高但需要比较器。

### 峰峰值测量

**最简单的方法**：ADC采样一段数据，取最大值和最小值。
Vpp = (max - min) × 电压换算系数

**问题**：噪声会导致max偏大、min偏小 → Vpp偏大。

**改进**：
1. 多个周期取平均：采集10个周期，每个周期分别取max和min，然后对max取平均、min取平均
2. 排序后去掉极端值：对所有采样点排序，取第99百分位和第1百分位

**我的选择**：方法1（多周期平均），简单有效。

### 占空比测量

**方法**：统计一个周期内信号高于中间值的采样点数占总采样点数的比例。

```
中间值 = (max + min) / 2
高电平点数 = 统计 data[i] > 中间值 的个数
占空比 = 高电平点数 / 总点数 × 100%
```

**注意**：这个方法对矩形波很准确，但对正弦波和三角波也会给出一个"占空比"（约50%）。
题目只要求测矩形波的占空比，所以先识别波形类型，只有矩形波才显示占空比。

## 发挥部分的真正挑战

### 50mV信号（信噪比问题）

50mVpp信号，ADC 12bit(0~3.3V)的分辨率=3.3V/4096=0.8mV/LSB。
50mVpp对应约62个LSB → 有效位数约6bit → SNR约36dB。

**这个SNR够不够？**
- 频率测量：过零检测需要信号穿过零点，62LSB的信号在零点附近只有几个LSB的变化，噪声可能导致虚假过零 → **需要放大**
- 波形识别：FFT需要足够的信噪比才能看到谐波，36dB意味着谐波必须>-36dB才能检测到 → 矩形波的3次谐波是-9.5dB，没问题；但5次谐波是-14dB，也没问题 → **不需要放大也能识别**
- 峰峰值测量：62LSB的分辨率，1%误差=0.62LSB → **不够！需要放大**

**结论**：50mV信号必须放大。放大10倍到500mV，对应约620LSB，精度足够。

### 10V信号（超出ADC范围）

ADC输入范围0~3.3V，10Vpp信号会烧ADC！
必须衰减。衰减到1/4 → 2.5Vpp → 在ADC范围内。

### 1Hz信号（采样时间问题）

1Hz信号的一个周期=1秒。
要做FFT至少需要1个完整周期的数据。
如果FFT点数=1024，采样率=1024Hz → 采集时间=1秒。

**问题**：题目要求"反应时间<3秒"。
1秒采集 + FFT计算(<0.1秒) + 显示(<0.1秒) = 约1.2秒 → 达标。

但如果信号频率未知，需要先粗测频率再同步采样：
粗测采样1秒 + 同步采样1秒 = 2秒 → 还是达标，但余量不大。

### 50kHz信号（采样率问题）

50kHz信号，奈奎斯特要求采样率≥100kHz。
如果要看到5次谐波(250kHz)，采样率需要≥500kHz。

大多数MCU的ADC可以达到1MSPS，所以不是问题。
但如果用的是低端MCU（如STM32F103，ADC最高1MSPS但实际约800kSPS），可能需要注意。

## 量程设计（这是硬件的核心）

### 三档量程方案

| 量程 | 输入范围 | 处理 | ADC输入范围 |
|------|----------|------|------------|
| 高量程 | 2V~10V | ÷4衰减 | 0.5V~2.5V |
| 中量程 | 200mV~2V | 直通 | 200mV~2V |
| 低量程 | 50mV~200mV | ×10放大 | 0.5V~2V |

### 自动量程切换逻辑

```
1. 先用中量程采样
2. 如果ADC峰值 > 80%满量程 → 切换到高量程
3. 如果ADC峰值 < 10%满量程 → 切换到低量程
4. 否则保持中量程
5. 切换后重新采样
```

### 量程切换硬件

用CD4053模拟开关（3通道2选1），MCU用2个GPIO控制。

```
输入 → [÷4衰减] ──→ CD4053通道A ──→ ADC
     → [直通]   ──→ CD4053通道B ──→
     → [×10放大] ──→ CD4053通道C ──→
     
MCU GPIO控制CD4053选择哪个通道接到ADC
```

## 评分策略（评委视角）

### 保底目标：基本要求50分 + 报告15分 = 65分

基本要求的21+12+12+5=50分，每一分都要拿到。

**最容易丢分的地方**：
- 波形识别错误（21分全丢）→ 必须用频域方法，不要用时域
- 频率误差>1%（12分全丢）→ 过零检测+插值，不要用简单的FFT峰值
- Vpp误差>1%（12分全丢）→ 多周期平均，不要只取一次max-min

### 冲刺目标：基本50 + 发挥30 + 报告18 = 98分

发挥部分(1)的30分是性价比最高的：
- 扩展Vpp范围到50mV~10V → 加量程切换电路
- 扩展频率范围到1Hz~50kHz → 调整采样率

发挥部分(2)的9分也容易拿：
- 反应时间<3秒 → 只要算法不太慢就行

发挥部分(3)的6分是锦上添花：
- 增加锯齿波、脉冲波、梯形波识别
- 增加上升时间、峰值因数、波形因子测量
