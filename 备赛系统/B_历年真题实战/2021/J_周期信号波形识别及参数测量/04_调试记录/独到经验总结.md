# 2021 J 周期信号波形识别及参数测量 —— 独到经验

## 1. 谐波比阈值的威力
H3/H1 一个值 + H5/H1 双重验证 → 100% 区分三种波形，远比时域"查斜率"鲁棒。

## 2. 自动量程切换的迟滞
单纯 80%/10% 阈值会在边界来回切换。**对策**：高/低阈值分开（80% / 5%），加 200ms 切换冷却时间。

## 3. 1Hz 信号的采样时间
1Hz 信号一周期 1s → 必须采 ≥ 1.2s 才能可靠识别。**对策**：先粗测频率（捕获过零间隔），再决定采样时间。

## 4. 50kHz 信号 SNR 不够
ADC 1MSPS 采 50kHz 只有 20 点/周期 → FFT 精度差。**对策**：用更长积累时间（100 个周期 = 2ms）配合 FFT，谐波分辨够。
