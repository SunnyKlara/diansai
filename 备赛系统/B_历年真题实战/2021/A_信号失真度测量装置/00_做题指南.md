## 核心算法流程

```
信号输入 → 信号调理(放大/衰减) → ADC采样 → FFT → 提取谐波 → 计算THD → 显示
```

## 关键技术挑战

### 1. 30mV小信号（发挥部分）
- 需要前置放大器，增益约100倍
- 低噪声运放：OPA2277或AD8065
- 放大后30mV×100=3V，在ADC范围内

### 2. 100kHz高频（发挥部分）
- 5次谐波=500kHz
- 采样率需≥1MSPS
- TI MCU片内ADC：C2000系列可达3.5MSPS

### 3. 同步采样
- 必须采集整数个周期，否则频谱泄漏
- 方法：先粗测频率→调整采样率→精确采样

### 4. THD计算精度
```c
// THD = √(U2²+U3²+U4²+U5²) / U1 × 100%
// 题目只要求到5次谐波

float calc_thd(float* fft_mag, uint32_t fund_idx) {
    float u1 = fft_mag[fund_idx];
    float sum_sq = 0;
    for (int h = 2; h <= 5; h++) {
        float uh = fft_mag[fund_idx * h];
        sum_sq += uh * uh;
    }
    return sqrtf(sum_sq) / u1 * 100.0f;
}
```

## 信号调理电路

```
输入 → [衰减/放大切换] → [抗混叠滤波] → ADC
        CD4051选择          RC低通
        
量程1: 直通 (300mV~600mV)
量程2: ×10放大 (30mV~60mV → 300mV~600mV)
量程3: ÷10衰减 (3V~6V → 300mV~600mV)  [预留]
```
