# 2021A 信号失真度测量装置 —— 代码工程说明

## 工程概述

一键启动后自动测量输入信号的总谐波失真(THD)，显示THD值、基波频率、各次谐波归一化幅值和一个周期的波形，同时通过蓝牙发送到手机。

## 文件结构

```
01_代码/
├── config.h              全局配置（每个参数都有计算依据）
├── adc_capture.c/.h      ADC采样（Timer_A触发+DMA，采样率可调）
├── freq_detect.c/.h      频率检测（插值过零检测+同步采样率计算）
├── thd_measure.c         THD计算核心（同步采样+平顶窗+FFT+谐波提取）
├── main.c                系统集成（一键测量状态机+多次平均+显示+蓝牙）
└── README.md             本文件
```

## 技术亮点（评委和工程师会关注的）

1. **同步采样消除频谱泄漏**：先粗测频率，再调整采样率使FFT点数恰好覆盖整数个周期
2. **插值过零检测**：线性插值将频率测量精度从1%提升到0.1%
3. **平顶窗双保险**：即使同步不完美，平顶窗也能保证幅度精度<0.01dB
4. **自动量程切换**：30mV~600mV全范围自动适应，无需人工干预
5. **多次平均降噪**：5次测量取平均，随机误差降低√5≈2.2倍
6. **误差预算闭环**：每个误差源都量化了，总误差<1.1%（见深度审题文档）

## 开发环境

- **IDE**: Code Composer Studio (CCS) 12.x
- **SDK**: MSP432 SimpleLink SDK 4.x
- **DSP库**: CMSIS-DSP（需要手动添加到工程）

## 编译步骤

1. 在CCS中创建MSP432P401R工程
2. 导入MSP432 DriverLib
3. 添加CMSIS-DSP库（arm_cortexM4lf_math.lib）
4. 将本目录下所有.c/.h文件添加到工程
5. 编译下载到MSP432 LaunchPad

## 一键测量的时序

```
按键 → 粗采样(10ms) → 频率检测(<1ms) → 量程切换(1ms)
     → 同步采样率计算(<1ms)
     → [采样(10~50ms) → FFT(10ms) → THD(<1ms)] × 5次平均
     → OLED显示(100ms) → 蓝牙发送(200ms) → 蜂鸣器(200ms)
     
总时间：约600ms（远小于10秒限制）
```
