## 信号调理电路设计

```
输入(BNC/SMA) → [保护电路(钳位二极管)]
    → [CD4051量程切换]
        量程1: 直通 (300mV~600mV)
        量程2: ×10放大 (30mV~60mV)
    → [抗混叠滤波(RC低通, fc=500kHz)]
    → [偏置(+1.65V)]
    → ADC输入(0~3.3V)
```
