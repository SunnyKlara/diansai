# 2021 A 信号失真度测量装置 — 电路设计说明

> 配套阅读：`02_硬件/材料清单_BOM.md` 与 `00_深度审题与方案论证.md`。  
> 信号链路设计的核心目标是**支撑 1.1% 总误差预算**：调理误差 < 0.5%，抗混叠 < 0.5%。

---

## 二、保护钳位

输入端可能接收到 5V 以上的信号，必须保护 ADC 输入。

```
BNC ──┬── 1kΩ 限流 ──┬── 后级
       │              │
     [TVS 5V6]      [BAT54S 双肖特基]
       │              │
      GND            VDD/GND（钳位到 0~3.3V）
```

## 三、量程切换（CD4051）

```
                   CD4051 8 选 1 模拟开关
信号 (后限流) ───▶ X (公共端，进 ADC)
                   ┌────────────┐
              X0 ──│ 直通       │  ← 高量程 (300~600mV 直接进 ADC)
              X1 ──│ ×10 放大   │  ← 低量程 (30~60mV → 放大到 300~600mV)
              X2 ──│ ÷10 衰减   │  ← (备用，超过 600mV 时使用)
                   └────────────┘
                          ↑
                  P2.0 (软件控制)
```

CD4051 静态电流 < 1μA，无功耗担忧。

## 四、×10 放大器（×10 通道）

```
                       Rf = 90.9kΩ (0.1%)
                ┌──────[========]──────┐
                │                      │
信号 ──[100nF]──┤+\                    │
            (隔直)\                    │
                  >─────┬─────────── ADC A0
                  /     │
              ──┤-/     │
                │       │
                Rg = 10.1kΩ (0.1%)
                │       │
               GND     GND
```

增益 = 1 + Rf/Rg = 1 + 90.9k/10.1k ≈ **9.0**（实际不是精确 10，是为了标准电阻库存）

或用 **OPA2277 同相放大**：
- Rf = 9 × Rg → 增益 10
- 用电位器微调到精确 10.0

## 五、抗混叠低通滤波器

ADC14 采样率 1MSPS（发挥部分），奈奎斯特 500kHz。  
基本要求 1kHz 信号的 5 次谐波 = 5kHz，发挥部分 100kHz 信号的 5 次谐波 = 500kHz。

**截止频率必须 ≥ 500kHz**（保留所有信号谐波），同时**抑制 500kHz 以上**（防混叠）。

```
                 R = 510Ω
信号 ─────[========]──────┬──── ADC
                          │
                       [C = 620pF]
                          │
                         GND

fc = 1 / (2π × 510 × 620e-12) ≈ 503 kHz
```

**单极 RC 在奈奎斯特处衰减只有 -3dB，对发挥部分不够**。  
升级方案：**二阶 Sallen-Key 低通**

```
        R1=510Ω      R2=510Ω
信号 ──[=====]──┬──[=====]──┬──── OPA2277 +输入
                │            │
              C1=620pF    C2=1.24nF
                │            │
              连缓冲器输出  GND
```

二阶滚降在 500kHz 处衰减 -6dB，1MHz 处 -18dB，**100kHz 5 次谐波纯 + 抗混叠安全**。

## 六、ADC 偏置电路

ADC14 输入范围 0~3.3V（默认 AVCC 参考），但信号是交流的（有正有负）。  
需加 1.65V 直流偏置使信号在 0~3.3V 范围内摆动。

```
REF3030 (3.0V) ─┬── 1kΩ 0.1% ─┬─── 1.65V (偏置)
                │              │
              [LDO 内置]   100kΩ 0.1%
                              │
                             GND
```

**用 REF3030 而不是 VCC 分压**：偏置稳定性 ±0.05%，相比电阻分压（受 VCC 波动影响）精度高 100×。

## 七、OLED I2C 接口

```
P1.6 (UCB0SDA) ──┬── 4.7kΩ 上拉 ── VDD
P1.7 (UCB0SCL) ──┴── 4.7kΩ 上拉 ── VDD
                 │
            SSD1306 OLED
            (I2C 地址 0x3C)
```

## 八、HC-05 蓝牙 UART

```
P3.3 (UCA2TXD) ──── HC-05 RX  (3.3V 电平兼容)
P3.2 (UCA2RXD) ──── HC-05 TX  (注意 HC-05 输出 3.3V，不要接 5V VCC)
                    HC-05 VCC ── 3.3V
                    HC-05 GND ── GND
```

## 九、PCB 布局要点

1. **ADC 输入线最短**：从 CD4051 输出到 P5.5 走线 ≤ 2cm
2. **数字 / 模拟分区**：MCU 数字部分 + I2C/UART 走板子右侧；信号调理 + ADC 输入走左侧
3. **接地**：信号地与数字地在 MCU GND 引脚处单点连接
4. **去耦**：每个 IC 的 VCC 引脚附近 100nF 陶瓷电容
5. **REF3030 远离热源**：基准电压源对温度敏感

| TP1 | BNC 输入 | 信号源 |

## 十一、常见问题与对策

| 现象 | 可能原因 | 对策 |
|---|---|---|
| ADC 读数偏离中值 | 偏置电路偏移 | 校准 ADC 零点（软件减偏移）|
| 低量程下噪声大 | ×10 放大器噪声放大 | 选 OPA2277（噪声 8nV/√Hz）|
| 高频信号幅度衰减 | 抗混叠 LPF 带宽不够 | 检查 fc 是否 ≥ 500kHz |
| FFT 测量基波偏低 | 频谱泄漏 | 切到同步采样模式 |
| THD 系统性偏小 | 谐波被抗混叠滤掉了 | 检查 5 次谐波是否在通带内 |
