## 这道题的物理本质

### 为什么磁悬浮是不稳定的？

Earnshaw定理：静态磁场不能实现稳定悬浮。
无论怎么摆放永磁体，总有一个方向是不稳定的。

**但题目要求底座台不用永磁体，全部用电磁铁！**
电磁铁的电流可以实时调节 → 可以用反馈控制实现稳定。

### 控制原理

```
设定高度h_ref → [+] → [PID控制器] → [功率放大] → [电磁铁]
                  ↑ -                                  │
                  │                              [悬浮盘]
                  │                                  │
                  └──────── [霍尔传感器] ←────────────┘
                              测量实际高度h
```

**关键**：电磁力与距离的关系是**非线性**的！
F ∝ I² / d²（I=电流，d=距离）
距离越近力越大 → 正反馈 → 不稳定
必须用足够快的控制器来抵消这个不稳定性。

### 控制器带宽要求

磁悬浮系统的不稳定极点频率约为：
ω_unstable = √(g/d) ≈ √(9.8/0.02) ≈ 22 rad/s ≈ 3.5Hz

控制器带宽必须远大于3.5Hz才能稳定。
**建议控制频率≥1kHz（是不稳定频率的300倍）。**

## 方案选择

### 悬浮方式

**方案A：吸引式（从上方吸）**
```
[电磁铁] ← 底座台（在上方）
    ↓ 吸力
[悬浮盘] ← 在下方悬浮
    ↓ 重力
```
- 电磁铁在上方，吸引下方的悬浮盘
- 悬浮盘含导磁材料（铁氧体）
- 控制：距离近→减小电流，距离远→增大电流
- **最常见的方案，推荐**

**方案B：排斥式（从下方推）**
```
    ↑ 排斥力
[悬浮盘] ← 在上方悬浮
    ↓ 重力
[电磁铁] ← 底座台（在下方）
```
- 需要交变磁场在导体中感应涡流产生排斥力
- 控制更复杂
- 不推荐

### 电磁铁布局

**单电磁铁（最简单）**
- 一个大电磁铁在中心
- 只能控制垂直方向
- 水平方向不稳定 → 需要机械导向

**多电磁铁阵列（推荐）**
```
俯视图：
  [E1]  [E2]
     [E0]      E0: 中心，主悬浮力
  [E3]  [E4]   E1~E4: 四角，水平稳定
```
- 中心电磁铁提供主要悬浮力
- 四角电磁铁提供水平稳定力
- 每个电磁铁独立控制
- 可以实现倾斜（发挥部分5）
