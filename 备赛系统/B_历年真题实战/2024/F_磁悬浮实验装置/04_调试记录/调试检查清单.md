# 2024 F 磁悬浮 —— 调试检查清单

## 上电前

- [ ] 电源电压确认 30V，电流上限设到 3A（防失控烧管）
- [ ] 5 路 IR2104 / IRF3205 焊接无短路
- [ ] 续流二极管 UF4007 极性正确（电磁铁两端反并）
- [ ] 5 路霍尔 SS49E 安装在电磁铁中心孔，VCC=5V 输出 ~ 2.5V

## 静态校准

- [ ] 上电不放悬浮盘 → 5 路 ADC 读数 ≈ 2.5V ± 0.05V
- [ ] 校准查表数据 `cal_voltage[]` / `cal_distance[]` 实测覆盖
- [ ] PWM 输出占空比 0% → 电磁铁不通电
- [ ] PWM 占空比 50% → 实测电流 ~ 满载 50%

## PD 整定

- [ ] **从 Kp = 5, Kd = 0.5, Ki = 0 开始**（不要直接上 15 / 0.8）
- [ ] 慢慢加悬浮盘到电磁铁下方 → 应能维持悬浮 ≥ 1s
- [ ] Kp 太小：盘掉下；Kp 太大：剧烈振荡
- [ ] 调到 Kd = 0.8 抑制振荡
- [ ] 最后慢慢加 Ki = 0.1 消除稳态误差

## 性能验收

- [ ] 基本(1) 悬浮 > 2s, h > 1cm ✅
- [ ] 基本(2) ≥ 10s, h ≥ 2cm ✅
- [ ] 基本(3) 加 20g 仍稳 ≥ 10s
- [ ] 发挥(4) 高度数字可调 ≥ 2cm（按键设定 + ramp 平滑）
- [ ] 发挥(5) 整体可倾斜（四周电磁铁补偿）

## 应急

- 失稳 / 弹飞 → 立即关电源
- MOSFET 烫手 → 检查死区（应 200ns）
- 霍尔输出漂 → 启动自校
