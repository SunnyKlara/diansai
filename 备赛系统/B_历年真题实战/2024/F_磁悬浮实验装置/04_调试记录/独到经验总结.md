# 2024 F 磁悬浮实验装置 —— 独到经验总结

## 1. 控制率必须 ≥ 1 kHz

不稳定极点 3.5 Hz，控制带宽必须 30 Hz 以上。  
**控制率 100 Hz 必失败**（带宽不够，悬浮盘永远稳不住）。  
**对策**：2 kHz 起步（300× 不稳定极点）。

## 2. 微分先行而非误差导数

设定值从 20mm 跳到 22mm 时，标准 PID 的 D 项产生 -Kd·(2/dt) 巨大尖峰 → 悬浮盘瞬间被吸上去贴电磁铁。

**对策**：D 项只对测量值求导，与设定值无关。本工程 maglev_pd.c 已实现。

## 3. I 项是双刃剑

I 项过大 → 缓慢漂移到极限 → 突然失稳。  
**对策**：Ki ≤ 0.1，且积分限幅在 30% 输出之内。

## 4. 续流二极管影响响应

普通 1N4007 反向恢复 1.5μs → PWM 20kHz（50μs 周期）下损耗大且电流震荡。  
**对策**：UF4007（75ns）或肖特基（SS54）。

## 5. 电源纹波耦合

20kHz PWM 5 通道并联 → 母线电流 5 个 IRF3205 同步开关 → 电源纹波 1V 起。  
**对策**：母线加 470μF + 0.1μF 高频去耦；MCU 与功率分两路供电。
