# 2024F 磁悬浮 —— 独到经验总结

## 一、PID调参的"生死线"

### 磁悬浮的PID参数空间

```
        Kd
        ↑
        │     ╔═══════╗
        │     ║ 稳定  ║  ← 这个区域非常小！
        │     ║ 悬浮  ║
        │     ╚═══════╝
        │   ╱           ╲
        │  ╱  振荡后掉   ╲  直接掉
        │ ╱               ╲
        └──────────────────→ Kp
```

**稳定区域像一个狭窄的岛屿**，Kp和Kd必须同时在正确范围内。
Kp太小→恢复力不够→掉；Kp太大→振荡→掉。
Kd太小→阻尼不够→振荡→掉；Kd太大→响应太慢→掉。

### 调参步骤（血泪经验）

1. **先不悬浮**，把悬浮盘放在电磁铁正下方，用手托住
2. 给一个固定的PWM占空比（比如50%），感受吸力大小
3. 调整占空比，找到"刚好能吸住但不会吸死"的值 → 这就是平衡点附近的占空比
4. **开启PD控制**，Kp从很小开始（比如1.0），Kd=0
5. 慢慢松手，观察悬浮盘是否能短暂悬浮
6. 如果直接掉 → Kp太小，增大
7. 如果吸上去 → Kp太大，减小
8. 找到"松手后能悬浮1~2秒"的Kp值
9. **加入Kd**，从Kp/10开始
10. Kd增大 → 振荡减小 → 悬浮时间延长
11. 找到"稳定悬浮>10秒"的Kd值
12. **微调**：在稳定区域内微调Kp和Kd，找到最稳定的点

**整个过程可能需要2~4小时！这是这道题最耗时的部分。**

## 二、电磁铁绕制的学问

### 匝数的选择

匝数多 → 磁力大 → 但电感大 → 电流响应慢
匝数少 → 磁力小 → 但电感小 → 电流响应快

**磁悬浮需要快速响应**，所以不能绕太多匝。

经验值：
- 铁芯截面积1cm² → 约200~400匝
- 线径0.5mm → 电阻约5~10Ω
- 电感约10~50mH
- 30V供电 → 稳态电流3~6A → 功率90~180W（很大！需要散热）

### 铁芯材料

- **硅钢片**：磁导率高，但有涡流损耗（适合低频<1kHz）
- **铁氧体**：涡流损耗小，但磁导率较低（适合高频）
- **铁粉芯**：折中

对于2kHz控制频率，硅钢片的涡流损耗可以接受。
如果用更高的PWM频率(>10kHz)，建议用铁氧体。

### 悬浮盘材料

- 必须是导磁材料（能被电磁铁吸引）
- **铁氧体圆盘**：轻、导磁性好、不导电（无涡流）→ 推荐
- 铁片：重、导电（有涡流损耗）→ 不推荐
- 钕铁硼永磁体：题目说悬浮盘无源，但没说不能有永磁体
  → 如果允许，永磁体+电磁铁的组合更容易悬浮

## 三、20g加载的挑战（要求3）

### 为什么加载后容易掉？

加载20g后，重力增加了。
如果控制器的I项（积分）不够，稳态电流不够大，悬浮盘会缓慢下降直到掉落。

### 解决方案

**方案A：加大I项**
- 让积分器慢慢增加电流补偿额外重力
- 风险：I项太大会导致不稳定

**方案B：重力前馈**
- 检测到加载后（高度突然下降），立即增加一个固定的电流偏置
- 偏置量 = 额外重力对应的电流增量
- 不需要等积分器慢慢累积

**方案C：自适应控制**
- 根据稳态电流估算悬浮盘重量
- 自动调整控制参数

**推荐方案B**：简单有效，响应快。

## 四、倾斜测试（要求5）——最难的15分

### 为什么倾斜后会掉？

倾斜后，重力不再垂直向下，而是有一个水平分量。
单个中心电磁铁只能提供垂直方向的力，无法抵消水平分量。
悬浮盘会向低处滑动，最终脱离电磁铁范围。

### 解决方案

**必须用多电磁铁阵列！**

```
倾斜时的力分析：

  [E1]  [E2]     ← 高侧电磁铁减小电流
     [E0]        ← 中心电磁铁保持
  [E3]  [E4]     ← 低侧电磁铁增大电流

低侧增大电流 → 产生更大的吸力 → 抵消重力的水平分量
```

需要4个角的霍尔传感器分别检测悬浮盘在各方向的偏移，
然后独立控制4个角的电磁铁电流。

**这本质上是一个多输入多输出(MIMO)控制问题。**
比单输入单输出(SISO)的中心控制复杂得多。

### 倾斜角度的极限

理论上，倾斜角度受限于水平方向的控制力。
如果角电磁铁的最大力 > 重力的水平分量：
  F_horizontal = m×g×sin(θ)
  F_max_horizontal = 角电磁铁最大力

  θ_max = arcsin(F_max_horizontal / (m×g))

实际中，10~20°已经很不错了。

## 五、功耗优化（要求6）

### 功耗的主要来源

1. 电磁铁铜损：I²R（最大的功耗来源）
2. 电磁铁铁损：涡流+磁滞（与频率有关）
3. 控制电路功耗：MCU+驱动（相对很小）

### 优化方法

1. **减轻悬浮盘重量** → 需要更小的悬浮力 → 更小的电流
2. **优化电磁铁设计** → 更小的气隙 → 同样的力需要更小的电流
3. **用PWM脉冲驱动** → 平均电流小于峰值电流
4. **悬浮盘用永磁体**（如果允许）→ 电磁铁只需要提供微调力

### 功耗估算

悬浮盘重量50g → 需要力F=0.05×9.8=0.49N
电磁铁效率约10%（大部分能量变成热）
电功率 ≈ F×d/η ≈ 0.49×0.02/0.1 ≈ 0.1W（理论最小值）
实际功率通常在5~20W（因为电磁铁效率低）
