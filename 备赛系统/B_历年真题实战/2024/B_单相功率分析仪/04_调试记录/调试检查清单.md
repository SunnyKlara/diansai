# 2024B 调试检查清单

## Day1 焊接

### 安全检查（最重要！）
- [ ] 电流互感器穿心方向正确
- [ ] 电压互感器接线正确
- [ ] 强电部分绝缘处理完成
- [ ] 用电插座接线安全可靠

### 采样电路
- [ ] 电流互感器+采样电阻输出电压在0~3.3V
- [ ] 电压互感器输出电压在0~3.3V
- [ ] 偏置电路(1.65V)正确
- [ ] OPA2333运放供电正常

### 显示
- [ ] 段码LCD HT1621驱动正常
- [ ] 能显示数字

### 功耗
- [ ] 万用表测总电流 < 15mA (50mW/3.3V)

## Day2 软件调试

### ADC采样
- [ ] MSP430 ADC12配置正确(双通道同步)
- [ ] Timer触发ADC，采样率10kHz
- [ ] 采样数据合理（用串口打印检查）
- [ ] 电压采样值与万用表对比校准
- [ ] 电流采样值与万用表对比校准

### 基本测量
- [ ] Vrms计算正确（与PA对比误差≤1%）
- [ ] Irms计算正确（与PA对比误差≤1%）
- [ ] P计算正确（与PA对比误差≤1%）
- [ ] PF计算正确（与PA对比误差≤1%）

### 谐波分析
- [ ] DFT计算各次谐波幅值
- [ ] H1~H10显示值合理
- [ ] THD计算正确
- [ ] 与PA对比误差≤2%

## Day3 完善

- [ ] 多种负载测试（纯阻/阻容/阻感）
- [ ] 功耗测量确认≤50mW
- [ ] LCD显示格式美观
- [ ] 写报告

## Day4 最终
- [ ] 所有测量项误差达标
- [ ] 功耗达标
- [ ] 报告打印
- [ ] 功耗测量端口预留（电池正极串联）

## 常见故障

| 问题 | 原因 | 解决 |
|------|------|------|
| 电压读数偏差大 | 互感器变比不准 | 用已知电压校准 |
| 电流读数偏差大 | 穿心匝数搞错 | 确认匝数并校准 |
| 功率因数不对 | 电压电流采样不同步 | 确保双通道同时触发 |
| THD偏差大 | 采样点数不是整周期 | 调整采样率或点数 |
| 功耗超50mW | 显示屏功耗太大 | 换段码LCD |
| MSP430死机 | 浮点运算溢出 | 检查除零和溢出 |
