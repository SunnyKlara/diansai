## 总体策略（仪表类核心）

```
┌─────────────────────────────────────────────────────────────────┐
│  策略 1：板分两区 —— 强电隔离区（左 1/3）+ 信号区（右 2/3）   │
│  策略 2：ZMPT/ZMCT 之间留空 1cm 物理槽，禁止飞线穿越           │
│  策略 3：MCU 安置板右下角，LCD 居中（赛场视觉效果）             │
│  策略 4：屏蔽 —— ADC 走线 < 5cm，必走在 MCU 正下方              │
│  策略 5：测试点（TP）暴露 6 个：V_ZMPT / I_ZMCT / 3V3 / GND     │
│         / VREF / DUMP_TX —— 故障定位 30 秒搞定                  │

```

## 强电区（左 1/3，约 2 cm 宽）

```
        ┌───────────┐
        │ AC 接线端子 (220V 火 + 零)│
        ├───────────┤
        │ ZMPT101B  │ ← 副边 0~5V → 信号区 ADC IN1
        │ 模块       │
        ├───────────┤
        │ ZMCT103C  │ ← 穿心电流互感器，原边穿过载电流线
        │ 模块       │
        ├───────────┤
        │ AC 入端子GND│
        └───────────┘
        
关键焊接：
- ZMPT 与 ZMCT 副边输出 → 用屏蔽线（1m 内可不屏蔽）走到信号区
- 强电区与信号区的 GND **不**直接合并；通过 47Ω 1W 电阻 + 100nF 桥接（隔离 ESD/共模）
- 强电区右侧 1cm 不焊任何元件，物理隔离槽
```

## ADC 信号调理（关键）

每路 ADC 调理（共 2 路：V / I）：

```
ZMPT 输出 0~5V (双向)
        │
        │  R1 100kΩ (1%)
        ├────┐
              │
              ●─── ADC IN（0~3.3V，偏置 1.65V）
              │
              │  R2 100kΩ (1%)（偏置）
              │  C1 100nF（去耦）
        ┌────┐
        │
        VREF/2 = 1.65V

布局占位：4 元件，约 1.5 cm × 1 cm
```

## 关键间距

| 项 | 距离 | 原因 |
|---|---|---|
| ZMPT/ZMCT ↔ MCU | ≥ 4 cm | 强电区物理隔离 |
| ADC 调理电路 ↔ MCU | < 2 cm | ADC 走线越短越好 |
| ADC 走线 ↔ I2C 走线 | ≥ 5 mm | I2C 上沿对 ADC 串扰 |
| TP 测试点间距 | ≥ 5 mm | 探针不打架 |

## 接地（仪表类核心）

```
              强电区 GND（ZMPT/ZMCT 副边）
                    │
                    │ 47Ω 1W + 100nF
                    │ （ESD/共模桥接）
                    ▼
              信号区 GND ← 单点接 MCU GND
                    │
                    ▼
              USB 调试时勿同时接 PA 电源（地环路）
```

## 走线规则

| 线种 | 截面积 | 走法 |
|---|---|---|
| 220V 火/零 | **必须 1.0mm² 以上**，绝缘 600V | 走板左 1cm 内，不与信号线交叉 |
| ZMPT/ZMCT 副边 | 0.3mm² 双绞 | 直接到 ADC 调理，<5 cm |
| ADC 模拟线 | 0.2mm² | 走板下层，远离 SPI/I2C |
| LCD I2C | 0.2mm² | 双绞，<15cm |

## 故障定位 × 布局（30 秒）

| 现象 | TP 检测 | 处理 |
|---|---|---|
| Vrms = 0 | TP1 V_ZMPT 测电压：应 1.65V±2.5V 摆动 | 0V → ZMPT 不通 |
| Irms = 0 | TP2 I_ZMCT | 同上 |
| MCU 不工作 | TP3 3V3 | < 3V → LP 板 3V3 输出问题 |
| 数据漂移大 | TP5 VREF | < 1.6V 或 > 1.7V → 偏置电路失调 |
| LCD 闪屏 | TP4 GND ↔ 强电 GND 串扰 | 47Ω 桥接缺失 |

## 安全（强电板必备）

```
☐ 220V 火/零线 **绝对不要** 焊在排针，必须用接线端子（ABB 蓝色）
☐ 万用板 **强电区** 与外壳 ≥ 8mm 间距
☐ 调试 / 上电前先用万用表测 220V 火/零、火/地
☐ 全程戴绝缘手套（赛场要求）
☐ ZMPT 副边短路保护：副边并联一个 510Ω 电阻（防开路烧）
☐ ZMCT 副边 **禁止开路**（互感器原理：副边开路 → 副边电压瞬时几百伏）
```
