# 2024 B 单相功率分析仪 — 电路设计说明

> ⚠️ **涉及 220V 市电，调试期必须先用 24V 变压器模拟，确认隔离可靠后再接市电！**

## 一、系统信号链路

```
220V 市电 (L/N)
   │
   ├──→ [电压互感器 ZMPT101B 1:1 隔离] ──→ [副边 100k/1k 分压] ──→ [OPA2333 缓冲 + 1.65V 偏置] ──→ ADC P1.0 (A0)
   │
   └──→ [电流互感器 ZMCT103C 1000:1] ──→ [100Ω 采样电阻 + N=10 匝穿心] ──→ [OPA2333 缓冲 + 1.65V 偏置] ──→ ADC P1.1 (A1)

MSP430FR5994
   │
   ├──→ HT1621 段码 LCD（3 线 SPI）
   ├──→ 2 个按键（翻页 / 校准）
   └──→ 2 个 LED（运行 / 故障）

供电：3V CR2032（或 4.5V → TPS7A02 LDO → 3.3V）
功耗测量端口：电池正极串联万用表 mA 档
```

## 二、电压采样电路

```
220V (L) ──┬── 1MΩ ──┐
            │          │
            ZMPT101B   │  ZMPT101B 副边
            一次侧     │  内阻 ~ 1kΩ
            │          │
220V (N) ──┘          ▼
                    副边输出（峰值 ~ ±220V × 1/1 = ±220V）
                                            ↓
                                    [100kΩ + 1kΩ 分压]
                                            ↓
                                    实际副边电压 / 100 ≈ ±2.2V
                                            ↓
                                    OPA2333 缓冲 + 加 1.65V 偏置
                                            ↓
                                    0.55V ~ 2.75V → MSP430 ADC A0
```

> **注**：ZMPT101B 副边输出电压取决于副边采样电阻。通常实际方案是 220V 输入时副边电流 2mA，乘以采样电阻得到副边电压。具体阻值需根据 ADC 范围反推。这里给出的是简化示意，实际焊接前用万用表测一遍副边波形幅度并调整分压比。

## 三、电流采样电路

```
220V 火线 ──┬── 负载 ──── N
              │
        ZMCT103C 穿心
        N = 10 匝（提高灵敏度 10×）
        变比 1000:1
              ↓
        副边电流 = 原方电流 × N / 1000 = 原方电流 × 0.01
              ↓
        100Ω 采样电阻
              ↓
        副边电压 = 0.01 × 100 = 原方电流 × 1V/A
              ↓
        OPA2333 缓冲 + 加 1.65V 偏置
              ↓
        ADC A1（动态范围 ±1.65V → 测量 ±1.65A）
```

**穿心匝数选择**：
- 测 1A：穿 10 匝（副边 100mV，噪声尚可）
- 测 5A：穿 1 匝（副边 50mV/匝，要软件校准准）
- 题目典型负载电流 < 5A，选 N = 10 兼顾灵敏度

## 四、抗混叠 + 低通滤波

ADC 采样率 10kHz → 奈奎斯特 5kHz。  
工频谐波只到 10 次 = 500Hz，**5kHz 以上都是噪声**。

| 位置 | 滤波器 | 设计 |
|---|---|---|
| 电压链路 | RC 一阶低通 | R=10k，C=10nF → fc = 1.6kHz（保 10 次谐波，抑制 5kHz+ 干扰）|
| 电流链路 | 同上 | 同上 |
| ADC 输入端 | 100pF 高频去耦 | 抑制 ADC 自身的开关噪声 |

## 五、低功耗设计要点

| 点 | 措施 |
|---|---|
| LDO 选型 | TPS7A02（25nA 静态电流），不要用 AMS1117（5mA！）|
| 上拉电阻 | 用 100kΩ 而不是 10kΩ（节省 10× 静态电流）|
| 浮空 GPIO | **必须**配为输出 + 拉低，**不能**浮空（浮空时 IO 处于翻转态消耗大）|
| 未用外设 | CCS 中关闭对应模块的 PMM 时钟门控 |
| 显示 | HT1621 段码 LCD（0.5mW），**绝对不要**用 OLED |
| 运放 | OPA2333（17μA），**绝对不要**用 OPA2277（800μA）|

## 六、220V 安全规则

⚠️ **保命级规则，全部必须做到**：

1. **强弱电完全隔离**：互感器是隔离屏障，副边到 MCU 不允许任何走线接近一次侧
2. **板子分两块**：强电板（含互感器一次侧 + 220V 接线）+ 信号板（互感器副边 + MCU），物理距离 ≥ 5cm
3. **强电部分加保险丝**：5A 速断保险（通常市电插座有，再加一道双保险）
4. **强电走线**：≥ 1mm² 漆包铜线，焊点要包热缩管
5. **调试顺序**：先 24V 变压器代替市电 → 万用表测互感器副边波形 → 对比期望值 → 接 220V
6. **永远不要在带电时调强电板**：动手前先拔插头 + 等 30 秒（电容放电）+ 万用表测确认 0V
7. **绝缘垫**：调试期电路板下垫绝缘垫（橡胶或厚纸板）

## 七、PCB 布局要点

1. **强电区与信号区物理隔离**：强电走板子一侧，信号走另一侧，中间留 5mm+ 隔离带
2. **互感器副边走线尽量短**：高频谐波采样精度对走线长度敏感
3. **ADC 输入线走在 GND 旁**：减少串扰
4. **电源去耦**：每个 IC 的 VCC-GND 加 100nF 陶瓷电容
5. **TimerA → ADC 触发线** + **DMA 数据线** 都做地线包裹

## 八、测试点预留

```
TP1: ZMPT101B 副边输出（看电压波形）
TP2: ADC A0 输入（看调理后波形）
TP3: ZMCT103C 副边输出
TP4: ADC A1 输入
TP5: ADC 1.65V 偏置基准
TP6: VCC（3.3V，监测电压稳定性）
TP7: 电池正极（功耗测量端）
TP8: 电池负极（万用表 mA 档另一端）
```

每个测试点焊一个排针，方便接示波器和万用表。
