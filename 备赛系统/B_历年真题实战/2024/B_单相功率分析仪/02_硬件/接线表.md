# 2024 B 单相功率分析仪 —— 接线表

> 平台：MSP430FR5994 LaunchPad

## 总览
```
     [220V/24V 隔离变压器]
              ↓
     [被测负载]
              ↓ V                ↓ I
     [ZMPT101B]              [ZMCT103C]
         ↓                       ↓
     [OPA2277 调理 ×2]       [OPA2277 调理 + 取样电阻]
         ↓                       ↓
     P1.0 (A0)               P1.1 (A1)
              ↓
        [MSP430FR5994]
              ↓
        [HT1621 LCD] + [按键 + LED]
```

## 1. 电源域

| # | 源 | 目的 | 颜色 | 长度 | 备注 |
|---|---|---|---|---|---|
| P1 | LP-FR5994 3.3V | OPA2277 V+ ×2 | 红 | 5cm | 共 2 个运放 |
| P2 | LP-FR5994 GND | OPA2277 V- ×2 | 黑 | 5cm | |
| P3 | LP-FR5994 3.3V | LCD VCC | 红 | 8cm | |
| P4 | LP-FR5994 GND | LCD GND | 黑 | 8cm | |
| P5 | LP-FR5994 3.3V | TL431 + R 网络 (1.65V 基准) | 红 | 4cm | ADC 偏置 |

## 2. 电压采样链

| # | 源 | 目的 | 备注 |
|---|---|---|---|
| V1 | 被测负载两端 | ZMPT101B 输入 | 电压互感器 1:1 |
| V2 | ZMPT101B 输出 | 1MΩ : 100kΩ 分压 | 24Vac → ±2.18V |
| V3 | 分压输出 | OPA2277 + 1.65V 偏置 | 0.5~2.8V |
| V4 | OPA2277 输出 | MSP430 P1.0 (A0) | ADC 输入 |

## 3. 电流采样链

| # | 源 | 目的 | 备注 |
|---|---|---|---|
| I1 | 被测负载电流 | ZMCT103C 穿心 | 1000:1 互感器 |
| I2 | ZMCT103C 输出 | 100Ω 1% 取样电阻 | ±0.5V @ 5A |
| I3 | 取样电阻两端 | OPA2277 ×2.5 + 1.65V 偏置 | 差分放大 |
| I4 | OPA2277 输出 | MSP430 P1.1 (A1) | ADC 输入 |

## 4. LCD HT1621B（4 位段式）

| # | MCU 端 | LCD 端 | 颜色 |
|---|---|---|---|
| L1 | P5.0 | DATA | 蓝 |
| L2 | P5.2 | WR | 黄 |
| L3 | P5.4 | CS | 绿 |
| L4 | 3.3V | VCC | 红 |
| L5 | GND | GND | 黑 |

## 5. 按键 + LED

| # | MCU 端 | 元件 | 备注 |
|---|---|---|---|
| K1 | P2.5 | KEY1 | 板载 |
| K2 | P3.4 | KEY2 | 外接 |

## 6. 接线检查表

- [ ] 220V → 24V 隔离变压器（**严禁直接接市电！**）
- [ ] ZMCT103C 取样电阻 100Ω 1% 金属膜
- [ ] OPA2277 单电源（3.3V）模式接线正确
- [ ] 1.65V 基准用 TL431 ± 0.5%（不要分压器）
- [ ] LCD CS 在不传输时拉高
