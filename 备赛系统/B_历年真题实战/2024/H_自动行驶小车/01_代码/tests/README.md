# 2024 H 自动行驶小车 — 算法层 PC 验证

`algo_reference.py` 是 PID + 循迹 + 路径管理的 Python 等价实现，PC 直接跑：

```bash
python algo_reference.py
```

## 用途

1. **PID 参数闭环仿真**：调整 `config.h` 里的 PID_YAW_KP/KI/KD 后，先在 Python 里跑一遍验证收敛性
2. **路径段序列检查**：4 个任务的段数 / 段类型一目了然
3. **MCU 移植对比**：MSPM0 端实现完成后，相同输入下输出应一致

## 关键约束

修改 `config.h` 后，必须同步修改本脚本顶部的常量定义。
否则金标准就失效了。
