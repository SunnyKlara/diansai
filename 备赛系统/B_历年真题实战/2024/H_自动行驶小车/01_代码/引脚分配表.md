## 2. 时钟配置

| 项 | 值 |
|---|---|
| HFCLK 源 | SYSPLL（80 MHz）|
| MCLK | 80 MHz |
| ULPCLK | 40 MHz（外设默认）|
| ADC0 时钟 | ULPCLK / 1 = 40 MHz |
| TIMA0/TIMG0 时钟 | MCLK = 80 MHz |

## 3. PWM 周期推导

```
PWM 频率 = 5 kHz（电机 / TB6612 推荐）
TIMA0 时钟 = 80 MHz
中心对齐 PWM 周期 = ARR × 2
=> ARR = 80MHz / (5kHz × 2) - 1 = 7999
但 PWM 分辨率取 1000（占空比 0..1000）
=> 实际 ARR = 80MHz / (5kHz) - 1 = 15999
=> 占空比 1000 → 全占
=> config.h 中 MOTOR_PWM_MAX = 1000
```

## 6. SysConfig 关键截图清单

> 用 SysConfig GUI 配置时按本表勾选；配置完后 `Generate` 生成 `ti_msp_dl_config.{c,h}`。

1. **System / SYSCTL**：HFXT 32MHz → SYSPLL 80MHz → MCLK 80MHz
2. **TIMA0**：PWM 模式 / Period=15999 / 中心对齐 / CC2/CC3 输出
3. **TIMG0+TIMG6**：QEI 编码器模式 / 4 倍频
4. **ADC0**：Sequential 5 通道（IN5/6/7/10/9）/ 软件触发 / DMA 搬运
5. **GPIO**：上面列出的所有 GPIO 配置成 OUT / IN-PullUp
6. **I2C0+I2C1**：100kHz 标准模式
7. **UART0**：115200 8N1（调试用）

## 7. 引脚冲突与避坑

- **PA10/PA11 是 I2C1 默认脚**，与某些 LED 复用，必须在 SysConfig 里去掉默认 LED 占用
- **PB18 是 STBY**，复位后默认拉低 → TB6612 不输出 PWM。一定要在 `Motor_Init()` 里第一时间拉高
- **MPU6050 INT 引脚**默认开漏，需要 MCU 这端配上拉
- **5V 信号**（HC-SR04 等）若混入需电平转换，本设计未用
