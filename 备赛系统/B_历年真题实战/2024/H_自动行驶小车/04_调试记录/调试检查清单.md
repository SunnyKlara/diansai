## 0. 上电前（不通电检查，5 min）

- [ ] 万用表蜂鸣档：VBAT 与 GND 不短路
- [ ] 5V 输出端：未短路
- [ ] LM2596 电压预先调到 5.0V（用实验电源给 7.4V 输入，万用表测输出）
- [ ] TB6612 安装方向（pin 1 标记朝预定方向）
- [ ] MPU6050 焊盘干净，I2C 上拉电阻在板（GY-521 自带，确认）
- [ ] 电池接口极性（XT60 不会插反，但插件式接口要看）
- [ ] 编码器线 A/B 不接反（接反不影响计数，但会反向）
- [ ] 主控板 USB-C 接 PC 时不要同时接电池（先测试再上电池）

## 1. 主控基础（30 min）

- [ ] 1.1 USB 连接，CCS 识别 LP-MSPM0G3507
- [ ] 1.2 烧录 SDK 例程 `gpio_toggle_output_LP_MSPM0G3507`：板载 LED 闪烁
- [ ] 1.3 串口 BackChannel UART 打开（115200 8N1），程序里 `printf("hello\n");` 能看到输出
- [ ] 1.4 编译我们的工程（即使 TODO 未填，至少能 link 通过 main.c）

## 2. PWM + 电机方向（30 min）

- [ ] 2.1 SysConfig 配置 TIMA0_C2/C3，烧入测试程序：占空比 50%
- [ ] 2.2 万用表测 PA8/PA9 输出：约 1.65V（PWM 平均）
- [ ] 2.3 接 TB6612 PWMA/PWMB 引脚，**先不接电机**，测 AO1/AO2 引脚电压
- [ ] 2.4 配 AIN1=1, AIN2=0，AO1 应 ≈ VBAT × 50%（AO2 ≈ 0）
- [ ] 2.5 切换 AIN1=0, AIN2=1，AO1/AO2 互换
- [ ] 2.6 接一只电机：低占空比（PWM=300）能慢转，方向控制正确
- [ ] 2.7 接两只电机：同 PWM 转速差异 < 10%（差异大则记录哪只更快）

## 3. 编码器（30 min）

- [ ] 3.1 SysConfig 配 TIMG6 Quadrature Mode，A/B 引脚正确
- [ ] 3.2 手转左轮一圈（视觉对齐），串口打印计数器值：应 ≈ 1320 ±5
- [ ] 3.3 反向转一圈：得到 -1320
- [ ] 3.4 同操作右轮（TIMG7）
- [ ] 3.5 调整 `WHEEL_DIAMETER_MM` 直到推 1m 直线计数 ≈ 4900~4950 脉冲

## 4. I2C + MPU6050（45 min）

- [ ] 4.1 SysConfig 配 I2C1，频率 400kHz
- [ ] 4.2 I2C scanner 测试程序：扫描总线，应只看到 0x68（MPU6050）和 0x3C（OLED 如有）
- [ ] 4.3 读 WHO_AM_I（0x75 寄存器）= 0x68
- [ ] 4.4 配置寄存器后读 GYRO_ZOUT（0x47）：静止时值在 ±100 LSB（即 ±6°/s）
- [ ] 4.5 加载 DMP 固件，启动 6 轴融合
- [ ] 4.6 主程序里读 yaw：旋转 90° 物理角度 → yaw 改变 ±90° ±2°
- [ ] 4.7 静止 30 秒：yaw 漂移 < 1°（如果 > 5° 表示 DMP/标定有问题）

## 5. 红外（15 min）

- [ ] 5.1 SysConfig 配 PB5~PB9 GPIO 输入弱上拉
- [ ] 5.2 模块离地 5~8mm，调电位器：白纸下 LED 灭，黑线下 LED 亮
- [ ] 5.3 串口打印 5 路状态：白纸 = 全 0，部分黑线 = 对应位 1
- [ ] 5.4 计算加权位置 `Track_CalcPosition()`：
  - 左压线：≈ -2.0
  - 中央压线：≈ 0.0
  - 右压线：≈ +2.0

## 6. 按键 + LED + 蜂鸣（15 min）

- [ ] 6.1 按键松开时引脚电平 1，按下时 0
- [ ] 6.2 软件防抖：连按 10 次不漏
- [ ] 6.3 三色 LED 各自能点亮
- [ ] 6.4 蜂鸣器响 200ms 不持续

## 7. 速度环单独验证（30 min）

- [ ] 7.1 设 `g_tgt_v_left = 200`，主循环喂电机：电机能稳定转
- [ ] 7.2 用手轻按车轮：增量 PID 应增加 PWM 抵抗
- [ ] 7.3 调 KP/KI：直到响应快、无超调

## 排错速查表

| 现象 | 可能原因 | 排查 |
|---|---|---|
| 编译过烧不进 | XDS110 驱动 / 板子 boot 模式 | 重插 USB / 按 RESET |
| LED 不闪 | SysTick 没启 / 时钟错 | 用 SDK 例程交叉验证 |
| 电机不转 | PWM 没开 / STBY=0 / 接线 | 万用表测 PWM 引脚 |
| 编码器读数=0 | TIMG 模式错 / A/B 引脚错 | 用示波器看脉冲 |
| 编码器只增不减 | A/B 接反 | 交换 A/B 线 |
| 编码器跳变 | 5V 域噪声 | 串 100Ω + 1nF 滤波 |
| MPU6050 无响应 | I2C 上拉 / 地址错 | 用 I2C scanner |
| MPU6050 yaw 漂 | DMP 没启 / 没标定 | 重做 IMU_Calibrate |
| 红外抖 | 阈值 / 环境光 | 调电位器 / 加遮光 |
| 直线偏 | KP_yaw 太小 / IMU 漂 | 加大 KP / 重标定 |
| 弧线甩出 | V_ARC_PWM 太大 | 降到 250 |
| 段切换不动 | 退出条件不满足 | 串口打印每周期判定值 |
