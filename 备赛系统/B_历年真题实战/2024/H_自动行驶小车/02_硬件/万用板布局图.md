# 2024 H 自动行驶小车 — 万用板布局图

> 配套阅读：`材料清单_BOM.md` / `接线表.md` / `电路设计说明.md`  
> 比例尺：1 格 = 2.54mm（标准万用板间距）  
> 板型：5×7 cm 万用板 ×2（电源板 + 信号板分开，是这道题的关键）

---

## 总体策略

```
┌─────────────────────────────────────────────────────────────┐
│  策略 1：电源板 + 信号板分开 → 大电流环路远离 ADC/IMU       │
│  策略 2：模块化插接（排针 + 排母）→ 调试 1 分钟换模块       │
│  策略 3：地线分两块（PGND / SGND），单点共地在电池负极       │
│  策略 4：所有模块用 4 个 M3 螺丝定位 → 振动不会松           │
└─────────────────────────────────────────────────────────────┘
```

### 焊点关键点
| 位置 | 处理 |
|---|---|
| XT60 焊点 | 用 **大功率烙铁 60W** 预热 5 秒，焊锡饱满（每秒过 8A 不发热）|
| 保险丝两脚 | RXEF050 引脚要插稳，不要"飞线焊接"（电流大易脱焊） |
| LM2596 IN/OUT | 焊接前**先调好输出电压**，防止输出 12V 烧 MCU |
| GND 单点 | 所有 GND 线汇集到 XT60 负极一个点，**禁止地环**|

### 关键间距
| 项 | 距离 | 原因 |
|---|---|---|
| MPU6050 ↔ 电机驱动 | ≥ 5 cm | 防止 PWM 对陀螺仪干扰 |
| 编码器线 ↔ 电机线 | 杜邦线分两路走 | 防止互感串扰 |
| OLED I2C 线长 | ≤ 15 cm | I2C 长线易拉低 |
| 电池正极 ↔ MPU 中心 | ≥ 8 cm | 大电流路径远离精密器件 |

## 接地策略（最关键）

```
                                  ┌──────────────────┐
                                  │  XT60 电池负极   │ ← 单点共地
                                  └────────┬─────────┘
                                           │
              ┌────────────────────────────┼────────────────────────┐
              │                            │                        │
              ▼                            ▼                        ▼
       PGND 总线（板 A）         SGND 总线（板 B）          电机外壳/底盘金属
       - 电机驱动 GND             - MSPM0 GND
       - LM2596 GND               - IMU GND
       - 5V 大电流回流            - OLED GND
                                  - 红外 GND
                                  - 按键 GND
```

**严禁**：
- 板 A 的 GND 与板 B 的 GND 多点连接（会形成地环路 → IMU 数据漂）
- 把 MPU6050 GND 接到 LM2596 输出地（PWM 纹波会进 IMU）

## 走线规则

| 线种 | 颜色 | 截面积 / 线规 | 走法 |
|---|---|---|---|
| 7.4V 主轨 | 红 | 0.5mm² 硅胶线 | 直走，避开信号板 |
| 5V 信号 | 黄 | 0.3mm² | 沿板边走 |
| GND | 黑 | 0.5mm²（电源）/ 0.2mm²（信号）| 单点汇集 |
| 电机 PWM | 灰 | 0.3mm² 杜邦线 | 双绞 |
| I2C SCL/SDA | 蓝/白 | 0.2mm² | 双绞 ≤15cm |
| 编码器 A/B | 绿/橙 | 0.2mm² | 双绞，远离电机线 |

## 散热

| 元件 | 散热处理 |
|---|---|
| LM2596 | 自带散热片，**长时间运行可加 5×5×3mm 铝散热片** |
| TB6612 | 1A 持续可不加散热；接近 1.2A 时加散热硅脂或散热片 |
| MSPM0 / MPU | 不发热 |

## 振动 / 机械

```
小车上电跑过 1 圈后必检：
☐ 所有杜邦线插接处不松动（关键：MPU6050 / 编码器 / 电机线）
☐ 万用板 4 个 M3 螺丝紧固
☐ 电池仍在原位（用魔术贴 / 扎带固定）
☐ 红外 5 路模块离地仍 5~8mm（震动易松）
☐ 编码器固定螺丝紧
```

## 故障定位 × 布局

| 现象 | 可能 | 用万用板布局快速定位 |
|---|---|---|
| 上电就断电 | 短路 | 用万用表蜂鸣档测板 A 的 7.4V 与 GND |
| MCU 不工作 | 5V 欠压 | 测板 B 5V 轨：< 4.8V → LM2596 输出问题 |
| IMU 全 0 | I2C 上拉缺失 | 测 SCL/SDA 静态电平：应 ≈ 3.3V，若 < 2V 则上拉电阻断 |
| 电机响 1 声不动 | TB6612 STBY 未接 | 测 STBY 引脚应 = 3.3V |
| 直线走偏 | 编码器线串扰 | 拆开编码器线与电机线，分两路走 |

## 与 5×7cm 万用板的现实约束

```
焊点数估算（以板 B 为例）：
- MSPM0 排母 J1+J3：40 个焊点
- TB6612 ×2 模块：32 个焊点
- MPU6050：8 个焊点
- OLED 4 脚 + 5 路红外 + 4 按键 + 蜂鸣器：30+ 焊点
- 5V/3V3/GND 总线焊接：50+ 焊点
─────────────────────────────────────
合计 ≈ 160+ 焊点（一个熟练焊匠 1.5h）

若用过孔元件 + 杜邦线插接，可降到 ~80 焊点。
建议：第 1 次焊接预留 3 小时（包含调试）。
```

## 备份策略（万一布局失败）

如果板布局不顺利或调试失败，**降级方案**：

1. **直接用现成开发板 + 母线板**（淘宝 ¥30 一套，全部排针引出，无需焊接）
2. **MSPM0 LaunchPad 作为信号板**，TB6612 模块直接用杜邦线连
3. **电源板做最小化**：只做 7.4V 进 + 1×LM2596 出，其余去掉

降级后**仍可拿基本要求 80 分**。
