## 关键参数交叉验证

| 项 | 计算 | 是否合理 |
|---|---|---|
| 单轮最大速度 | 333RPM × π × 65mm / 60 = 1.13 m/s | ✅ 1m 直线 <1s |
| 编码器分辨率 | 65π / 1320 = 0.155 mm/脉冲 | ✅ 1m 内误差 <1 脉冲 |
| 电机额定电流 | 0.3A × 2 = 0.6A | ✅ TB6612 1.2A 单路够 |
| 系统电流 | 1A 峰值 | ✅ 2S 1500mAh 续航 ~1.5h |
| 5V 负载 | MCU 0.1A + IMU 5mA + OLED 30mA + 红外 50mA = 0.2A | ✅ LM2596 3A 富余 |

## 安装注意

- TB6612 的 STBY 引脚务必接 3.3V（拉低则芯片休眠）
- MPU6050 INT 引脚朝固定方向安装，**禁止运行中歪斜**
- 红外模块离地 5~8mm，太低会刮地
- 编码器线尽量短且远离电机线（电磁干扰大）
- 电池放在底盘**正中靠后**，平衡前后重量
