## 电源域划分

| 电源域 | 电压 | 来源 | 负载 | 容量 |
|---|---|---|---|---|
| VBAT | 6.4~8.4V | 2S LiPo | TB6612 VM 输入 | 35A 峰值（电池侧）|
| 5V | 5.0V | LM2596 | MCU LaunchPad、MPU6050、OLED、红外、蜂鸣 | 3A（LM2596 限）|
| 3.3V | 3.3V | MSPM0 板载 LDO | 仅 MCU 内部 + GPIO 上拉 | 250mA |
| 电机两端 | 0~6.4V PWM | TB6612 | 电机 | 1.2A 单路 |

### 关键规则

1. **TB6612 VM 接 VBAT，VCC 接 5V**：VM 是功率，VCC 是控制逻辑
2. **MSPM0 LaunchPad 通过 5V 接口供电**（不要从 USB 供电跑实地，电流不够）
3. **电池电压采样**：VBAT 经分压（10k + 3.3k）→ ADC，注意分压比 = 3.3/13.3，最高 7.4×3.3/13.3=1.84V，安全
4. **共地**：VBAT 地、5V 地、3.3V 地必须**单点共地**（推荐在主控板地端汇合）

## 关键电路细节

### 1. TB6612 接线
```
LiPo 7.4V ──┬── VM (Pin 12)         电机 A+ (Pin 1)──→ 左电机 +
            └── 100uF 电解 + 0.1uF 陶瓷
                                     电机 A- (Pin 14)──→ 左电机 -
5V ──── VCC (Pin 13)                 电机 B+ (Pin 16)──→ 右电机 +
3.3V ── STBY (Pin 18)                电机 B- (Pin 11)──→ 右电机 -
GND ─── GND (Pin 5/9/15)
PWMA ── PWMA (Pin 17) 50kHz PWM
PWMB ── PWMB (Pin 6) 50kHz PWM
AIN1 ── AIN1 (Pin 19)
AIN2 ── AIN2 (Pin 20)
BIN1 ── BIN1 (Pin 4)
BIN2 ── BIN2 (Pin 3)
```

去耦电容**必须在 TB6612 引脚 1cm 内**（VM-GND），否则起步瞬间电流变化会导致 5V 域跌落。

### 2. 红外阈值滞回（软件实现）

电位器调到黑/白电压差 ≥ 2V 后，软件做：
```c
#define IR_THRESH_HIGH  1   // GPIO 数字版直接读取
#define IR_THRESH_LOW   0
```
本设计用的是数字版红外模块（板载比较器），所以不需软件滞回。如果赛场上发现误触发严重，**改用模拟版 + 软件滞回**作为兜底。

### 3. MPU6050 I2C 上拉

MPU6050 模块板载 4.7kΩ 上拉到 VCC（3.3V 或 5V）。如果用 5V 供电模块，注意 SDA/SCL 也是 5V 电平，**MSPM0 5V 容忍引脚才行**。  
**安全做法**：MPU6050 用 3.3V 供电（GY-521 默认 3.3~5V，板载 LDO），SDA/SCL 直接 3.3V。

### 4. 电池电压采样

```
VBAT ──┬── 10kΩ ──┬── ADC（PA25）
       │           │
       │           ├── 3.3kΩ
       │           │
       └────────── GND
```

分压比 = 3.3/(10+3.3) = 0.248  
ADC 最大输入 = 8.4V × 0.248 = 2.08V < 3.3V ✅  
软件还原：`vbat_real = adc_volt / 0.248`

### 5. 接地策略

- 主控板的 GND 是中央汇集点
- TB6612 GND 短粗线接主控板 GND
- 电池 GND 经保险丝后接主控板 GND
- 所有信号 GND（红外、IMU、编码器）就近接主控 GND
- **禁止环形地**（GND 走线绕一圈回主控会形成接地环路，引入噪声）

## 硬件调试顺序（先验证再上电）

1. **不上电检查**：用万用表蜂鸣档检查 VBAT 与 GND 是否短路
2. **空载上电**：电池接上，**未接电机**，测 5V 输出是否稳定
3. **5V 加负载**：插上 MCU + IMU，测 5V 是否仍稳定（不应跌落 > 0.2V）
4. **TB6612 不接电机加电**：测 PWMA/PWMB 引脚电压（无 PWM 时应 0V）
5. **接一只电机正反转测试**：确认 AIN1/AIN2 切换时电机方向正确
6. **接两只电机**：再次测 5V 稳定（电机起步瞬间是最大冲击）
7. **接 IMU + OLED**：用串口打印 IMU ID（应 0x68）+ OLED 显示文字
8. **接红外**：放白纸/黑线下方，串口打印 5 路状态
9. **接编码器**：手动转轮子，串口打印计数值（每圈应 ±1320）

每完成一步用万用表测**关键节点电压 + 电流**，记录到 `04_调试记录/调试检查清单.md`。

## 安全 / 应急

- **小车失控**：按 KEY_STOP，PWM 立即归零；同时硬件设计上保证 STBY 接 GPIO，软件可主动关 TB6612
- **电池过放**：ADC 监测 < 6.5V 蜂鸣 + 强制停车
- **电机堵转**：长时间高 PWM 但编码器无变化 → 软件检测后停车（TB6612 自带过温保护，但响应慢）
- **场地之外**：编码器+IMU 推算位置超出场地边界 1.2 倍 → 强制停车
