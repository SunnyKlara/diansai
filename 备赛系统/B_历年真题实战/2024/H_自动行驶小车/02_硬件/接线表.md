## 8. 接线检查表（焊完必走一遍）

- [ ] 电池 + - 不能反接：用万用表先量电池电压再插
- [ ] TB6612 VM 与 VCC 不要短接
- [ ] STBY 默认必须拉高（万用表量 PB18 = 3.3V 才输出 PWM）
- [ ] MPU6050 SDA/SCL 实测有 4.7kΩ 上拉
- [ ] 编码器 A/B 不要接反，否则方向反向
- [ ] 5 路 IR 上电后，黑色物体下输出 < 0.5V，白色 > 2.5V
- [ ] 共地：TB6612 GND / MCU GND / 电池 GND 必须一线连通

## 9. 常见错误

| 现象 | 大概率原因 |
|---|---|
| 电机不转 | STBY 没拉高 / 方向引脚未配置输出 |
| 编码器不计数 | A/B 接反 / TIMG0 没配 QEI 模式 |
| MPU6050 I2C 通信失败 | 没接上拉电阻 / SDA/SCL 接反 |
| 5 路 IR 输出乱跳 | VCC 不稳（电池电量低）/ IR 板高度不对 |
| 上电瞬间小车冲出 | PWM 默认值非 0 / 方向位悬空 |
