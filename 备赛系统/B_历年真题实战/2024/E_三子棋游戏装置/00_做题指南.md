## 机械方案选择

### 方案A：XY滑台+Z轴升降（推荐）
- 两根导轨+步进电机/舵机实现XY移动
- 一个舵机实现Z轴升降
- 电磁铁吸取棋子（棋子内嵌铁片）
- 精度高，控制简单
- 可用3D打印制作

### 方案B：二自由度机械臂
- 两个舵机实现肩关节+肘关节
- 末端夹爪或电磁铁
- 工作空间大但精度控制难

### 推荐方案A，理由：
- 精度更容易保证（30mm方格要求不低）
- 控制算法简单（直角坐标，不需要逆运动学）
- 3D打印结构件容易制作

## 博弈算法（三子棋已解）

```c
// Minimax算法 - 三子棋状态空间很小，可以穷举
// 棋盘用int board[9]表示，0=空，1=己方，-1=对方

int minimax(int board[9], int depth, int is_maximizing) {
    int winner = check_winner(board);
    if (winner == 1) return 10 - depth;   // 己方赢
    if (winner == -1) return depth - 10;  // 对方赢
    if (is_full(board)) return 0;          // 平局
    
    if (is_maximizing) {
        int best = -100;
        for (int i = 0; i < 9; i++) {
            if (board[i] == 0) {
                board[i] = 1;
                int score = minimax(board, depth + 1, 0);
                board[i] = 0;
                if (score > best) best = score;
            }
        }
        return best;
    } else {
        int best = 100;
        for (int i = 0; i < 9; i++) {
            if (board[i] == 0) {
                board[i] = -1;
                int score = minimax(board, depth + 1, 1);
                board[i] = 0;
                if (score < best) best = score;
            }
        }
        return best;
    }
}

// 找最佳落子位置
int find_best_move(int board[9]) {
    int best_score = -100;
    int best_move = -1;
    for (int i = 0; i < 9; i++) {
        if (board[i] == 0) {
            board[i] = 1;
            int score = minimax(board, 0, 0);
            board[i] = 0;
            if (score > best_score) {
                best_score = score;
                best_move = i;
            }
        }
    }
    return best_move;
}
```

**三子棋结论**：先手最优策略必不败（平局或赢），后手最优策略也必不败（平局）。
