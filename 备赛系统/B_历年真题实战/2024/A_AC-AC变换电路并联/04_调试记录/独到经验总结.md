# 2024 A AC-AC 变换电路并联 —— 独到经验总结

## 1. AC-AC 直接变换的"违规"陷阱

题目要求**严禁 AC→DC→AC**，评委会拆开外壳验证。常见违规：
- 加母线大电容（看起来是滤波，实际是 DC 母线）
- 任何全桥整流 → 直接 -10 分

合规判据：从输入到输出**没有单向元件**形成 DC 路径。

## 2. 四步换流的电流方向自适应

题目里 io 方向不固定（容性/感性负载会让 io 与 vin 不同步）。  
真机做法：实时采 io 极性，**根据 io 方向选 4 步顺序**。

```c
if (io > 0) {
    // 正→负切换：先关 S1 不导通管 → 开 S2 即将导通 → ...
} else {
    // 反过来
}
```

**未做电流方向自适应 → 切换瞬间烧 1~2 个管**（实测踩过坑）。

## 3. 双机并联的"下垂电阻必须一致"

下垂电阻不一致：R_d1=0.1, R_d2=0.15 → 满载时 I_1 / I_2 = 1.5（K=1.5），评委直接扣 (4) 项分数。

校准方法：
1. 单台开机，在不同电流（1, 2, 3, 4A）下测 V_out
2. 拟合斜率 → R_d_actual
3. 软件下垂系数补偿到目标 0.10Ω（精度 ±5%）

## 4. 重量考核的真实分值差距

10 分占比看似小，但实际：
- < 400g：满分 10
- 400~600g：8
- 600~800g：5
- 800g+：0

**减重 200g 直接 +5 分，比调 0.1% 调整率划算**。

## 5. 过零检测光耦的延迟陷阱

PC817 上升沿延迟 ~ 18μs，下降沿 ~ 50μs。  
若软件用同一阈值检测过零 → **正负半周不对称**，导致输出有直流偏置（评委测电压有 0.5V 直流分量直接扣 5 分）。

对策：
- 用 LM393 比较器 + 过零参考点（不用光耦）
- 或软件补偿：检测到过零后**等 50-18=32μs** 才认为真过零
