# 2024 A AC-AC 变换电路并联运行 —— 深度审题与方案论证

> 与 2025 A 三相能量回馈、2023 A 单相逆变并联同属电源类。  
> 核心难点：**AC-AC 直接变换 + 双向开关 + 四步换流 + 双机并联无通信均流**。

### 隐含约束

**约束 1：四步换流是 AC-AC 必备技能**  
不能简单 PWM 切换两组双向开关：换流瞬间若同时关断 → 电感电流无回路 → 烧管；同时导通 → 输入短路。  
正确顺序（4 步）：先关断不导通管 → 开通对侧不导通管 → 关断对侧导通管 → 开通对侧导通管。  
每步死区 ≥ 1 μs。

**约束 2：双机并联无通信约束**  
两台变换器**只共享输入和输出端**，不允许 CAN/UART 等通信。**必须靠自然均流**，即下垂特性（Droop）：  
- 输出阻抗精确匹配 → 同电压设定下电流自然均分
- 也可以通过载频微调（一台 19.95kHz，一台 20.05kHz）打散开关纹波，避免两台同步谐振

**约束 3：负载调整率 0.5% = 175 mV @ 35V 输出**  
开环跌落约 1V（电感 ESR + 死区 + MOSFET Rds_on）→ **必须电压闭环 PI**。

## 第二层：误差与效率预算

### 效率 ≥ 90%

满载 P_out = 35V × 4A = 140W → P_loss ≤ 15.5 W。

| 损耗源 | 估算 |
|---|---|
| 4×IRF3205 导通（Rds = 8mΩ × 2 串联）| 4² × 16mΩ × 4 = 1.0W |
| 开关损耗（fsw=20kHz, ts=100ns）| 4 × ½ × 36 × 4 × 100ns × 20kHz = 0.6W |
| 电感铜损（DCR=80mΩ）| 4² × 80mΩ × 2台 = 2.6W |
| 电感铁损 | ~ 2.0W |
| 死区 + 体二极管 | 0.8W |
| 驱动 IC 静态 | 4 × 100mA × 12V = 4.8W ⚠️ |
| **合计** | **~ 11.8W** |
| **效率** | **140/151.8 = 92.2%** ✅ |

**关键发现**：IR2110 自身静态电流是大头。冲刺 92%+ 需换 EG3002（< 5mA 静态）。

## 第五层：硬件难点

### 难点 1：四步换流的时序

```
正→负切换（电压过零，正半周即将结束）：
  t0: S1 完全导通，S2 完全关断
  t1: 关 S1 不导通的那个管（视电流方向定）
  t2: 开 S2 将导通的那个管
  t3: 关 S1 导通的管
  t4: 开 S2 另一个管 → 完全切到 S2
  
每步间隔 1 μs，4 步共 4 μs（远小于 50Hz 半周期 10ms）
```

⇒ HRTIM 4 通道独立控制 + GPIO 软件触发即可。

### 难点 2：双机并联自然均流（无通信）

下垂方程：

$$ V_{out} = V_{set} - R_d \cdot I_{out} $$

R_d 取 0.1 Ω：满载 4A 偏差 0.4V（占 35V 的 1.1%）。  
两台 R_d 一致 → 电流自动均分（K = 1.0±0.05 容易，K = 0.5~2 需主动调 V_set）。

发挥项 (6) K=0.5~2：通过给一台改变 V_set 使其分担更多电流（用拨码或拨盘开关）。

### 难点 3：输入过零附近的窄脉冲畸变

Vin 过零附近（< 5V），即使 D = 0.95 也只能输出 4.75V，**无法进一步抬高**。  
对策：
- 占空比限幅：当 |Vin| < 8V 时，duty 至少 0.5 防止饱和畸变
- 或：开环模式只在过零附近 ±1ms 切换到电流跟随

### 难点 4：体二极管反并问题

共源极双向开关由两个 N-MOS 背靠背组成，**两个寄生体二极管也是背靠背**——在不导通管的体二极管承担反向电压。死区期间瞬时通流路径需要走体二极管，损耗大。  
对策：死区 ≤ 800 ns，或用集成 SiC 双向器件（贵）。
