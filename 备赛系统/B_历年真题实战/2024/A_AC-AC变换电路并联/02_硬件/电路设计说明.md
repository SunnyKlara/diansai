# 2024 A AC-AC 变换电路并联 —— 电路设计说明

## 系统总图（×2 套并联）

```
36V AC ──┬─→ [斩波器 1: 4×IRF3205 + L+C]──┬→ Vout 1~35V → 负载（~9Ω）
         │                                 │
         └─→ [斩波器 2: 同上]              ┘
              ↑                            ↑
         独立 STM32G474 ×2，共享 Vin / Vout，无通信
```

## 模块 1：双向开关（每台 4 管）

```
          Vin (+)
           │
        ┌──┴──┐
        │ S1a │  共源极
        │     │  N-MOSFET
        │ S1b │
        └──┬──┘
           │
           ├─── L (1mH) ─── Vout (+)
           │
        ┌──┴──┐
        │ S2a │  共源极
        │     │
        │ S2b │
        └──┬──┘
           │
          Vin (-)

正半周：S1a/S1b 同步 PWM，S2a/S2b 关
负半周：S1a/S1b 关，S2a/S2b 同步 PWM
切换瞬间：四步换流避免直通/开路
```

## 模块 2：四步换流时序

```
正→负切换（io > 0）：
  t0    : (CCR1=d, CCR2=d, CCR3=0, CCR4=0)
  t0+1us: 关 S1b → CCR2=0
  t0+2us: 开 S2a → CCR3=d
  t0+3us: 关 S1a → CCR1=0
  t0+4us: 开 S2b → CCR4=d
```

## 模块 3：滤波器与电流采样

| 元件 | 值 | 备注 |
|---|---|---|
| L | 1 mH / 5A | T130-26 70 匝 0.8mm |
| C | 4.7 μF / 100V CBB | X2 安规 |
| 取样 | ACS712-5A | 霍尔，相位准 |

LC fc = 1/(2π·√(1e-3 · 4.7e-6)) = 2.32 kHz，对 20 kHz 衰减 38 dB。

## 模块 4：过零检测

```
36V AC ─[100kΩ]─→ PC817 LED ─→ PC817 集电极 ─[10kΩ↑]─→ EXTI
                  │
                 [反向并联保护二极管 1N4148]
```

光耦自身延迟 < 50μs，不影响 50Hz 周期检测。  
软件滤波：连续 3 次相同电平才确认。

## 模块 5：辅助电源（轻量化方案）

| 输出 | 来源 | 方案 | 备注 |
|---|---|---|---|
| +12V | 36V AC 整流 + 7812 | 5W | 驱动 IC |
| +5V  | +12V → AMS1117 | 0.5W | LCD |
| +3.3V | +5V → AMS1117 | 0.3W | MCU、ADC |

不能用变压器降压 → **必须** 用 AC 整流后线性稳压（或开关电源模块）。

## 重量优化

| 部件 | 原方案 g | 优化后 g |
|---|---|---|
| 电感（T130-26）| 90 | 50（PQ 平面）|
| 散热片 | 16 | 8（铝合金薄）|
| PCB 主板 | 80 | 60（双面 1.0mm）|
| **单台合计** | **260** | **180** |
| **两台合计** | **520** | **360** |

## 与 2023 A / 2025 A 的差异

| 维度 | 2023 A 单相逆变 | 2025 A 三相回馈 | **2024 A AC-AC** |
|---|---|---|---|
| 拓扑 | DC→AC 全桥 | DC↔AC 三相桥 | **AC↔AC 双向开关** |
| 控制律 | SPWM | SVPWM + 同步整流 | **PWM 斩波 + 四步换流** |
| 双机 | 同步 + 锁相 | — | **下垂均流（无通信）** |
