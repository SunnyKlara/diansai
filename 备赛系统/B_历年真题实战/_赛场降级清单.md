# 赛场降级清单 —— 翻车时的最小可行方案

> 比赛翻车几乎不可避免。这份清单告诉你：**故障 X → 切换到方案 Y → 至少保住 Z 分**。  
> 哲学：**满分是奢侈品，及格是必需品**。

---

## 总体降级原则

```
完整方案 (100%)
    ↓ 出故障
降级方案 1 (75%)
    ↓ 还出故障
降级方案 2 (50%)  ← 多数情况能稳到这里
    ↓ 极端情况
降级方案 3 (25%)  ← 至少拿基本要求 + 报告
```

## 1. 2024 H 自动行驶小车

### 完整方案：5 路 IR + IMU + 编码器 + 双 PI

### 降级树
| 故障 | 降级 | 保住分数 |
|---|---|---|
| MPU6050 坏 | **关 IMU 航向，纯循迹模式**（注释 imu_mpu6050.c 调用）| 70% |
| 编码器单边坏 | **去掉速度环，用纯 PWM 占空比开环**（调小 PWM 防震荡）| 60% |
| 一路 IR 坏 | **改 4 路 IR 加权**（修改 ir_line.c 中的权重数组）| 80% |
| TB6612 烧 | **换备用模块**（5 分钟动作）| 100% |
| 主板烧 | **换备用主板**（已烧好软件）| 100% |
| 全部传感器异常 | **退到键盘控制小车**（KEY1 前进，KEY2 左转）| 25% |

### 现场操作流程
```
出故障 → 看 LED 状态 → 翻 故障决策树 → 5 分钟修不好 → 切降级
```

## 2. 2024 B 单相功率分析仪

### 完整方案：6.4 kSPS DFT + LCD 段显 + LPM3 低功耗

### 降级树
| 故障 | 降级 | 保住分数 |
|---|---|---|
| 段显 LCD 坏 | **改用串口输出到 PC 显示**（uart_dbg.c 已实现）| 90% |
| LPM3 进不去 | **不进低功耗模式**（注释主循环 LPM 调用），多耗电但功能正常 | 90% |
| 单频 DFT 不准 | **切到时域 RMS**（algo_simple_rms.c）| 70% |
| ADC 单通道坏 | **只测电压不测电流**，给 P 估算值（V × I_estimated）| 50% |
| 整套软件挂 | **退到串口手动测**：MCU 当 ADC 转 USB | 30% |

### 现场操作
```
LCD 不亮 → 检查 SPI 时钟 → 不行 → uart_dbg 输出 → 评委也认
```

## 3. 2025 G 电路模型探究装置

### 完整方案：DDS + 双 ADC 同步 + FFT + DAC X-Y

### 降级树
| 故障 | 降级 | 保住分数 |
|---|---|---|
| AD9833 坏 | **改用 MCU 内部 PWM 滤波生成正弦**（精度低 10 倍但能跑）| 60% |
| ADC2 坏 | **单 ADC 切换通道**（精度 / 速度都降，但能算）| 70% |
| OPA2277 坏 | **改用 LM358**（带宽差但能用），重新校准增益 | 65% |
| DAC 输出 X-Y 失败 | **OLED 显示 |Z| 数字**，不展示 X-Y | 85% |
| FFT 算法挂 | **改用单频 DFT**（更慢但更稳）| 80% |
| 整套挂 | **退到万用表测电阻 / LCR 表演示**（带备用 LCR 表）| 30% |

## 4. 2023 A 单相逆变器并联

### 完整方案：HRTIM SPWM + 电压外环 + 双机并联

### 降级树
| 故障 | 降级 | 保住分数 |
|---|---|---|
| MOSFET 烧 1 个 | **换备用**（带 4 个备件）| 100% |
| MOSFET 烧 2 个 | **关并联，单机模式**（拉掉第二台主板）| 70% |
| 母线电容失效 | **加并联备用电容**（带 470μF × 2 备件）| 95% |
| BKIN 误触发 | **临时禁用**（短接 PA12 到 GND，**只在确定不会真过流时**）| 80% |
| 双机不同步 | **关一台**，单机交付 | 70% |
| THD 超 5% | **降低输出电流到 1A 以下**（损耗减少 → THD 改善）| 60% |
| 母线降到 30V | **电源调到 36V 保持母线**（如果允许调）| 100% |
| 主控挂 | **换备用主板**（5 分钟）| 100% |
| 完全失败 | **只演示电路设计 + 答辩**，拿 20 分报告 | 17% |

## 5. 2025 A 能量回馈变流器

### 完整方案：三相 SVPWM + 同步整流 + dq 双环

### 降级树（电源类难度最高 = 翻车率最高）
| 故障 | 降级 | 保住分数 |
|---|---|---|
| 同步整流不工作 | **关同步整流，纯三相 H 桥**（损 4% 效率）| 85% |
| 一相烧 | **退到单相工作模式**（关 B 相 PWM）| 50% |
| 母线异常 | **断电 + 检查 + 重充**（只能等待）| 60% |
| dq 锁相失败 | **改用开环 SVPWM 50Hz 固定频率**（牺牲 PF）| 60% |
| 三相对称度差 | **手动调三相增益系数**（赛场实测调）| 80% |
| BKIN 频繁触发 | **降低输出电流**（限到 1A）| 70% |
| 完全失败 | **演示 SVPWM 算法 + 答辩**，拿 20 分报告 | 17% |

## 7. 5 分钟原则

```
故障开始
   ↓ 1 min
查 故障决策树
   ↓ 2 min
尝试主修复
   ↓ 3 min
尝试备用修复
   ↓ 4 min
还修不好？
   ↓ 5 min
**切降级方案，立即转下一项目**
```

**赛场不是练习场。剩余 25 分钟还有 3 个项目要测，不要在一个故障上耗死。**

## 8. 不同分数段的策略

| 你的目标 | 策略 |
|---|---|
| 只要进省赛 | 完整方案打 50%，全部基本要求 + 报告 = 65 分 |
| 冲省一 | 完整方案打 80%，2 个发挥项 + 报告 = 80 分 |
| 冲国奖 | 完整方案打 100%，**做出至少 1 项国一级别发挥** |

**降级方案适合前两类**（实战务实）。第三类必须**极致追求完整**，但仍要带备件防翻车。

## 9. 心态管理

```
降级 ≠ 失败
降级 = 把 30 分的损失变成 5 分的损失
```

最坏的情况不是降级——是你**不敢降级**，硬刚 1 小时还是失败 + 错过其他项目。

**电赛比拼整体得分，不是某一项满分**。

---

## 10. 最后兜底（实在跑不起来）

如果整套装置完全无响应：
1. **关电**（防起火）
2. 演示 PCB / 焊接工艺（评委也会给点工艺分）
3. **现场答辩为重**（30 问速查卡）—— 答辩好可挽回 5~10 分
4. 报告**精装**版主动上交，争取报告满分（20 分）

**最差状态：报告 20 + 答辩 5 = 25 分**，至少不是零分。
