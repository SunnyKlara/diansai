# 2024年电赛C题 - 自动行驶小车

## 项目概述
基于STM32的自动循迹行驶小车，使用灰度传感器进行黑线检测，PID算法控制转向和速度。

## 硬件方案
- **主控**: STM32F103C8T6 (或 STM32F407)
- **传感器**: 6路灰度传感器（循迹）、MPU6050（姿态）、编码器（测速）
- **驱动**: TB6612 / L298N 电机驱动
- **电机**: 带编码器的直流减速电机 x2
- **电源**: 7.4V锂电池 + 5V/3.3V稳压

## 核心算法
1. **灰度传感器加权算法** - 将6路传感器值加权计算偏差
2. **位置式PID** - 根据偏差计算转向修正量
3. **速度PID** - 编码器闭环控制车速
4. **弯道检测** - 根据传感器状态判断弯道并减速
